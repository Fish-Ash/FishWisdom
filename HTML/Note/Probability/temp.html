数学期望（\(\mu\)）

性质：

  • \(E(C) = C\)
  • \(E(CX) = CE(X)\)
  • \(E(X+Y) = E(X) + E(Y)\)
  • 当\(X\)与\(Y\)为两相互独立的随机变量时：\(E(XY) = E(X)E(Y)\)
  • \(E[g(x)] = \int_{-\infty}^{+\infty}g(x)f(x)dx\)
  • \(E[g(x,y)] = \int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty}g(x,y)f(x,y)dxdy\)

方差（\(\sigma^2\)）： $$\begin{align} D(X) &= E\{[X-E(X)]^2\}\\ &= E[X^2-2XE(X)+E^2(X)]\\ &= E(X^2) - 2E(X)E(X) + E^2(X)\\ &= E(X^2) - E^2(X) \end{align}$$ 性质：

  • \(D(C) = 0\)
  • \(D(aX+b) = a^2D(X)\)
  • \(D(X\pm Y) = D(X) + D(Y) \pm 2Cov(X,Y)\)
    若\(X\)与\(Y\)相互独立，则\(D(X\pm Y) = D(X) + D(Y)\)

标准差（均方差）(\(\sigma\))： $$\sigma(X) = \sqrt{D(X)}$$

协方差： $$\begin{align} Cov(X,Y) &= E\{[X-E(X)][Y-E(Y)]\}\\ &= E(XY) - E(X)E(Y) \end{align}$$ 性质：

  • \(Cov(X,X) = D(X)\)
  • \(Cov(X,Y) = Cov(Y,X)\)
  • \(Cov(C,X) = Cov(X,C) = 0\)
  • \(Cov(aX, bY) = abCov(X,Y)\)
  • \(Cov(X_1+X_2, Y) = Cov(X_1,Y) + Cov(X_2, Y)\)
  • 若\(X\)与\(Y\)相互独立，则\(Cov(X,Y) = 0\)

相关系数： $$\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{D(X)}\sqrt{D(X)}}$$

离散型随机变量

数学期望： $$E(X) = \sum\limits_{k=1}^{\infty}x_kp_k$$ 方差： $$D(X) = \mathop{\sum}\limits_{k=1}^{\infty}[x_k - E(X)]^2p_k$$ 最大似然估计： $$L(\theta) = \prod\limits_{i=1}^nP\{X=x_i\}$$

退化分布（单点分布）

$$ P\{X = C\} = 1 $$

二项分布


    n重伯努利试验中，事件A发生的概率为\(p\)，\(X\)表示n重伯努利试验中事件A发生的次数，则称\(X\)服从参数为\(n,p\)的二项分布，记作\(X\sim b(n,p)\) $$ P\{X = k\} = C_n^kp^k(1-p)^{n-k},k=0,1,2,\cdots,n $$

（0-1）分布（两点分布）


    二项分布\(n = 1\)时的特殊情况，记作\(X\sim b(1,p)\)$$P\{X = k\} = p^k(1-p)^{(1-k)}, k=0,1$$

几何分布


    若事件A发生的概率为\(p\)，\(X\)表示事件A首次发生时已进行的次数，则称X服从参数为p的几何分布，记作\(X\sim G(p)\) $$ P\{X = k\} = p(1-p)^{k-1},~k=1,2,\cdots $$

泊松分布

定义：
    若\(X\)服从参数为\(\lambda\)的泊松分布，记作\(X\sim P(\lambda)\) $$P\{X = k\} = \frac{\lambda^k e^{-\lambda}}{k!},~k = 0,1,2,\cdots,~\lambda>0 $$ 最大似然估计：
    设\(x_1, x_2,\cdots, x_n\)为样本\(X_1, X_2, \cdots,X_n\)的观测值。 $$\begin{align} L(\lambda) &= \prod\limits_{i=1}^n\frac{\lambda^{x_i} e^{-\lambda}}{x_i!}\\ &= e^{-n\lambda}\frac{\lambda^{\sum\limits_{i=1}^nx_i}}{\prod\limits_{i=1}^nx_i!}\\ \therefore lnL(\lambda) &= -n\lambda + \sum\limits_{i=1}^nx_iln\lambda - \sum\limits_{i=1}^nln(x_i!)\\ 令 \frac{dlnL(\lambda)}{d\lambda} &= -n + \frac 1\lambda\sum\limits_{i=1}^nx_i = 0\\ \therefore 最大似然估计值：\hat \lambda &= \frac1n\sum\limits_{i=1}^nx_i = \bar x\\ \therefore 最大似然估计量：\hat \lambda &= \bar X \end{align}$$

连续型随机变量

数学期望: $$E(X) = \int_{-\infty}^{+\infty}xf(x)dx$$ 方差： $$D(X) = \int_{-\infty}^{+\infty}[x - E(X)]^2f(x)dx$$ 最大似然估计： $$L(\theta) = \prod\limits_{i=1}^nf(x_i;\theta)$$

正态分布

定义：
    若连续型随机变量\(X\)的概率密度为 $$ f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}},~-\infty< x < +\infty $$ 称\(X\)服从参数为\(\mu,\lambda\)的正态分布，记作\(X\sim N(\mu, \sigma^2)\)
