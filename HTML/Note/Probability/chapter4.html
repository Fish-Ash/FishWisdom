第四章 随机变量的数字特征

第一节 数学期望

一、随机变量的数学期望

数学期望（\(\mu\)）

离散型随机变量：

$$E(X) = \mathop{\sum}\limits_{k=1}^{\infty}x_kp_k$$

泊松分布：\(x\sim P(\lambda)\) $$P\{X = k\} = \frac{\lambda^ke^{-\lambda}}{k!}, k=0,1,\cdots$$ $$\begin{align} E(X) &= \mathop{\sum}\limits_{k=0}^\infty k\cdot\frac{\lambda^ke^{-\lambda}}{k!}\\ &= \lambda e^{-\lambda}\mathop{\sum}\limits_{k=1}^\infty\frac{\lambda^{k-1}}{(k-1)!}\\ &= \lambda e^{-\lambda} e^{\lambda}\\ &= \lambda \end{align}$$

连续型随机变量：

$$E(X) = \int_{-\infty}^{+\infty}xf(x)dx$$

均匀分布：\(x\sim U(a,b)\) $$f(x) = \begin{align}\begin{cases} \frac{1}{b-a}, ~& a\leq x\leq b\\ 0,~&others \end{cases}\end{align}$$ $$\begin{align} E(X) &= \int_{-\infty}^{+\infty}xf(x)dx\\ &= \int_a^b\frac{x}{b-a}dx\\ &= \frac{a+b}{2} \end{align}$$

二、随机变量函数的数学期望

三、数学期望的性质

  • \(E(C) = C\)
  • \(E(CX) = CE(X)\)
  • \(E(X+Y) = E(X) + E(Y)\)
  • 当\(X\)与\(Y\)为两相互独立的随机变量时：\(E(XY) = E(X)E(Y)\)

第二节 方差

一、方差的概念

方差（\(\sigma^2\)）： $$\begin{align} D(X) &= E\{[X-E(X)]^2\}\\ &= E[X^2-2XE(X)+E^2(X)]\\ &= E(X^2) - 2E(X)E(X) + E^2(X)\\ &= E(X^2) - E^2(X) \end{align}$$

标准差（均方差）：
    \(\sigma(X) = \sqrt{D(X)}\)

    方差和标准差均用于刻画\(X\)的取值与数学期望\(E(X)\)的偏离程度。\(D(X)\)越小，\(X\)取值越集中。

    标准差的优点是其与数学期望具有相同的量纲。

二、方差的计算

离散型随机变量：

$$D(X) = \mathop{\sum}\limits_{k=1}^{\infty}[x_k - E(X)]^2p_k$$

连续型随机变量：

$$D(X) = \int_{-\infty}^{+\infty}[x - E(X)]^2f(x)dx$$
