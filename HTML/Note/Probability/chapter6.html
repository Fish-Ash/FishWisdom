第二节 抽样分布

统计量：
    不含未知参数的\(g(X_1, X_2, \cdots, X_n)\)

常用统计量：

  • 均值： $$\bar X = \frac1n \mathop{\sum}\limits_{i=1}^nX_i$$
  • 方差： $$\begin{align} S^2 &= \frac{1}{n-1}\mathop{\sum}\limits_{i=1}^n(X_i - \bar X)^2\\ &= \frac{1}{n-1}\mathop{\sum}\limits_{i=1}^n(X_i^2 - 2\bar X X_i + \bar X^2)\\ &= \frac{1}{n-1}(\mathop{\sum}\limits_{i=1}^nX_i^2 - 2\bar X\mathop{\sum}\limits_{i=1}^nX_i + n\bar X^2)\\ &= \frac{1}{n-1}(\mathop{\sum}\limits_{i=1}^nX_i^2 - 2\bar Xn\bar X + n\bar X^2)\\ &= \frac{1}{n-1}(\mathop{\sum}\limits_{i=1}^nX_i^2 - n\bar X^2) \end{align}$$

\(\chi^2\)分布（卡方分布）：
定义：
    设\(X_1, X_2, \cdots, X_n\)是来自正态总体\(N(0,1))的样本，称统计量： $$\chi^2 = X_1^2 + X_2^2 + \cdots + X_n^2$$ 为服从自由度为\(n\)的\(\chi^2\)分布，记作\(\chi^2\sim \chi^2(n)\)
性质：

  • \(E(\chi^2) = n\)
  • \(D(\chi^2) = 2n\)

\(t\)分布：
定义：
    设\(X\sim N(0,1), Y\sim \chi^2(n)\)，且\(X\)与\(Y\)相互独立，称\(T = \frac{X}{\sqrt{Y/n}}\)为服从自由度为\(n\)的\(t\)分布，记作\(T\sim t(n)\)
