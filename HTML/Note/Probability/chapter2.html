第二节 离散型随机变量及其概率分布

一、离散型随机变量及其概率分布

离散型随机变量：可能的取值只有有限个或可列无限多个的随机变量\(X\)，

离散型随机变量\(X\)的概率分布（概率函数、分布律（列））：\(X\)取所有可能值的概率，即\(P\{X = x_k\} = p_k,~k =1,2,\cdots\)

二项分布的泊松近似：当\(n\rightarrow \infty\)而\(p\rightarrow 0\)时，\(b(n,p)\)可以近似看作\(P(np)\)

第三节 随机变量的分布函数

一、随机变量的分布函数

随机变量\(X\)的分布函数：\(F(X) = P\{X \leq x\} \)

分布函数的性质：

  • 非负性：\(0\leq F(x)\leq 1,~x\in (-\infty, +\infty)\)
  • 单调不减性：当\(x_1 < x_2\)时，有\(F(x_1)\leq F(x_2)\)
  • 规范性：\(F(+\infty) = \mathop{lim}\limits_{x\rightarrow +\infty}F(x)=1\)，\(F(-\infty) = \mathop{lim}\limits_{x\rightarrow -\infty}F(x)=0\)
  • 对于连续型随机变量：\(F(x) = F(x\pm 0)\)

几何意义：\(F(x)\)为\((-\infty, x)\)区间上\(f(x)\)围成的面积

$$f(x)为概率密度 \Leftrightarrow \begin{cases} f(x) \geq 0, x \in \mathbb{R}\\ \int_{-\infty}^{+\infty}f(x)dx = 1 \end{cases}$$

连续型随机变量函数分布

设随机变量\(X\)具有概率密度\(f_X(x), -\infty < x < +\infty\)，且\(g(x)\)处处可导且单调，则\(Y = g(X)\)的概率密度为 $$f_Y(y) = \begin{align} \begin{cases} f_X[h(y)]|h'(y)|,~&\alpha < y < \beta\\ 0,~&others \end{cases} \end{align}$$ 其中，\(h(y)\)为\(g(x)\)的反函数，\(\alpha = min\{g(-\infty), g(+\infty)\}, \beta = max\{g(-\infty), g(+\infty)\}\)
