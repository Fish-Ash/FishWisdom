树莓派初始化

远程连接树莓派

VScode连接树莓派

  1. Install the extension: Remote Development
  2. Install the extension: Python
  3. Open Remote Explorer
  4. Select Remote(Tunnels/SSH)
  5. Select SSH-Settings
  6. Select C:\Users\FishAsh\.ssh\config

网络配置文件

wpa_supplicant.conf 
    country=CN
    ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
    update_config=1
    network={
            ssid="FishWiFi"
            psk="12345666"
            priority=10
    }

开启VNC功能

使用putty连接树莓派

运行命令：sudo raspi-config

选择Interface Options，启用VNC

配置静态ip地址

命令行窗口输入ipconfig查找电脑

sudo nano /etc/dhcpcd.conf
