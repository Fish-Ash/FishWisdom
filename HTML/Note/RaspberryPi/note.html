树莓派引脚图


树莓派初始化

远程连接树莓派

VScode连接树莓派

  1. Install the extension: Remote Development
  2. Install the extension: Python
  3. Open Remote Explorer
  4. Select Remote(Tunnels/SSH)
  5. Select SSH-Settings
  6. Select C:\Users\FishAsh\.ssh\config

网络配置文件

wpa_supplicant.conf 
    country=CN
    ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
    update_config=1
    network={
            ssid="FishWiFi"
            psk="12345666"
            priority=10
    }

开启VNC功能

使用putty连接树莓派

运行命令：sudo raspi-config

选择Interface Options，启用VNC

配置静态ip地址

命令行窗口输入ipconfig查找电脑

sudo nano /etc/dhcpcd.conf

安装putty，登录树莓派，运行命令sudo apt-get install xrdp

简单Liunx命令

  • cd Desktop/：转到桌面目录
  • ls：查看目录
  • mkdir floder：创建文件夹
  • rm -rf floder：删除文件夹
  • touch file：创建文件
  • python3 -m venv venvname创建虚拟环境
  • source venvname/bin/activate

树莓派虚拟环境管理

        sudo apt-get install virtualenv -y

树莓派设置开机自启动

创建/home/fishpi/.config/autostart文件夹

在文件夹下创建xxx.desktop文件

写入


        [Desktop Entry]
        Encoding=UTF-8
        Type=Application
        Name=Bin
        Exec=lxterminal -e "source /home/fishpi/Venv/venv_name/bin/activate && python3 /home/fishpi/Desktop/py_name.py;$SHELL"
        Termal=true
并保存

输入


        sudo chmod a+r /home/fishpi/.config/autostart/Bin.desktop
