Layout

创建布局

横向布局 

            from PyQt5.QtWidgets import QHBoxLayout
            Layout = QHBoxLayout()
纵向布局 

            from PyQt5.QtWidgets import QVBoxLayout
            Layout = QVBoxLayout()

[注]主窗口中的布局需要指定主部件.


            Layout = QVBoxLayout(CentralWidget)

在部件内创建布局

在部件内创建布局时，需要主动调用.


            self.setLayout(Layout)

美化

控制部件与布局的边距 

            Layout.setContentsMargins(left, top, right, bottom)
控制布局窗口挤压控件 

            Layout.addStretch()
