方法

类中定义的函数.

特殊方法

__init__

__init__方法的作用是，创建示例时自动初始化.


        class Creature:
            def __init__(self):
                pass
自定义方法 

        class Creature:
        def Attack(self):
        print("Attack!")

[注]定义方法（无论是特殊方法还是自定义方法）时，必须将self作为第一个参数声明；但在实例时，不需要传入self参数.

[注]是否要在__init__()中增加参数，取决于创建实例时是否依赖给定的初始值.

属性


    class Creature:
        def __init__(self, element):
            self.Element = element

实例化类


        fish = Creature(element)

称fish为Creature类的一个实例

[注]实例化类时，一定要在类名后加括号，不然就是复制类.

调用属性 

        fish.Element

继承


        class Animal(Creature):
            def __init__(self):
                super().__init__()

super().__init__()用于使子类继承父类的__init__方法.
