Mathematical Optimization Problem的基本形式： $$\begin{align} &\mathrm{minimize}~f_0(x)\\ &\mathrm{subject~to}~f_i(x)\leq b_i,~i=1,\cdots,m \end{align}$$
$$x = (x_1, \cdots, x_n)^T$$ optimization variable
$$f_0:\mathbb{R}^n \rightarrow \mathbb{R}$$ objective function
$$f_i:\mathbb{R}^n \rightarrow \mathbb{R},~i=1,\cdots,m$$ (inequality) constraint function
