第二章 尺寸精度与检测

第一节 基本术语及定义

尺寸

孔 轴
公称尺寸 \(D\) \(d\)
上极限尺寸 \(D_{max}\) \(d_{max}\)
下极限尺寸 \(D_{min}\) \(d_{min}\)
实际尺寸 \(D_a\) \(d_a\)
上偏差 \(ES\) \(es\)
下偏差 \(EI\) \(ei\)
公差 \(T_h\) \(T_s\)

孔 圆柱工件的内尺寸要素，包括非圆柱形的内尺寸要素（如键槽的宽度表面）
轴 圆柱工件的外尺寸要素，包括非圆柱形的外尺寸要素（如平键的宽度表面）

尺寸要素

用于表达理想要素参数方程的线性尺寸或角度尺寸。

尺寸
（线性尺寸）		 表示几何要素特征的尺寸要素，通过两点法测量得到的两点之间的距离。
公称尺寸 由图样规范定义的理想形状要素的尺寸，设计时给定。
是根据零件的强度、刚度等的计算和结构的设计确定的，应依据GB/T 2822-2005《标准尺寸》化整为优先数。
极限尺寸 尺寸所允许的极限值，设计时给定 上极限尺寸 尺寸所允许的最大值
下极限尺寸 尺寸所允许的最小值
实际尺寸 零件加工后的实际表面通过测量获得的尺寸，再通过拟合操作，从非理想表面模型或实际要素中建立理想尺寸要素
合格条件：\(\begin{cases}D_{min}\leq D_a\leq D_{max}\\d_{min}\leq d_a\leq d_{max}\end{cases}\)

正偏差 偏差（尺寸偏差） 某尺寸减公称尺寸的代数差。 极限偏差 上偏差
（上极限偏差）		 上极限尺寸减去公称尺寸所得的代数差。$$\begin{cases}ES = D_{max} - D\\es = d_{max} - d\end{cases}$$
负偏差 下偏差
（下极限偏差）		 下极限尺寸减去公称尺寸所得的代数差。$$\begin{cases}EI = D_{min} - D\\ei = d_{min} - d\end{cases}$$
零偏差 实际偏差 实际尺寸减去公称尺寸所得的代数差。
公差（尺寸公差）

上极限尺寸与下极限尺寸之差，或上极限偏差与下极限偏差之差。是实际尺寸的允许变动量。

$$T_h = D_{max} - D_{min} = ES - EI$$ $$T_s = d_{max} - d_{min} = es - ei$$ 基本偏差

最接近公称尺寸的极限偏差。

一般来说，孔的基本偏差是指\(EI\)，轴的极限偏差是指\(es\)。

配合相关术语及定义

配合

设计时定义的具有理想形状的工件模型，公称尺寸相同的、相互结合的孔和轴公差带之间的关系。

间隙与过盈

孔的尺寸与相配合的轴的尺寸之差

  • 孔的直径大于轴的直径时，差为正值，表示为间隙\(X\)
  • 孔的直径小于轴的直径时，差为负值，表示为过盈\(Y\)
配合的分类

（一）间隙配合

孔和轴装配时总存在间隙（包括最小间隙为零）的配合。

孔的下极限尺寸\(\geq\)轴的上极限尺寸，即孔的公差带在轴的公差带之上。

  • 最大间隙：\(X_{max} = D_{max} - d_{min} = ES - ei\)
  • 最小间隙：\(X_{min} = D_{min} - d_{max} = EI - es\)
  • 平均间隙：\(X_{ave} = (X_{max} + X_{min})/2\)
注：间隙值前必须冠以正号。

（二）过盈配合

孔和轴装配时总是存在过盈（包括最小过盈为零）的配合。

孔的上极限尺寸\(\leq\)轴的下极限尺寸，即孔的公差带在轴的公差带之下。

  • 最大过盈：\(Y_{max} = D_{min} - d_{max} = EI - es\)
  • 最小过盈：\(Y_{min} = D_{max} - d_{min} = ES - ei\)
  • 平均过盈：\(Y_{ave} = (Y_{max} + Y_{min})/2\)
注：过盈值前必须冠以负号。

（三）过渡配合

孔和轴装配时可能具有间隙或过盈配合。

孔公差带和轴公差带交叠。

  • 最大间隙：\(X_{max} = D_{max} - d_{min} = Y_{min}\)
  • 最大过盈：\(Y_{max} = D_{min} - d_{max} = X_{min}\)
  • 平均间隙或平均过盈：\(X_{ave}~or~Y_{ave} = (X_{max} + Y_{max})/2\)
（四）配合公差

组成配合的两个尺寸要素的尺寸公差之和，亦即配合的最大量值与最小量值之差，是对其配合所允许的变动量。

  • 间隙配合：\(T_f = T_h + T_s = X_{max} - X_{min}\)
  • 过盈配合：\(T_f = T_h + T_s = Y_{min} - Y_{max}\)
  • 过渡配合：\(T_f = T_h + T_s = X_{max} - X_{min}\)

五、ISO配合制

基孔制

孔的基本偏差为0的配合，即孔的下极限偏差为0。

基轴制

轴的基本偏差为0的配合，即轴的上极限偏差为0。

第二节 极限与配合国家标准的构成

一、标准公差系列

GB/T 1800.1-2020规定了\(20\)个等级的标准化公差数值：

标准公差分20个等级，分别用\(IT01、IT0、IT1、IT2\cdots IT18\)表示。

其中，\(IT01\)精度最高、\(IT18\)精度最低。

二、基本偏差系列

GB/T 1800.1-2020规定了\(28\)种极限偏差：
  • 孔：\(26\)个英文字母去掉\(I、L、O、Q、W\)，加上\(CD、EF、FG、JS、ZA、ZB、ZC\)
  • 孔：\(26\)个英文字母去掉\(i、l、o、q、w\)，加上\(cd、ef、fg、js、za、zb、zc\)
e.g.求\(\phi 50f6\)轴的极限偏差数值。

解：

\(\phi 50f6\)：

  • \(50\)：公称尺寸
  • \(f\)：基本偏差
  • \((IT)6\)：标准公差
