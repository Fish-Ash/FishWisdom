第三章 几何精度与检测

几何误差
  • 实际要素存在形状误差
  • 要素与要素存在方向误差或位置误差
几何公差

控制几何误差的评价指标

被测要素的提取要素相对于图纸上给定理想形状、理想方向、理想位置的允许变动量。

用于表示要素的形状、要素与要素间方向和位置的精度。

要素（几何要素）

几何公差的研究对象

构成机械零件几何特征的点、线、面

分类
按制造过程分为三个范畴
设计范畴
（机械制图中绘制的几何要素）		 公称组成要素 由技术制图或其他方法确定的理论正确的组成要素。即构成工件表面的理论轮廓要素。
公称导出要素 由一个或几个公称组成要素导出的中心点、轴线或中心平面。即构成工件表面的理论对称中心要素。
工件范畴 实际（组成）要素 加工后工件表面实际存在的几何要素。
检验范畴 提取组成要素 按规定方法，由实际（组成）要素提取有限数目的点所形成的实际（组成）要素的近似替代。
提取导出要素 由一个或几个提取组成要素计算导出的中心点、中心或中心面。
拟合组成要素 按规定方法，由非理想表面模拟或实际表面建立的理想组成要素。
拟合导出要素 由一个或几个拟合组成要素导出的中心点、轴线或中心平面。
按所处的检测关系分类
被测要素 图样上给出了形状、方向、位置或跳动公差要求的公称组成要素或公称导出要素。
基准要素

图样上规定用来确定被测要素方向或位置的公称组成要素或公称导出要素。

基准要素自身也是被测要素。
按功能关系分类
单一要素 按本身功能要求仅给出形状公差要求的被测要素。
关联要素 相对于基准要素有功能关系而给出方向、位置或跳动公差要求的被测要素。

跳动公差带的特点

跳动公差带相对于基准轴线有确定的位置或方向。

  • 径向圆跳动公差带形状与圆度公差带形状相同，其圆心位于基准轴线上。
  • 径向全跳动公差带形状与圆柱度公差带形状相同，其轴线与基准轴线同轴。
  • 轴向圆跳动公差带轴线与基准轴线重合。
  • 轴向全跳动公差带方向与基准轴线方向垂直。

跳动公差可综合控制被测要素的形状、方向或位置误差。

  • 径向圆跳动公差：圆度误差+同轴度误差
  • 径向全跳动公差：圆柱度误差+同轴度误差
  • 轴向圆跳动公差：被测端面不同半径位置的平面度误差+相对于基准轴线的垂直度误差
  • 轴向全跳动公差：被测端面的平面度误差+相对于基准轴线的垂直度误差

第四节 公差原则

独立原则 同一要素的尺寸公差与几何公差相互独立、彼此无关。
相关公差要求
体外作用尺寸
孔 \(D_{fe} = D_a - f\) 被测孔在给定长度上，与实际孔体外相接的最大理想轴的直径或宽度。
轴 \(d_{fe} = d_a + f\) 被测轴在给定长度上，与实际轴体外相接的最小理想孔的直径或宽度。

最大实体状态MMC 提取组成要素的局部尺寸处处位于极限尺寸，且使其具有材料最多的状态。 孔处于下极限尺寸时的状态 轴处于上极限尺寸时的状态
最大实体尺寸MMS 实际要素在最大实体状态下的极限尺寸。 $$D_M = D_{min}$$ $$d_M = d_{max}$$
最大实体实效状态
MMVC		 在给定长度上，实际要素处于最大实体状态且其导出要素的几何误差等于给出的几何公差值时的综合极限状态。
最大实体实效尺寸
MMVS		 实际要素在最大实体实效状态下的体外作用尺寸。 $$D_{MV} = D_{min} - t$$ $$d_{MV} = d_{max} + t$$
最小实体状态LMC 提取组成要素的局部尺寸处处位于极限尺寸，且使其具有材料最少的状态。 孔处于上极限尺寸时的状态 轴处于下极限尺寸时的状态



边界

设计时给定的具有理想形状的极限包容面。

实际轴的极限包容面为具有理想形状的孔表面。

实际孔的极限包容面为具有理想形状的轴表面。

最大实体边界
MMB		 极限包容界面的尺寸采用最大实体尺寸。
最大实体实效边界
MMVB		 极限包容界面的尺寸采用最大实体实效尺寸。
包容要求

包容要求采用最大实体边界。

孔 \(D_a \le D_{max}\) 且 \(D_{fe} \ge D_{min}\)
轴 \(d_a \ge d{min}\) 且 \(d_{fe} \le d_{max}\)
最大实体要求

最大实体要求采用最大实体实效边界。

孔 \(D_{min}\le D_a \le D_{max}\) 且 \(D_{fe} \ge D_{MV}\)
轴 \(d_{min}\le d_a \le d_{max}\) 且 \(d_{fe} \le d_{MV}\)

第五节 几何精度设计

几何公差值的选择
  • 满足功能的条件下，取低不取高。
  • 同一要素上，形状公差值\(\lt\)方向公差值\(\lt\)位置公差值。
未注几何公差等级
  • H：高精度
  • K：中等精度
  • L：低精度
