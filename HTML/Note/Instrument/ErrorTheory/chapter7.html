第七章 测量结果的不确定度评定

第二节 不确定度的基本概念

测量不确定度

测量结果带有的一个参数，意味着对测量结果可信任性、有效性的怀疑程度或不肯定程度。

测量不确定度一般包含若干分量。

按数值评定方法的不同分为两类：

  • A类评定：由一系列测量数据的统计分布获得的不确定度，用实验标准差表征。
  • B类评定：基于经验或资料及假设的概率分布，用估计的标准偏差表征。
标准不确定度（\(u\)）

以标准差表示的不确定度，其分量一般用\(u_1, u_2, \cdots, u_n\)表示。

A类评定方法

对同一被测量\(x\)独立重复观测\(n\)时，当用\(\bar x\)作为测量结果时，测量结果的A类评定的标准不确定度为：

$$u_A(x) = s(\bar x) = s(x)/\sqrt n$$ 其中，\(n\geq 6\)时，\(s(x)\)用贝塞尔公式计算；\(2\leq n\leq 5\)时，\(s(x)\)用极差法计算。 B类评定方法 $$u_B(x) = a/k$$

一般取\(k = \sqrt{3}\)

  • 舍入误差
  • 引用误差
  • 示值误差
  • 仪器基本误差
  • 仪器分辨力
  • 仪器的滞后

第五节 扩展不确定度

扩展不确定度（展伸不确定度、范围不确定度，\(U\)、\(U_p\)）

为了提高置信概率，用包含因子\(k\gt 1\)乘合成标准不确定度得到的一个区间来表示的测量不确定度。规定了测量结果取值区间的半宽度。

表示方式
  • 标准差的倍数，即合成标准不确定度乘包含因子：\(U = ku_c\)
  • 根据给定的置信概率\(p\)确定扩展不确定数：\(U_p = k_pu_c\)
包含因子（覆盖因子，\(k\)、\(k_p\)）

为确定扩展不确定度，对合成标准不确定度所乘的大于1的数。

确定包含因子的方法
  • 自由度法
  • 超越系数法
  • 简易法

第六节 测量结果的表示方法

若某标准砝码质量为\(m\)，测量结果为\(100.02147g\)，合成标准不确定度\(u_c(m) = 0.35\)mg，自由度为9。

一、合成标准不确定度表示方法

（1）\(m = 100.02147\)g，\(u_c(m) = 0.35\)mg或\(u_c(m) = 0.00035\)g，（自由度\(\nu = 9\)）。
（2）\(m = 100.02147(35)\)g，（\(\nu=9\)）。括号内的数值按标准差给出，末位与测量结果的最低位对齐。
（3）\(m = 100.02147(0.00035)\)g，（\(\nu = 9\)）。括号内的数值按标准差给出，单位同测量结果一致。

二、扩展不确定度表示方法

取包含因子\(k = 2\)，扩展不确定度\(U(m) = ku_c(m) = 0.00070\)g

（1）\(M = m \pm U(m) = (100.02147 \pm 0.00070)\)g，\(k = 2\)，（\(\nu = 9\)）
（1）\(m = 100.02147\)g，\(U(m) = 0.00070\)g，\(k = 2\)，（\(\nu = 9\)）
