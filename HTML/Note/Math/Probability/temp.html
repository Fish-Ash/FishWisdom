数学期望（\(\mu\)）

性质：

  • \(E(C) = C\)
  • \(E(CX) = CE(X)\)
  • \(E(X+Y) = E(X) + E(Y)\)
  • 当\(X\)与\(Y\)为两相互独立的随机变量时：\(E(XY) = E(X)E(Y)\)
  • \(E[g(x)] = \int_{-\infty}^{+\infty}g(x)f(x)dx\)
  • \(E[g(x,y)] = \int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty}g(x,y)f(x,y)dxdy\)

方差（\(\sigma^2\)）： $$\begin{align} D(X) = Var(X) &= E\{[X-E(X)]^2\}\\ &= E[X^2-2XE(X)+E^2(X)]\\ &= E(X^2) - 2E(X)E(X) + E^2(X)\\ &= E(X^2) - E^2(X) \end{align}$$ 性质：

  • \(D(C) = 0\)
  • \(D(aX+b) = a^2D(X)\)
  • \(D(X\pm Y) = D(X) + D(Y) \pm 2Cov(X,Y)\)
    若\(X\)与\(Y\)相互独立，则\(D(X\pm Y) = D(X) + D(Y)\)

标准差（均方差）(\(\sigma\))： $$\sigma(X) = \sqrt{D(X)}$$
    方差和标准差均用于刻画\(X\)的取值与数学期望\(E(X)\)的偏离程度。\(D(X)\)越小，\(X\)取值越集中。
    标准差的优点是其与数学期望具有相同的量纲。

协方差： $$\begin{align} Cov(X,Y) &= E\{[X-E(X)][Y-E(Y)]\}\\ &= E(XY) - E(X)E(Y) \end{align}$$ 性质：

  • \(Cov(X,X) = D(X)\)
  • \(Cov(X,Y) = Cov(Y,X)\)
  • \(Cov(C,X) = Cov(X,C) = 0\)
  • \(Cov(aX, bY) = abCov(X,Y)\)
  • \(Cov(X_1+X_2, Y) = Cov(X_1,Y) + Cov(X_2, Y)\)
  • 若\(X\)与\(Y\)相互独立，则\(Cov(X,Y) = 0\)

相关系数： $$\rho_{XY} = \frac{Cov(X,Y)}{\sqrt{D(X)}\sqrt{D(X)}}$$
    若\(\rho_{XY} = 0\)，此时\(Cov(X,Y) = 0\)称\(X\)与\(Y\)不相关

切比雪夫不等式： $$P\{|X-\mu|\geq\epsilon\}\leq\frac{\sigma^2}{\epsilon^2}$$
