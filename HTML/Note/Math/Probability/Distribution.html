离散型随机变量

数学期望： $$E(X) = \sum\limits_{k=1}^{\infty}x_kp_k$$ 方差： $$D(X) = \mathop{\sum}\limits_{k=1}^{\infty}[x_k - E(X)]^2p_k$$ 独立性： $$P\{X = x_i, Y = y_i\} = P\{X = x_i\}P\{Y = y_i\}$$ 最大似然估计： $$L(\theta) = \prod\limits_{i=1}^nP\{X=x_i\}$$

二项分布

n重伯努利试验中，事件A发生的概率为\(p\)，\(X\)表示n重伯努利试验中事件A发生的次数，则称\(X\)服从参数为\(n,p\)的二项分布，记作\(X\sim b(n,p)\)

（0-1）分布（两点分布）

二项分布\(n = 1\)时的特殊情况.

几何分布

若事件A发生的概率为\(p\)，\(X\)表示事件A首次发生时已进行的次数，则称X服从参数为p的几何分布，记作\(X\sim G(p)\)

泊松分布

最大似然估计：
    设\(x_1, x_2,\cdots, x_n\)为样本\(X_1, X_2, \cdots,X_n\)的观测值。 $$\begin{align} L(\lambda) &= \prod\limits_{i=1}^n\frac{\lambda^{x_i} e^{-\lambda}}{x_i!}\\ &= e^{-n\lambda}\frac{\lambda^{\sum\limits_{i=1}^nx_i}}{\prod\limits_{i=1}^nx_i!}\\ \therefore lnL(\lambda) &= -n\lambda + \sum\limits_{i=1}^nx_iln\lambda - \sum\limits_{i=1}^nln(x_i!)\\ 令 \frac{dlnL(\lambda)}{d\lambda} &= -n + \frac 1\lambda\sum\limits_{i=1}^nx_i = 0\\ \therefore 最大似然估计值：\hat \lambda &= \frac1n\sum\limits_{i=1}^nx_i = \bar x\\ \therefore 最大似然估计量：\hat \lambda &= \bar X \end{align}$$

分布 概率 符号 数学期望 方差
\(X\)服从退化分布（单点分布） $$P\{X = C\} = 1,~C\in \mathbb{R}$$
\(X\)服从参数为\(n,p\)的二项分布 $$ P\{X = k\} = C_n^kp^k(1-p)^{n-k},k=0,1,2,\cdots,n $$ $$X \sim b(n,p)$$ $$E(X) = np$$ $$D(X) = np(1-p)$$
\(X\)服从参数为\(p\)的\((0-1)\)分布（两点分布） $$P\{X = k\} = p^k(1-p)^{1-k}, k=0,1$$ $$X\sim b(1,p)$$
\(X\)服从参数为\(p\)的几何分布 $$ P\{X = k\} = (1-p)^{k-1}p,~k=1,2,\cdots $$ $$X \sim G(p)$$ $$E(X) = p$$
\(X\)服从参数为\(\lambda\)的泊松分布 $$P\{X = k\} = \frac{\lambda^k e^{-\lambda}}{k!},~\lambda \gt 0,~k=0,1,2,\cdots$$ $$X \sim P(\lambda)$$
$$E(X) = \lambda$$
$$\begin{align} E(X) &= \mathop{\sum}\limits_{k=0}^\infty k\cdot\frac{\lambda^ke^{-\lambda}}{k!}\\ &= \lambda e^{-\lambda}\mathop{\sum}\limits_{k=1}^\infty\frac{\lambda^{k-1}}{(k-1)!}\\ &= \lambda e^{-\lambda} e^{\lambda}\\ &= \lambda \end{align}$$
$$D(X) = \lambda$$

连续型随机变量

数学期望: $$E(X) = \int_{-\infty}^{+\infty}xf(x)dx$$ 方差： $$D(X) = \int_{-\infty}^{+\infty}[x - E(X)]^2f(x)dx$$ 独立性： $$f(x,y) = f_X(x)f_Y(y)$$ 最大似然估计： $$L(\theta) = \prod\limits_{i=1}^nf(x_i;\theta)$$

分布 概率密度 分布函数 符号 数学期望 方差
\(X\)在区间\([a,b]\)上服从均匀分布 $$f(x) = \begin{align}\begin{cases} \frac{1}{b-a},~&a\leq x\leq b\\ 0,~&others \end{cases}\end{align}$$ $$F(x)= \begin{align}\begin{cases} 0,~&x\lt a\\ \frac{x-a}{b-a},~&a\leq x\leq b\\ 1,~&x\gt b \end{cases}\end{align}$$ $$X\sim U[a,b]$$
$$E(X) = \frac{a + b}{2}$$
$$\begin{align} E(X) &= \int_{a}^{b}\frac{x}{b-a}dx\\ &= \frac{x^2}{2(b-a)}|_a^b\\ &= \frac{a+b}{2} \end{align}$$
$$D(X) = \frac{(b-a)^2}{12}$$
\(X\)服从参数为\(\lambda\)的指数分布 $$ f(x)= \begin{align}\begin{cases} 0,~&x\leq 0\\ \lambda e^{-\lambda x},~&x>0 \end{cases}\end{align},~\lambda > 0 $$ $$ F(x)= \begin{align}\begin{cases} 0,~&x\leq 0\\ 1-e^{-\lambda x},~&x>0 \end{cases}\end{align} $$ $$X\sim e(\lambda)$$ $$E(X) = \frac 1\lambda$$ $$D(X) = \frac{1}{\lambda^2}$$
\(X\)服从参数为\(\mu,\sigma\)的正态分布 $$f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}},~-\infty \lt x \lt +\infty$$ $$X\sim N(\mu, \sigma^2)$$ $$E(X) = \mu$$ $$D(X) = \sigma$$

均匀分布也可表示为\(U(a.b)\)等，与\(x\)在\(a,b\)处是否取等号有关.

正态分布

标准正态分布
分布 概率密度 分布函数 符号
\(X\)服从标准正态分布 $$\phi(x) = \frac{1}{\sqrt{2\pi}}e^{-\frac{x^2}{2}},~-\infty\lt x\lt +\infty$$ $$\Phi(x) = \frac{1}{\sqrt{2\pi}}\int_{-\infty}^x e^{-\frac{t^2}{2}}\mathrm{d}t,~-\infty\lt x\lt +\infty$$ $$X\sim N(0,1)$$
$$X\sim N(\mu, \sigma^2) \Rightarrow Y = \frac{X - \mu}{\sigma} \sim N(0,1)$$ $$\Phi(x) = 1 - \Phi(-x)$$ $$\Phi(0) = 0.5$$

指数分布

指数分布的无记忆性
$$X\sim e(\lambda) \Rightarrow P\{x\gt a+b|x\gt a\} = P\{x\gt b\}$$

当\(b\lt 0\)时：

$$P\{x\gt a+b|x\gt a\} = \frac{P\{x\gt a\}}{P\{x\gt a\}} = 1 = P\{x\gt b\}$$

当\(b\gt 0\)时：

$$\begin{align} P\{x\gt a+b|x\gt a\} &= \frac{P\{x\gt a+b\}}{P\{x\gt a\}}\\ &= \frac{\int_{a+b}^{+\infty}f(x)\mathrm{d}x}{\int_a^{+\infty}f(x)\mathrm{d}x}\\ &= \frac{(1-e^{-\lambda x})|_{a+b}^{+\infty}}{(1-e^{-\lambda x})|_{a}^{+\infty}}\\ &= \frac{1- [1 - e^{-(a+b)}]}{1 - (1 - e^{-a})}\\ &= \frac{e^{-\lambda(a+b)}}{e^{-\lambda a}}\\ &= e^{-\lambda b}\\ &= \int_b^{+\infty}f(x)\mathrm{d}x\\ &= P\{x\gt b\} \end{align}$$
