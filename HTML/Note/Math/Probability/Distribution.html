分布

离散型随机变量

数学期望： $$E(X) = \sum\limits_{k=1}^{\infty}x_kp_k$$ 方差： $$D(X) = \mathop{\sum}\limits_{k=1}^{\infty}[x_k - E(X)]^2p_k$$ 独立性： $$P\{X = x_i, Y = y_i\} = P\{X = x_i\}P\{Y = y_i\}$$ 最大似然估计： $$L(\theta) = \prod\limits_{i=1}^nP\{X=x_i\}$$

退化分布（单点分布）

$$ P\{X = C\} = 1 $$

二项分布

定义：
    n重伯努利试验中，事件A发生的概率为\(p\)，\(X\)表示n重伯努利试验中事件A发生的次数，则称\(X\)服从参数为\(n,p\)的二项分布，记作\(X\sim b(n,p)\) $$ P\{X = k\} = C_n^kp^k(1-p)^{n-k},k=0,1,2,\cdots,n $$ 数学期望： $$E(X) = np$$ 方差： $$D(X) = np(1-p)$$

（0-1）分布（两点分布）


    二项分布\(n = 1\)时的特殊情况，记作\(X\sim b(1,p)\)$$P\{X = k\} = p^k(1-p)^{(1-k)}, k=0,1$$

泊松分布

定义：
    若\(X\)服从参数为\(\lambda\)的泊松分布，记作\(X\sim P(\lambda)\) $$P\{X = k\} = \frac{\lambda^k e^{-\lambda}}{k!},~k = 0,1,2,\cdots,~\lambda>0 $$ 数学期望： $$\begin{align} E(X) &= \mathop{\sum}\limits_{k=0}^\infty k\cdot\frac{\lambda^ke^{-\lambda}}{k!}\\ &= \lambda e^{-\lambda}\mathop{\sum}\limits_{k=1}^\infty\frac{\lambda^{k-1}}{(k-1)!}\\ &= \lambda e^{-\lambda} e^{\lambda}\\ &= \lambda \end{align}$$ 方差： $$D(X) = \lambda$$ 最大似然估计：
    设\(x_1, x_2,\cdots, x_n\)为样本\(X_1, X_2, \cdots,X_n\)的观测值。 $$\begin{align} L(\lambda) &= \prod\limits_{i=1}^n\frac{\lambda^{x_i} e^{-\lambda}}{x_i!}\\ &= e^{-n\lambda}\frac{\lambda^{\sum\limits_{i=1}^nx_i}}{\prod\limits_{i=1}^nx_i!}\\ \therefore lnL(\lambda) &= -n\lambda + \sum\limits_{i=1}^nx_iln\lambda - \sum\limits_{i=1}^nln(x_i!)\\ 令 \frac{dlnL(\lambda)}{d\lambda} &= -n + \frac 1\lambda\sum\limits_{i=1}^nx_i = 0\\ \therefore 最大似然估计值：\hat \lambda &= \frac1n\sum\limits_{i=1}^nx_i = \bar x\\ \therefore 最大似然估计量：\hat \lambda &= \bar X \end{align}$$

连续型随机变量

数学期望: $$E(X) = \int_{-\infty}^{+\infty}xf(x)dx$$ 方差： $$D(X) = \int_{-\infty}^{+\infty}[x - E(X)]^2f(x)dx$$ 独立性： $$f(x,y) = f_X(x)f_Y(y)$$ 最大似然估计： $$L(\theta) = \prod\limits_{i=1}^nf(x_i;\theta)$$

正态分布

定义：
    若连续型随机变量\(X\)的概率密度为 $$ f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}},~-\infty< x < +\infty $$ 称\(X\)服从参数为\(\mu,\lambda\)的正态分布，记作\(X\sim N(\mu, \sigma^2)\)
数学期望： $$E(X) = \mu$$

均匀分布

定义：
    若连续型随机变量\(X\)的概率密度为 $$f(x)= \begin{align} \begin{cases} \frac{1}{b-a},~&a\leq x\leq b\\ 0,~&others \end{cases} \end{align}$$ 称\(X\)在区间[a,b]上服从均匀分布，记作\(X\sim U[a,b]\)。其分布函数为 $$F(x)= \begin{align} \begin{cases} 0,~&x< a\\ \frac{x-a}{b-a},~&a\leq x\leq b\\ 1,~&x>b \end{cases} \end{align}$$ 数学期望： $$\begin{align} E(X) &= \int_{a}^{b}\frac{x}{b-a}dx\\ &= \frac{x^2}{2(b-a)}|_a^b\\ &= \frac{a+b}{2} \end{align}$$ 方差： $$D(X) = \frac{(b-a)^2}{12}$$

指数分布

定义：
    若连续型随机变量\(X\)的概率密度为 $$f(x)= \begin{align} \begin{cases} \lambda e^{-\lambda x},~&x>0\\ 0,~&x\leq 0 \end{cases} \end{align},~\lambda > 0 $$ 称\(X\)服从参数为\(\lambda\)的指数分布，记作\(X\sim e(\lambda)\)。其分布函数为 $$F(x)= \begin{align} \begin{cases} 1-e^{-\lambda x},~&x>0\\ 0,~&x\leq 0 \end{cases} \end{align} $$ 数学期望： $$E(X) = \frac 1\lambda$$ 方差： $$D(X) = \frac{1}{\lambda^2}$$
