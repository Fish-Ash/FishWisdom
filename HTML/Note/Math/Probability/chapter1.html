第一节 随机事件

随机试验的特点

可重复性：试验可以在相同条件下重复进行.

可观测性：每次试验结果具有多种可能性，并能事先明确试验的所有可能结果.

随机性（不确定性）：试验前不能确定哪一个结果会出现.

随机试验（\(E\)）

样本点（\(\omega\)）

随机试验\(E\)的每一种可能结果.

样本空间\(\Omega, S\)

随机试验\(E\)所有样本点的集合.

事件（随机事件）

样本空间的子集，通常记为\(A,B,C,\cdots\).

事件\(A\)发生，即属于该事件的某一个样本点在随机事件中出现.

基本事件

由一个样本点组成的事件.

事件间的关系
名称 定义 符号 含义
子事件 若事件\(A\)的样本也属于事件\(B\)，则称\(A\)为\(B\)的子事件，也称事件\(B\)包含事件\(A\)或事件\(A\)包含于事件\(B\). $$A \subset B$$ 事件\(A\)发生必然导致事件\(B\)发生.
相等事件 若事件\(A\)与事件\(B\)互为子事件，则称事件\(A\)与事件\(B\)相等. $$A = B$$
和事件 事件\(A\)和\(B\)的全部样本点组成的集合称为事件\(A\)和\(B\)的和事件. $$A \cup B = \{\omega | \omega \in A~or~\omega \in B\}$$ 事件\(A\cup B\)表示事件\(A\)和\(B\)至少有一个发生.
积事件 既属于事件\(A\)又属于事件\(B\)的样本点组成的集合称为事件\(A\)与事件\(B\)的积事件. $$AB = A\cap B = \{\omega | \omega \in A~and~\omega \in B\}$$ 事件\(AB\)表示事件\(A\)和\(B\)同时发生.
互斥事件
（互不相容事件）		 若事件\(A\)与事件\(B\)没有公共的样本点，即\(AB = \emptyset\)，称事件\(A\)与\(B\)为互斥事件. 事件\(A\)与\(B\)不可能同时发生.
对立事件
（逆事件）		 若\(AB = \emptyset\)且\(A\cup B = \Omega\)，称事件\(A,B\)互为对立事件. \(A\)的对立事件记为\(\bar{A}\)
差事件 属于事件\(A\)而不属于事件\(B\)的样本点组成的集合称为事件\(A\)与事件\(B\)的差事件. $$A - B = A\bar{B} = A - AB = \{\omega | \omega \in A~and~\omega \notin B\}$$ 事件\(A\)发生而事件\(B\)不发生.
事件的运算规律
交换律 $$A\cup B = B\cup A, AB = BA$$
结合律 $$(A\cup B)\cup C = A\cup (B\cup C), (AB)C = A(BC)$$
分配律 $$(A\cup B)C = AC\cup BC, (AB)\cup C = (A\cup C)(B\cup C)$$
德摩根定律（对偶律） $$\overline{A\cup B} = \bar{A} \cap \bar{B}$$ $$\overline{A\cap B} = \bar{A} \cup \bar{B}$$ $$\overline{\mathop{\cup}\limits_{i=1}^{n} A_i} = \mathop{\cap}\limits_{i=1}^n \bar{A_i}$$ $$\overline{\mathop{\cap}\limits_{i=1}^{n} A_i} = \mathop{\cup}\limits_{i=1}^n \bar{A_i}$$
吸收率 \(A \subset B \Rightarrow A\cup B = B, AB = A\)

第二节 随机事件的概率

古典概型

古典概型

样本空间中的基本事件个数有限且具有等可能性的试验模型.

古典概率

设随机试验\(E\)的样本空间\(\Omega\)由\(n\)个样本点组成，每个样本等可能发生.

事件\(A\)由\(m\)个样本点组成，事件\(A\)发生的概率即为古典概率，记作\(P(A)\).

$$P(A) = \frac{m}{n}$$

排列组合公式

排列公式

从\(n\)个不同元素中任取\(k(1\leq k \leq n)\)个的不同排列总数.

$$A_n^k = \frac{n!}{(n-k)!} = n(n-1)\cdots(n-k+1)$$

\(k=n\)时称为全排列.

组合公式

从\(n\)个不同元素中任取\(k(1\leq k \leq n)\)个的不同组合总数.

$$C_n^k = \frac{A_n^k}{A_k^k} = \frac{n!}{k!(n-k)!} = \frac{n(n-1)\cdots(n-k+1)}{k(k-1)\cdots 1}$$
超几何分布概率公式

设有\(N\)件产品，其中\(D\)件次品，从中任取\(n\)件，其中恰有\(k(k\leq D)\)件次品的概率.

$$p = \frac{C_D^k C_{N-D}^{n-k}}{C_N^n}$$

从\(N\)件产品中取出\(n\)件，所有可能的取法有\(C_N^n\)种.

恰有\(k\)件次品，即从\(D\)件次品中取出\(k\)件，从\(N-D\)件合格品中取出\(n-k\)件，所有可能的取法有\(C_D^kC_{N-D}^{n-k}\)种.

几何概型

几何概型

若一个随机试验相当于从直线、平面或空间的某区域\(\Omega\)内任取一点，而所取的点落在\(\Omega\)中任意两个度量（长度、面积、体积）相等的子区域内是等可能的，则称此试验模型为几何概型.

几何概率

几何概型中，对任意有度量的子区域\(A \subset \Omega\)，定义事件“\(\Omega\)内任取一点落在\(A\)上的概率”为几何概率，记为\(P(A)\).

$$P(A) = \frac{A}{\Omega}$$
$$P(\bar{A}) = 1 - P(A)$$ $$A = \emptyset \Rightarrow P(A) = 0$$ $$P(A) = 0 \nRightarrow A = \emptyset$$ $$P(AB) \leq \min\{P(A), P(B)\}$$ 加法公式 $$P(A\cup B) = P(A) + P(B) - P(AB)$$ $$P(A\cup B\cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$$ 减法公式 $$P(A - B) = P(A\bar{B}) = P(A) - P(AB)$$

第三节 条件概率

条件概率

在事件\(A\)发生的条件下，事件\(B\)发生的概率.

$$P(B|A) = \frac{P(AB)}{P(A)}$$
完备事件组

设\(\Omega\)为某一随机试验\(E\)的样本空间，\(A_1, A_2, \cdots, A_n\)为\(E\)的一组事件，且满足：

  • \(A_iA_j = \emptyset,~i,j=1,2,\cdots,n,~i\neq j\)
  • \(A_1\cup A_2\cup\cdots\cup A_n = \Omega\)

称\(A_1, A_2, \cdots, A_n\)为\(\Omega\)的一个划分，或称为\(\Omega\)的一个完备事件组.

全概率公式

设\(\Omega\)为某随机试验\(E\)的样本空间，\(B\)为\(E\)内的任意事件，\(A_1, A_2, \cdots, A_n\)为\(\Omega\)的一个完备事件组，且\(P(A_i) \gt 0,~i=1,2,\cdots,n\)，则：

$$P(B) = \mathop{\sum}\limits_{i=1}^n P(A_i)P(B|A_i)$$
$$B = A_1B \cup A_2B \cup \cdots \cup A_nB$$ $$\begin{align} P(B) &= P(A_1B) + P(A_2B) + \cdots + P(A_nB)\\ &= \mathop{\sum}\limits_{i=1}^n P(A_iB)\\ &= \mathop{\sum}\limits_{i=1}^n P(A_i)P(B|A_i) \end{align}$$
理解

在两步模型中，求第二步情况\(B\)发生的概率，对第一步不同情况\(A_i\)下第二步\(B\)发生的概率进行加权求和.

贝叶斯公式

设\(\Omega\)为某随机试验\(E\)的样本空间，\(B\)为\(E\)内的任意事件，\(P(B)\gt0\)，\(A_1, A_2, \cdots, A_n\)为\(\Omega\)的一个完备事件组，且\(P(A_i) \gt 0,~i=1,2,\cdots,n\)，则在事件\(A\)发生的条件下，事件\(B\)发生的概率为：

$$P(A_i|B) = \frac{P(A_i)P(B|A_i)}{\sum\limits_{j=1}^n P(A_j)P(B|A_j)}$$
$$\begin{align} P(A_i|B) &= \frac{P(A_iB)}{P(B)}\\ &= \frac{P(A_i)P(B|A_i)}{\sum\limits_{j=1}^n P(A_j)P(B|A_j)} \end{align}$$
理解

在两步模型中，第二步情况\(B\)发生的条件下，第一步情况\(A_i\)发生的概率.

第四节 事件的独立性与伯努利检验

事件的独立性

若事件\(A, B\)满足

$$P(AB) = P(A)P(B)$$

则称事件\(A\)与事件\(B\)相互独立，简称为\(A,B\)独立.

\(A,B\)独立\(\Rightarrow\)事件\(A,\bar{B}\)，事件\(\bar{A}, B\)，事件\(\bar{A}, \bar{B}\)独立.

\(A,B\)独立\(\Rightarrow P(AB) = P(A)P(B)\)

$$\begin{align} P(A\bar{B}) &= P(A-AB)\\ &= P(A) - P(A)P(B)\\ &= P(A)[1 - P(B)]\\ &= P(A)P(\bar{B}) \end{align}$$ $$\begin{align} P(\bar{A}B) &= P(B-AB)\\ &= P(B) - P(A)P(B)\\ &= P(B)[1 - P(A)]\\ &= P(\bar{A})P(B) \end{align}$$ $$\begin{align} P(\bar{A}\bar{B}) &= P(\overline{A\cup B})\\ &= 1 - P(A\cup B)\\ &= 1 - [P(A) + P(B) - P(A)P(B)]\\ &= 1 - P(A) - P(B)[1 - P(A)]\\ &= P(\bar{A}) - P(B)P(\bar{A})\\ &= P(\bar{A})P(\bar{B}) \end{align}$$

必然事件\(\Omega\)与任何事件\(A\)独立，不可能事件\(\emptyset\)与任何事件\(A\)独立.

$$P(\Omega) = 1, P(\emptyset) = 0$$ $$P(A\Omega) = P(A) = P(A)P(\Omega)$$ $$P(A\emptyset) = P(\emptyset) = P(A)P(\emptyset)$$

\(P(A)\gt 0, P(B)\gt 0 \Rightarrow A,B\)互斥则不独立，独立则不互斥.

\(A,B\)互斥\(\Rightarrow P(AB) = P(\emptyset) = 0\).

\(A,B\)独立\(\Rightarrow P(AB) = P(A)P(B) \gt 0\).

多个事件的独立性

若事件\(A_1, A_2, A_3\)满足

$$P(A_1A_2) = P(A_1)P(A_2)$$ $$P(A_2A_3) = P(A_2)P(A_3)$$ $$P(A_1A_3) = P(A_1)P(A_3)$$

则称事件\(A_1,A_2,A_3\)两两相互独立.

若在此基础上还满足

$$P(A_1A_2A_3) = P(A_1)P(A_2)P(A_3)$$

则称事件\(A_1A_2A_3\)相互独立.

相互独立的事件一定是两两相互独立的，两两相互独立的事件不一定相互独立.

若事件\(A_1,A_2,\cdots,A_n\)相互独立，则其中任意\(m(1\leq m\leq n)\)个事件也相互独立.

若事件\(A_1,A_2,\cdots,A_n\)相互独立，则将其中任意\(m\)个事件换成它们的对立事件，所得的\(n\)个事件也相互独立.

伯努利试验

在试验\(E\)中，可能的结果只有两个，记为事件\(A\)发生与事件\(A\)不发生.

伯努利概型（\(n\)重伯努利试验）

在相同条件下独立重复\(n\)次伯努利试验的模型.

伯努利定理

在一次伯努利试验中，事件\(A\)发生的概率为\(p(0\lt p \lt 1)\)，则在\(n\)重伯努利试验中，事件\(A\)恰好发生\(k\)次的概率为\(P_n(k) = C_n^kp^k(1-p)^{n-k}(k=0,1,2,\cdots,n)\).
