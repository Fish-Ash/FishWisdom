第一节 点估计

1、矩估计

$$\mu_k = E(x^k)\Rightarrow \begin{cases} \hat{\mu} = \bar{X}\\ \hat{\sigma^2} = B_2 \end{cases}$$     即样本均值等于总体均值；样本方差等于总体二阶中心矩
    证明如下： $$\begin{cases} \mu_1 = E(X) = \mu\\ \mu_2 = E(X^2) = D(X) + E^2(X) = \sigma^2 + \mu^2 \end{cases}\\ $$ $$ \therefore \begin{cases} \begin{align} \mu&= \frac1n \mathop{\sum}\limits_{i=1}^nX_i\\ \sigma^2 + \mu^2&= \frac1n\mathop{\sum}\limits_{i=1}^nX_i^2, \end{align}\end{cases}\\ $$     解得矩估计量为： $$ \\ \hat \mu = \frac1n \mathop{\sum}\limits_{i=1}^nX_i = \bar X\\ \begin{align}\hat \sigma^2 &= \frac1n\mathop{\sum}\limits_{i=1}^nX_i^2 - \mu\\ &= \frac1n\mathop{\sum}\limits_{i=1}^nX_i^2 - \bar X^2\\ &= \frac1n\mathop{\sum}\limits_{i=1}^nX_i^2 - 2\bar X^2 + \bar X^2\\ &= \frac1n\mathop{\sum}\limits_{i=1}^nX_i^2 - \frac1n\mathop{\sum}\limits_{i=1}^n2\bar XX_i + \frac1n\mathop{\sum}\limits_{i=1}^n\bar X^2\\ &= \frac1n\mathop{\sum}\limits_{i=1}^n(X_i - \bar X)^2\\ &= B_2 \end{align} $$

\(X\sim U(a,b)\)： $$E(X) = \frac{a+b}{2} = \bar X\\ D(X) = \frac{(b-a)^2}{12} = B_2\\ \therefore \begin{cases} \hat a = \bar X - \sqrt{3B_2}\\ \hat b = \bar X + \sqrt{3B_2} \end{cases}$$

2、最大似然估计

离散型随机变量：

第二节 估计量的评价标准

无偏性：
    对\(\forall\theta\)，满足\(E(\hat \theta) = \theta\)，称\(\hat\theta\)为\(\theta\)的无偏估计。

有效性：
    若\(D(\hat\theta_1) < D(\hat\theta_2)\)，则称\(\hat\theta_1\)比\(\hat\theta_2\)更有效。

第三节 区间估计

置信度为\(1-\alpha\)的置信区间：
    满足\(P\{\hat{\theta_1}<\theta<\hat{\theta_2}\} = 1-\alpha\)的区间\((\hat{\theta_1}, \hat{\theta_2})\)
    表明置信区间\((\hat{\theta_1}, \hat{\theta_2})\)内包含\(\theta\)的概率为\(1-\alpha\)

第四节 正态总体均值的区间估计

1、\(\sigma\)已知： $$\bar X\sim N(\mu,\sigma/\sqrt n)$$ $$\frac{\bar X-\mu}{\sigma/\sqrt{n}}\sim N(0,1)$$ $$P\{|\frac{\bar X-\mu}{\sigma/\sqrt{n}}|\leq z_{\frac \alpha 2}\} = 1 - \alpha$$ 置信区间：\([\bar X - \frac{\sigma}{\sqrt n}z_{\frac \alpha 2}, \bar X + \frac{\sigma}{\sqrt n}z_{\frac \alpha 2}]\)
2、\(\sigma\)未知： $$P\{|\frac{\bar X-\mu}{S/\sqrt{n}}|\leq t_{\frac \alpha 2}(n-1)\} = 1 - \alpha$$ 置信区间：\([\bar X - \frac{S}{\sqrt n}t_{\frac \alpha 2}(n-1), \bar X + \frac{S}{\sqrt n}t_{\frac \alpha 2}(n-1)]\)
