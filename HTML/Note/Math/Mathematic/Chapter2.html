第二章 导数与微分

第二节 微分

微分的定义

设函数\(y = f(x)\)在点\(x_0\)的某邻域内有定义，\(\Delta x\)是\(x\)在\(x_0\)处的增量，\(x_0 + \Delta x\)在该邻域内。

曲线的增量 $$\Delta y = f(x_0 + \Delta x) - f(x_0)$$ 直线\(y = Ax + B\)的增量 $$\mathrm{d} y = f(x_0 + \Delta x) - f(x_0) = A\Delta x$$ 若 $$\lim\limits_{\Delta x \rightarrow 0}\frac{\Delta y - A\Delta x}{\Delta x} = 0\Leftrightarrow\Delta y = A\Delta x + o(\Delta x)$$ 则称函数\(y = f(x)\)在点\(x_0\)处可微.

\(\mathrm{d}y = \mathrm{d}f(x) = A\Delta x\)称为\(y=f(x)\)在\(x_0\)处的微分。

自变量\(x\)的增量\(\mathrm{d}x = \Delta x\)称为自变量的微分。

由于\(A = f'(x)\)，故函数\(y = f(x)\)的积分又可写为\(\mathrm{d}y = f'(x)\mathrm{d}x\)

$$可导\Leftrightarrow 可微$$

可微一定连续，连续不一定可微

微分近似计算

\(\Delta y = f(x_0 + \Delta x) - f(x) \approx \mathrm{d}y = f'(x_0)\Delta x\)

[注]\(\mathrm{d}(f(x) + C) = f'(x)\mathrm{d}x\)
