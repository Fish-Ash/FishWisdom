第二章 导数与微分

第一节 导数

导数的概念 $$f'(x_0) = \lim\limits_{\Delta x\rightarrow 0}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x} = \lim\limits_{x\rightarrow x_0}\frac{f(x)-f(x_0)}{x-x_0}$$
  • 左导数：\(f'_-(x_0) = \lim\limits_{\Delta x\rightarrow 0^-}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}\)
  • 右导数：\(f'_+(x_0) = \lim\limits_{\Delta x\rightarrow 0^+}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}\)
$$f'(x_0)\Leftrightarrow f'_-(x_0) = f'_+(x_0)$$ 注：
  • \(\lim\limits_{x\rightarrow x_0} f(x)\)与\(f(x_0)\)无关。
  • \(f'(x_0)\)与\(f(x_0)\)有关。
  • 可导一定连续，连续不一定可导。

导数的几何意义

\(f'(x_0)\)表示曲线\(y=f(x)\)在点\((x_0, f(x_0))\)处切线的斜率。

  • 切线方程：\(y - f(x_0) = f'(x_0)(x-x_0)\)
  • 法线方程：\(y - f(x_0) = -\frac{1}{f'(x_0)}(x-x_0)\)

计算

奇函数的导函数是偶函数

偶函数的导函数是奇函数

周期函数的导函数仍是周期函数

反函数的导数

设函数\(y = f(x)\)的导数为\(f'(x)\)，则其反函数\(x = g(y)\)的导数为\(g'(y) = \frac{1}{f'(x)}\)

参数方程的导数

设\(y = y(x)\)由参数方程\(\begin{cases}x = x(t)\\y = y(t)\end{cases}\)确定

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\frac{\mathrm{d}y}{\mathrm{d}t}}{\frac{\mathrm{d}x}{\mathrm{d}t}} = \frac{y'(t)}{x'(t)}, x'(t)\neq 0$$ $$\begin{align} \frac{\mathrm{d}^2y}{\mathrm{d}x^2} &= \frac{\mathrm{d}}{\mathrm{d}x}(\frac{\mathrm{d}y}{\mathrm{d}x})\\ &= \frac{\mathrm{d}}{\mathrm{d}t}(\frac{\mathrm{d}y}{\mathrm{d}x})\frac{\mathrm{d}t}{\mathrm{d}x}\\ &= \frac{\mathrm{d}}{\mathrm{d}t}[\frac{y'(t)}{x'(t)}]\frac{1}{x'(t)}\\ &= [\frac{y'(t)}{x'(t)}]'\frac{1}{x'(t)} \end{align}$$ 常用高阶导数公式
  • \((\sin x)^{(n)} = \sin(x + n\frac{\pi}{2})\)
  • \((\cos x)^{(n)} = \cos(x + n\frac{\pi}{2})\)
  • \((u\cdot v)^{(n)} = \sum\limits_{k=0}^{n}C_n^ku^{(k)}v^{(n-k)}\)（莱布尼茨公式）

题型

利用导数定义判定函数可导性

将导数存在问题转化为\(f'(x_0) = \lim\limits_{x\rightarrow x_0}\frac{f(x) - f(x_0)}{x - x_0}\)存在问题。

证明函数不等式

构造辅助函数，将函数不等式问题转化为函数零点问题

常用结论

设\(f(x) = \phi |x - a|\)，\(\phi(x)\)在\(x = a\)处连续，则\(f(x)\)在\(x=a\)处可导\(\Leftrightarrow \phi(a) = 0 \)

第二节 微分

微分的定义

设函数\(y = f(x)\)在点\(x_0\)的某邻域内有定义，\(\Delta x\)是\(x\)在\(x_0\)处的增量，\(x_0 + \Delta x\)在该临域内，若函数增量

$$\Delta y = f(x + \Delta x) - f(x_0)$$ 可表示为 $$\Delta y = A\Delta x + o(\Delta x)$$ 则称函数\(y = f(x)\)在点\(x_0\)处可微，其中\(A\)为常数。

\(\mathrm{d}y = \mathrm{d}f(x) = A\Delta x\)称为\(y=f(x)\)在\(x_0\)处的微分。

自变量\(x\)的增量\(\mathrm{d}x = \Delta x\)称为自变量的微分。

由于\(A = f'(x)\)，故函数\(y = f(x)\)的积分又可写为\(\mathrm{d}y = f'(x)\mathrm{d}x\)

$$可导\Leftrightarrow 可微$$

可微一定连续，连续不一定可微

微分近似计算

\(\Delta y = f(x_0 + \Delta x) - f(x) \approx \mathrm{d}y = f'(x_0)\Delta x\)
