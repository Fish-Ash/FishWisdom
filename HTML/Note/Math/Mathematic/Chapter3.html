第三章 微分中值定理与导数的应用

微分中值定理

费马引理

定义

设函数\(f(x)\)在\(x_0\)处可导，若函数\(f(x)\)在\(x_0\)处取得极值，则\(f'(x_0) = 0\).

描述

\(f(x)\)极值处导数为\(0\).

罗尔定理

若函数\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)上可导，\(f(a) = f(b)\)，则至少存在一点\(\xi \in (a,b)\)，使得\(f'(\xi) = 0\).

拉格朗日中值定理

定义

设\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导，则至少存在一点\(\xi\in(a,b)\)，使得\(f'(\xi) = \frac{f(b) - f(a)}{b-a}\)

形式2

设\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导，则至少存在一点\(\theta\in(0,1)\)，使得\(f'(a + \theta(b-a)) = \frac{f(b) - f(a)}{b-a}\)

$$a\lt \xi \lt b\Rightarrow 0\lt \xi-a \lt b-a \Rightarrow 0\lt \frac{\xi-a}{b-a}\lt 1$$

令\(\frac{\xi - a}{b - a} = \theta\)

$$\xi = a + \theta(b-1)$$

柯西中值定理

设\(f(x), F(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导，且\(F'(x) \neq 0\)，则至少存在一点\(\xi\in(a,b)\)，使得

$$\frac{f(b) - f(a)}{F(b) - F(a)} = \frac{f'(\xi)}{F'(\xi)}$$

泰勒中值定理

设\(f(x)\)在含有\(x_0\)的\((a,b)\)内有\(n + 1\)阶导数，则当\(x\in (a,b)\)时，

$$f(x) = f(x_0) + f'(x_0)(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \frac{f^{(n)}(x_0)}{n!}(x - x_0)^n + \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}$$ 其中\(\xi \in (x_0, x)\)，\(R(n) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x-x_0)^{n+1}\)称为拉格朗日余项。
