导数

导数 $$f'(x_0) = \lim\limits_{\Delta x\rightarrow 0}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x} = \lim\limits_{x\rightarrow x_0}\frac{f(x)-f(x_0)}{x-x_0}$$
  • 左导数：\(f'_-(x_0) = \lim\limits_{\Delta x\rightarrow 0^-}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}\)
  • 右导数：\(f'_+(x_0) = \lim\limits_{\Delta x\rightarrow 0^+}\frac{f(x_0+\Delta x)-f(x_0)}{\Delta x}\)
$$f'(x_0)\Leftrightarrow f'_-(x_0) = f'_+(x_0)$$

[注]可导一定连续，连续不一定可导。

二阶导数 $$f''(x_0) = \lim\limits_{x\rightarrow x_0}\frac{f'(x) - f'(x_0)}{x - x_0}$$
常用导数公式
三角函数导数
$$(\sin x)' = \cos x$$ $$(\cos x)' = -\sin x$$
$$(\arcsin x)' = \frac{1}{\sqrt{1 - x^2}}$$ $$(\arccos x)' = -\frac{1}{\sqrt{1-x^2}}$$
$$(\tan x)' = \sec^2x$$ $$(\cot x)' = -\csc^2x$$
$$(\arctan x)' = \frac{1}{1 + x^2}$$ $$(\arctan x)' = -\frac{1}{1 + x^2}$$
$$(\sec x)' = \sec x\tan x$$ $$(\csc x)' = -\csc x\cot x$$
指对函数导数
$$(a^x)' = a^x\ln x(a\gt 0, a\neq 1)$$ $$(e^x)' = e^x$$
$$(\log_a x)' = \frac{1}{x\ln a}(a\gt 0, a\neq 1)$$ $$(\ln x)' = \frac{1}{x}$$
其他函数导数
$$(C)' = 0$$ $$(x^a)' = ax^{a-1}$$

导数的几何意义

\(f'(x_0)\)表示曲线\(y=f(x)\)在点\((x_0, f(x_0))\)处切线的斜率。

求曲线的切线方程或法线方程

过曲线上一点

切线方程：\(y - f(x_0) = f'(x_0)(x-x_0)\)

法线方程：\(y - f(x_0) = -\frac{1}{f'(x_0)}(x-x_0)\)

过曲线外一点

设切线或法线经过曲线上\((x_0, f(x_0))\)，求出切线方程或法线方程.

将曲线外点带入方程求出\(x_0\).

求反函数的导数

设函数\(y = f(x)\)的导数为\(f'(x)\)，则其反函数\(x = g(y)\)的导数为

$$g'(y) = (f^{-1})'(y) = \frac{1}{f'(x)}$$

奇函数的导函数是偶函数

偶函数的导函数是奇函数

周期函数的导函数仍是周期函数

求参数方程的导数

设\(y = y(x)\)由参数方程\(\begin{cases}x = x(t)\\y = y(t)\end{cases}\)确定

一阶导数

$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{y'(t)}{x'(t)}$$
$$\frac{\mathrm{d}y}{\mathrm{d}x} = \frac{\frac{\mathrm{d}y}{\mathrm{d}t}}{\frac{\mathrm{d}x}{\mathrm{d}t}} = \frac{y'(t)}{x'(t)}, x'(t)\neq 0$$

二阶导数

$$\frac{\mathrm{d}^2y}{\mathrm{d}x^2} = [\frac{y'(t)}{x'(t)}]'\frac{1}{x'(t)}$$
$$\begin{align} \frac{\mathrm{d}^2y}{\mathrm{d}x^2} &= \frac{\mathrm{d}}{\mathrm{d}x}(\frac{\mathrm{d}y}{\mathrm{d}x})\\ &= \frac{\mathrm{d}}{\mathrm{d}t}(\frac{\mathrm{d}y}{\mathrm{d}x})\frac{\mathrm{d}t}{\mathrm{d}x}\\ &= \frac{\mathrm{d}}{\mathrm{d}t}[\frac{y'(t)}{x'(t)}]\frac{1}{x'(t)}\\ &= [\frac{y'(t)}{x'(t)}]'\frac{1}{x'(t)} \end{align}$$

[注]使用对数求导法时，应将\(y(x)\)代回\(y'\)的表达式中表示.
