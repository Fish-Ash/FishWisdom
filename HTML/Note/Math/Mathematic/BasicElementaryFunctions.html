基本初等函数

幂函数

指数函数

基本形式 $$y = a^x(a\gt 0,a\neq 1)$$ 性质 $$a^k b^k = (ab)^k$$ $$a^m a^n = a^{m+n}$$ $$\frac{a^m}{a^n} = a^{m-n}$$ $$(a^m)^n = a^{mn}$$
$$x^{\frac{1}{2k}} = \sqrt[2k]{x}$$ $$x\geq 0$$
$$x^{-k} = \frac{1}{x^k}$$ $$x\neq0$$
幂指互换： $$x^n = e^{n\ln x}$$

对数函数

基本形式 $$y = \log_ax(a\gt 0, a\neq 1)$$
\(a\) 对数的底数
\(x\) 对数的真数
指对互换公式 $$y = log_ax \Leftrightarrow x = a^y$$ 换底公式 $$\log_ab = \frac{\log_cb}{\log_ca}$$ 性质 $$\log_am + \log_an = \log_a(mn)$$

$$\begin{cases} x = log_am\\ y = log_an \end{cases}\Rightarrow \begin{cases} m = a^x\\ n = a^y \end{cases}\\ mn = a^{x+y} \Rightarrow log_amn = x+y = log_am + log_an $$

$$\log_am - \log_an = \log_a\frac{m}{n}$$

$$\begin{cases} x = log_am\\ y = log_an \end{cases}\Rightarrow \begin{cases} m = a^x\\ n = a^y \end{cases}\\ \frac{m}{n} = a^{x-y} \Rightarrow log_a\ = x-y = log_am - log_an $$

$$\log_{a^m}k^n = \frac{n}{m}\log_ak$$

$$ \begin{cases} x = log_{a^m}k^n\\ y = log_ak \end{cases}\Rightarrow \begin{cases} k^n = a^{mx}\\ k = a^y \end{cases}\\ a^{ny} = a^{mx}\Rightarrow x = \frac{n}{m}y\\ \therefore \log_{a^m}k^n = \frac{n}{m}\log_ak $$

$$a^{log_ak} = k$$

$$ log_ak = log_ak\\ k = a^{log_ak} $$
