函数极限

化简

（1）四则运算：

$$\lim f(x)g(x) = \lim f(x)\lim g(x)$$ $$\lim [f(x) + g(x)] = \lim f(x) + \lim g(x)$$

（2）等价无穷小：

$$1 - \cos x = \frac{1}{2}x^2$$ $$\ln(1 + x) \sim e^x - 1 \sim x$$ $$x - \ln(1 + x) \sim \frac12 x^2$$ $$a^x - 1 \sim x\ln a$$ $$(1 + x)^\alpha -1 = \alpha x$$

（3）洛必达法则

\(\frac{0}{0}\)型 \(\frac{\infty}{\infty}\)型

$$\lim\frac{f(x)}{g(x)} = \lim\frac{f'(x)}{g'(x)}$$

（5）\(1^\infty\)型极限

$$ \begin{cases} \lim \alpha(x) = 0\\ \lim \beta(x) = \infty\\ \end{cases}\Rightarrow \lim [1 + \alpha(x)]^{\beta(x)} = e^{\lim \alpha(x)\beta(x)} $$

（6）\(\lim\limits_{x\rightarrow 0}a^{\frac{1}{x}}\)

$$\lim\limits_{x\rightarrow 0^+}a^{\frac{1}{x}} = a^{+\infty} = \infty$$ $$\lim\limits_{x\rightarrow 0^-}a^{\frac{1}{x}} = (\frac{1}{a})^{+\infty} = 0$$ 无穷小量阶的比较

\(\alpha(x)\)为\(\beta(x)\)的：

  • 高阶无穷小：\(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = 0\) \(\alpha(x) = o[\beta(x)]\)
  • 低阶无穷小：\(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = \infty\)
  • 同阶无穷小：\(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = c\neq 0\)
  • 等价无穷小：\(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = 1\) \(\alpha(x)\sim \beta(x)(x\rightarrow x_0)\)
  • \(k\)阶无穷小：\(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta^k(x)} = c\neq 0\)
