第五章 定积分

定积分定义

定义

\(\int_0^1f(x)\mathrm{d}x = \lim\limits_{n\rightarrow \infty}\sum\limits_{k=1}^nf(\frac{k}{n})\frac{1}{n}\)

\(\int_a^b f(x)\mathrm{d}x\)

几何意义

函数\(y = f(x)\)在区间\([a,b]\)上与\(x\)轴围成区域的面积。

在\(x\)轴上方的定积分面积为正值，在\(x\)轴下方的定积分面积为负值，且可线性相加。

注意：

积分的上下限并无大小关系，下限\(a\)可以大于上限\(b\)，当换元的时候也无需交换位置。

举例来说，若换元后\(a'=b, b'=a\)，则\(\int_a^b f[\phi(x)]\mathrm{d}x = \int_{b}^{a} f(t)\mathrm{d}t\)

定积分存在的充分条件

\(f(x)\)在\([a,b]\)上连续\(\Rightarrow \int_a^b f(x)\mathrm{d}x\)存在

\(f(x)\)在\([a,b]\)上有界，且只有有限个间断点\(\Rightarrow \int_a^b f(x)\mathrm{d}x\)存在

\(f(x)\)在\([a,b]\)上只有有限个第一类间断点\(\Rightarrow \int_a^b f(x)\mathrm{d}x\)存在

定积分存在的必要条件、

\(\int_a^b f(x)\mathrm{d}x\)存在\(\Rightarrow[a,b]\)有界且\(f(x)\)在\([a,b]\)上有界

定积分性质

不等式性质

在\([a,b]\)上\(f(x)\leq g(x) \Rightarrow \int_a^bf(x)\mathrm{d}x\leq \int_a^bg(x)\mathrm{d}x\)

若\(f(x)\)在\([a,b]\)上最大值为\(M\)，最小值为\(m \Rightarrow m(b-a)\leq \int_a^bf(x)\mathrm{d}x\leq M(b-a)\)

\(|\int_a^b f(x)\mathrm{d}x|\leq \int_a^b|f(x)|\mathrm{d}x\)

积分中值定理

若\(f(x)\)在\([a,b]\)内连续，则至少存在一点\(\xi \in [a,b]\)使得\(\int_a^bf(x)\mathrm{d}x=f(\xi)(b-a)\)

若\(f(x)\)在\([a,b]\)内连续，则至少存在一点\(\xi \in (a,b)\)使得\(\int_a^bf(x)\mathrm{d}x=f(\xi)(b-a)\)

设\(f(x), g(x)\)在\([a,b]\)内连续，且\(g(x)\)不变号，则至少存在一点\(\xi \in [a,b]\)，使得

$$\int_a^b f(x)g(x) \mathrm{d}x = f(\xi)\int_a^bg(x)\mathrm{d}x$$

\(f(x)\)在\([a,b]\)上连续\(\Rightarrow f(x)\)在\([a,b]\)上存在最大值\(M\)与最小值\(m\)使得\(m\leq f(x)\leq M\)

\(g(x)\)不变号\(\Rightarrow\)不妨设\(g(x)\gt 0 \Rightarrow mg(x)\leq f(x)g(x)\leq Mg(x)\)

定积分的计算

牛顿-莱布尼茨公式 $$\int_a^b f(x)\mathrm{d}x = F(x)|_a^b = F(b) -F(a)$$ 换元积分法

设f(x)在区间\(U\)上连续，且\(x = \phi(t)\)满足：

\(\phi(\alpha) = a, \phi(\beta) = b\)

\(\phi(t)\)在\([\alpha, \beta]\)上具有连续导数

$$\int_a^bf(x)\mathrm{d}x = \int_\alpha^\beta f[\phi(t)]\phi'(t)\mathrm{d}t$$ 奇偶函数积分

奇函数：\(\int_{-a}^a f(x)\mathrm{d}x = 0\)

偶函数：\(\int_{-a}^a f(x)\mathrm{d}x = 2\int_{-a}^a f(x)\mathrm{d}x\)

周期函数积分 $$\int_a^{a+nT}f(x)\mathrm{d}x = n\int_b^{b+T}f(x)\mathrm{d}x$$ 沃利斯公式 $$\int_0^{\frac\pi2}\sin^n x\mathrm{d}x = \int_0^\frac\pi2 \cos^nx \mathrm{d}x = \begin{cases} \frac{\pi}{2}\cdot \frac{1}{2}\cdots\frac{n-3}{n-2}\cdot\frac{n-1}{n}, n为偶数\\ \frac{2}{3}\cdot\frac{4}{5}\cdots\frac{n-3}{n-2}\cdot\frac{n-1}{n},n为奇数\\ \end{cases} $$ $$\int_0^\frac\pi2 f(\sin x)\mathrm{d}x = \int_0^\frac\pi2 f(\cos x)\mathrm{d}x$$ $$\int_0^\pi xf(\sin x)\mathrm{d}x = \frac{\pi}{2}\int_0^\pi f(\sin x)\mathrm{d}x$$

积分上限函数

变上限积分\(\int_a^x f(t)\mathrm{d}t\)是上限\(x\)的函数，称为积分上限函数。

$$\int_a^x f(t)\mathrm{d}t = F(t)|_a^x = F(x) - F(a)$$ 性质

若\(f(x)\)在\([-l,l]\)上连续：

  • \(f(x)\)为奇函数\(\Rightarrow \int_a^xf(t)\mathrm{d}t\)为偶函数
  • \(f(x)\)为偶函数\(\Rightarrow \int_0^xf(t)\mathrm{d}t\)为奇函数

若\(f(x)\)为奇函数\(\Rightarrow F(x)\)为偶函数

$$\int_a^xf(t)\mathrm{d}t = F(x) - F(a)$$ $$\therefore F(-x) - F(a) = F(x) - F(a)$$

故\(\int_a^xf(t)\mathrm{d}t\)为偶函数

若\(f(x)\)为偶函数\(\Rightarrow F(x)\)为奇函数

$$\int_0^xf(t)\mathrm{d}t = F(x) - F(0) = F(x)$$ $$\therefore F(-x)= -F(x)$$

故\(\int_0^xf(t)\mathrm{d}t\)为奇函数

积分上限函数的导数 $$\begin{align} (\int_{a(x)}^{b(x)} f(t)\mathrm{d}t)' &= (F[b(x)] - F[a(x)])'\\ &= f'[b(x)]b'(x) - f'[a(x)]a'(x) \end{align}$$ $$\begin{align} (\int_{a(x)}^{b(x)} f(t)g(x)\mathrm{d}t)' &= (g(x)\int_{a(x)}^{b(x)} f(t)\mathrm{d}t)'\\ &= g'(x)\int_{a(x)}^{b(x)} f(t)\mathrm{d}t + g(x)(\int_{a(x)}^{b(x)} f(t)\mathrm{d}t)' \end{align}$$ $$\int_{a(x)}^{b(x)}f[g(x,t)]\mathrm{d}t \overset{\text{u=g(x,t)}}{=\!=\!=\!=} \int_{\alpha(u)}^{\beta(u)} f(u)\mathrm{d}u $$

反常积分

设\(f(x)\)在\([a,+\infty)\)上连续，若\(\lim\limits_{t\rightarrow +\infty}\int_a^t f(x) \mathrm{d}x\)存在，则称此极限为\(f(x)\)在\([a,+\infty)\)上的反常积分，记作：

$$\int_a^{+\infty}f(x)\mathrm{d}x = \lim\limits_{t\rightarrow +\infty}\int_a^t f(x) \mathrm{d}x$$

设\(f(x)\)在\((-\infty,b]\)上连续，若 \(\lim\limits_{t\rightarrow -\infty}\int_t^b f(x) \mathrm{d}x\)存在，则称此极限为\(f(x)\)在\((-\infty,b]\)上的反常积分，记作：

$$\int_{-\infty}^bf(x)\mathrm{d}x = \lim\limits_{t\rightarrow -\infty}\int_t^b f(x) \mathrm{d}x$$

设\(f(x)\)在\((-\infty, +\infty)\)上连续，若\(\int_{-\infty}^a f(x)\mathrm{d} x\)与\(\int_a^{+\infty} f(x)\mathrm{d}x\)都存在，则:

$$\int_{-\infty}^{+\infty} f(x) \mathrm{d}x = \int_{-\infty}^a f(x) \mathrm{d}x + \int_a^{+\infty} f(x) \mathrm{d}x $$

若反常积分存在，称其收敛；若反常积分不存在，称其发散。

比较判别法（积分形式）

设\(f(x), g(x)\)在\([a,+\infty)\)上连续，且\(0\leq f(x)\leq g(x)\)：

  • \(\int_a^{+\infty}g(x)\mathrm{d}x\)收敛\(\Rightarrow \int_a^{+\infty}f(x)\mathrm{d}x\)收敛
  • \(\int_a^{+\infty}f(x)\mathrm{d}x\)发散\(\Rightarrow \int_a^{+\infty}g(x)\mathrm{d}x\)发散
比较判别法（极限形式）

设\(f(x), g(x)\)在\([a,+\infty)\)上连续，且\(\lim\limits_{x\rightarrow +\infty}\frac{f(x)}{g(x)} = \lambda\)：

  • \(\lambda \neq 0\)：\(\int_a^{+\infty}f(x)\mathrm{d}x\)与\(\int_a^{+\infty}g(x)\mathrm{d}x\)同敛散
  • \(\lambda = 0\)：\(\int_a^{+\infty}g(x)\mathrm{d}x\)收敛\(\Rightarrow\int_a^{+\infty}f(x)\mathrm{d}x\)收敛
  • \(\lambda = 0\)：\(\int_a^{+\infty}g(x)\mathrm{d}x\)发散\(\Rightarrow\int_a^{+\infty}f(x)\mathrm{d}x\)发散
常用结论 $$\int_a^{+\infty} \frac{1}{x^p}\mathrm{d}x \begin{cases} 收敛, p\gt 1\\ 发散, p\leq 1 \end{cases}$$
$$\int_a^{+\infty} \frac{1}{x^p}\mathrm{d}x = \frac{x^{1-p}}{1-p}\Bigg|_a^{+\infty}$$

当\(1-p\lt 0\)，即\(p\gt 1\)时：\(\int_a^{+\infty} \frac{1}{x^p}\mathrm{d}x\)收敛；

当\(1-p\geq 0\)，即\(p\leq 1\)时：\(\int_a^{+\infty} \frac{1}{x^p}\mathrm{d}x\)发散。

无界函数的反常积分

设\(f(x)\)在\((a,b]\)上连续，\(\lim\limits_{x\rightarrow a^+}f(x) = \infty\)，则称\(x=a\)为\(f(x)\)的瑕点。

若极限\(\lim\limits_{t\rightarrow a^+}\int_t^b f(x)\mathrm{d}x\)存在，称此极限为\(f(x)\)在\([a,b]\)上的反常积分，记为：

$$\int_a^b f(x) \mathrm{d}x = \lim\limits_{t\rightarrow a^+}\int_t^b f(x)\mathrm{d}x$$

此时称\(\int_a^b f(x)\mathrm{d}x\)收敛；若上述极限不存在，则称其发散。

设\(f(x)\)在\([a,b)\)上连续，\(\lim\limits_{x\rightarrow b^-}f(x) = \infty\)，称\(x=b\)是\(f(x)\)的瑕点。

若极限\(\lim\limits_{t\rightarrow b^-}\int_a^t f(x)\mathrm{d}x\)存在，称此极限为\(f(x)\)在\([a,b]\)上的反常积分，记为：

$$\int_a^b f(x) \mathrm{d}x = \lim\limits_{t\rightarrow b^-}\int_a^t f(x)\mathrm{d}x$$

此时称\(\int_a^b f(x)\mathrm{d}x\)收敛；若上述极限不存在，则称其发散。

设\(f(x)\)在\([a,b]\)上除点\(C(a\lt C\lt b)\)外连续，\(\lim\limits_{x\rightarrow c}f(x)\mathrm{d}x = \infty\)，称\(c\)为\(f(x)\)的瑕点。

若反常积分\(\int_a^c f(x)\mathrm{d}x, \int_c^b f(x)\mathrm{d}x\)均收敛，则反常积分\(\int_a^b f(x)\mathrm{d}x\)也收敛，且\(\int_a^b f(x)\mathrm{d}x = \int_a^c f(x)\mathrm{d}x + \int_c^b f(x)\mathrm{d}x\)

比较判别法

设\(f(x), g(x)\)在\((a,b]\)上连续，且\(0\leq f(x)\leq g(x), x=a\)是\(f(x)\)和\(g(x)\)的瑕点。

  • \(\int_a^b g(x)\mathrm{d}x\)收敛\(\Rightarrow \int_a^b f(x)\mathrm{d}x\)收敛。
  • \(\int_a^b f(x)\mathrm{d}x\)发散\(\Rightarrow \int_a^b g(x)\mathrm{d}x\)发散。
比较判别法（极限形式）

设\(f(x), g(x)\)在\((a,b]\)上连续，\(x=a\)是\(f(x)\)和\(g(x)\)的瑕点，且\(\lim\limits_{x\rightarrow a^+}\frac{f(x)}{g(x)} = \lambda\)（有限或无穷）

  • \(\lambda \neq 0\)时，\(\int_a^b f(x)\mathrm{d}x\)与\(\int_a^b g(x)\mathrm{d}x\)同收敛。
  • \(\lambda = 0\)时，\(\int_a^b g(x)\mathrm{d}x\)收敛\(\Rightarrow\int_a^b g(x)\mathrm{d}x\)收敛。
常用结论 $$\int_a^b \frac{1}{(x-a)^p}\mathrm{d}x \begin{cases} p\lt 1,收敛\\ p\geq 1,发散 \end{cases} $$ $$\int_a^b \frac{1}{(b-x)^p}\mathrm{d}x \begin{cases} p\lt 1,收敛\\ p\geq 1,发散 \end{cases} $$
