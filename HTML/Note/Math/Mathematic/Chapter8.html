多元函数的基本概念

二元函数

$$z = f(x,y)$$
  • 自变量：\(x,y\)
  • 因变量：\(z\)
  • 定义域：\((x,y)\in \mathbb{D}\)
  • 值域：\(z \in \mathbb{R}\)
几何意义

表示空间中的曲面.

二元函数的极限

设\(z = f(x,y)\)在区域\(D\)上有定义，点\(P_0(x_0, y_0)\in D\)或在区域\(D\)的边界上，对于\(\forall \epsilon\gt 0, \exists \delta \gt 0\)，使得\(P(x,y)\in D\)，且\(0\lt \sqrt{(x-x_0)^2 + (y-y_0)^2}\lt \delta\)时，都有\(|f(x,y) - A|\lt \epsilon\)

称\(f(x,y)\)在\((x,y)\rightarrow (x_0, y_0)\)时极限值为\(A\)，记为

$$\lim\limits_{(x,y)\rightarrow(x_0,y_0)}f(x,y)=A$$

[注]二元函数极限路径任意

判断\(\lim\limits_{(x,y)\rightarrow(x_0,y_0)}f(x,y)\)是否存在

特殊路径法

令\(y = kf(x)\)，代入极限得函数\(g\).

若\(g\)与\(k\)的取值有关，则说明极限不存在.

[注]预设\(kf(x)\)时，尽量将原函数中的\(x\)消去.

求二元极限的方法

夹逼定理 $$\lim f(x) = 0 \Leftrightarrow \lim |f(x)| = 0$$

判断\(f(x,y)\)在点\((x_0,y_0)\)处是否连续

判断\(\lim\limits_{(x,y)\rightarrow(x_0,y_0)}f(x,y)\)是否等于\(f(x_0, y_0)\)

多元函数连续性

设多元函数\(z = f(x,y)\)在区域\(D\)上有定义，点\(P_0(x_0, y_0) \in D\)，若\(\lim\limits_{(x,y)\rightarrow (x_0, y_0)}f(x,y) = f(x_0,y_0)\)，称\(f(x,y)\)在\(P_0(x_0, y_0)\)处连续。

若\(f(x,y)\)在区域\(D\)上每个点\((x,y)\)都连续，称\(z = f(x,y)\)在区域\(D\)上连续。

多元连续函数的性质

多元连续函数的四则运算均为连续函数

多元连续函数的复合函数仍为连续函数

多元初等函数在其定义域内连续

最值定理：有界闭区域\(D\)上的多元连续函数必有最大值和最小值

偏导数

设\(z = f(x,y)\)在点\(P_0(x_0,y_0)\)的某邻域内有定义，若

$$\lim\limits_{\Delta x \rightarrow 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x} = \lim\limits_{x\rightarrow x_0}\frac{f(x, y_0) - f(x_0, y_0)}{x-x_0}$$ 存在，称其为\(f(x,y)\)在\(P_0(x_0,y_0)\)处关于\(x\)的偏导数，记为\(\frac{\partial f}{\partial x}|_{(x_0, y_0)}\)或\(f'_x(x_0, y_0)\).

同理，\(f(x,y)\)在\(P_0(x_0,y_0)\)处关于\(y\)的偏导数记为\(\frac{\partial f}{\partial y}|_{(x_0, y_0)}\)或\(f'_y(x_0, y_0)\).

二元函数偏导数的几何意义

设\(M(x_0, y_0, z_0)\)为曲面\(z = f(x,y)\)上一点，过\(M\)作平面\(y = y_0\)与曲面\(z = f(x,y)\)相交，其交线为平面\(y = y_0\)上的曲线\(z = f(x,y_0)\)，即\(\begin{cases}z = f(x,y_0)\\y = y_0\end{cases}\).

则\(f'_x(x_0,y_0)\)表示该曲线在点\(M\)处切线的斜率。

判断\(f(x,y)\)在\((x_0,y_0)\)处是否可求偏导

若\(\lim\limits_{\Delta x \rightarrow 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x} = \lim\limits_{x\rightarrow x_0}\frac{f(x, y_0) - f(x_0, y_0)}{x-x_0}\)存在，则\(f(x,y)\)在\((x_0,y_0)\)处有关于\(x\)的偏导数\(f'_x\).

若\(\lim\limits_{\Delta y \rightarrow 0} \frac{f(x_0, y_0 + \Delta y) - f(x_0, y_0)}{\Delta y} = \lim\limits_{y\rightarrow y_0}\frac{f(x_0, y) - f(x_0, y_0)}{y-y_0}\)存在，则\(f(x,y)\)在\((x_0,y_0)\)处有关于\(y\)的偏导数\(f'_y\).

求偏导数方法

定义法

$$f'_x(x_0, y_0) = \lim\limits_{\Delta x \rightarrow 0} \frac{f(x_0 + \Delta x, y_0) - f(x_0, y_0)}{\Delta x} = \lim\limits_{x\rightarrow x_0}\frac{f(x, y_0) - f(x_0, y_0)}{x-x_0}$$ $$f'_y(x_0, y_0) = \lim\limits_{\Delta y \rightarrow 0} \frac{f(x_0, y_0 + \Delta y) - f(x_0, y_0)}{\Delta y} = \lim\limits_{y\rightarrow y_0}\frac{f(x_0, y) - f(x_0, y_0)}{y-y_0}$$

一般函数求偏导数\(f'_x(x,y)\)

将\(y\)视为常数，对\(x\)求导数.

一般函数求偏导数\(f'_x(x, y_0)\)

将\(y_0\)代入\(f(x, y)\)得到\(f(x, y_0)\)，将\(f(x,y_0)\)对\(x\)求导得\(f'_x(x,y_0)\).

复合函数求偏导

设\(z = f(u,v)\)在\((u,v)\)处有连续的偏导数，\(u = u(x,y), v = v(x,y)\)在\((x,y)\)处偏导数存在，则\(z = f[u(x,y), v(x,y)]\)的偏导数为：

$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial x} = f'_u\frac{\partial u}{\partial x} + f'_v\frac{\partial v}{\partial x}$$

求二阶导数时对\(f'_u\)的处理：

$$\frac{\partial f'_u}{\partial x} = \frac{\partial f'_u}{\partial u}\frac{\partial u}{\partial x} + \frac{f'_u}{\partial v}\frac{\partial v}{\partial x}$$

二阶偏导数

$$f''_{xy} = \frac{\partial^2 f}{\partial x\partial y} = \frac{\partial}{\partial y}(\frac{\partial f}{\partial x})$$

定理：若\(f(x,y)\)的两个二阶混合导数\(\frac{\partial^2 f}{\partial x \partial y}, \frac{\partial^2 f}{\partial y \partial x}\)在区域\(D\)内连续，则\(\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f}{\partial y \partial x}\)

全微分法求隐函数偏导数

$$\mathrm{d}z = \frac{\partial z}{\partial x}\mathrm{d}x + \frac{\partial z}{\partial y}\mathrm{d}y$$

全微分

若二元函数\(z = f(x,y)\)在\(x_0, y_0\)处的全增量

$$\Delta z = f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)$$ 可以表示为 $$\Delta z = A\Delta x + B\Delta y + o(\rho)$$ 其中\(A,B\)与\(\Delta x,\Delta y\)无关，\(\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}\)

称\(z = f(x,y)\)处可微，而\(A\Delta x + B\Delta y\)称为\(z = f(x,y)\)在\((x_0,y_0)\)处的全微分，记为

$$\mathrm{d}z = A\Delta x + B\Delta y = = \frac{\partial z}{\partial x}\mathrm{d}x + \frac{\partial z}{\partial y}\mathrm{d}y$$
可微的等价条件

对于\(z = f(x,y)\)，记

$$\Delta f = f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)$$ $$\mathrm{d}f = f'_x(x_0, y_0)\Delta x + f'_y(x_0, y_0)\Delta y$$ $$\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$$

若\(\lim\limits_{\rho \rightarrow 0}\frac{\Delta f - \mathrm{d}f}{\rho} = 0\)，则称\(z = f(x,y)\)在\((x_0, y_0)\)处可微.

可微的充分条件

若\(z = f(x,y)\)的偏导数\(\frac{\partial z}{\partial x}, \frac{\partial z}{\partial y}\)在\((x_0, y_0)\)处连续，则\(z = f(x, y)\)在\((x_0, y_0)\)处可微.

偏导数连续\(\Rightarrow\)可微

可微\(\Rightarrow\)连续\(\Rightarrow\)极限存在

可微\(\Rightarrow\)偏导数存在

对比一元函数：可导\(\Leftrightarrow\)可微\(\Rightarrow\)连续\(\Rightarrow\)极限存在

判断是否可微

不连续\(\Rightarrow\)不可微

全微分求法

一般函数求全微分

$$\mathrm{d}z = \frac{\partial z}{\partial x}\mathrm{d}x + \frac{\partial z}{\partial y}\mathrm{d}y$$

隐函数求全微分

$$f(x,y,z) = 0 \Rightarrow \mathrm{d}f(x,y,z) = 0$$

化简后提出\(\mathrm{d}z\)，得

$$\mathrm{d}z = \frac{\partial z}{\partial x}\mathrm{d}x + \frac{\partial z}{\partial y}\mathrm{d}y$$

复合函数微分法

全微分形式不变性

设\(z = f(u,v), u = u(x,y), v = v(x,y)\)都有连续的一阶偏导数，则\(z = f[u(x,y), v(x,y)]\)的全微分为：

$$\begin{align} \mathrm{d}z &= \frac{\partial z}{\partial x}\mathrm{d}x + \frac{\partial z}{\partial y}\mathrm{d}y\\ &= (\frac{\partial z}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial x})\mathrm{d}x + (\frac{\partial z}{\partial u}\frac{\partial u}{\partial y} + \frac{\partial z}{\partial v}\frac{\partial v}{\partial y})\mathrm{d}y\\ &= \frac{\partial z}{\partial u}(\frac{\partial u}{\partial x}\mathrm{d}x + \frac{\partial u}{\partial y}\mathrm{d}y) + \frac{\partial z}{\partial v}(\frac{\partial v}{\partial x}\mathrm{d}x + \frac{\partial v}{\partial y}\mathrm{d}y)\\ &= \frac{\partial z}{\partial u}\mathrm{d}u + \frac{\partial z}{\partial v}\mathrm{d}v \end{align}$$ 常见的多元函数求微分 $$u = f(x,y,z), z = z(x,y)$$ $$\frac{\partial u}{\partial x} = \frac{\partial f}{\partial x} + \frac{\partial f}{\partial z}\frac{\partial z}{\partial x}$$ $$u = f(x,y,z), y = y(x), z = z(x)$$ $$\frac{\mathrm{d}u}{\mathrm{d}x} = \frac{\partial f}{\partial x} + \frac{\partial f}{\partial y}\frac{\mathrm{d}y}{\mathrm{d}x} + \frac{\partial f}{\partial z}\frac{\mathrm{d}z}{\mathrm{d}x}$$

隐函数求导法

\(F(x,y) = 0\Rightarrow y(x)\)

$$\frac{\mathrm{d}y}{\mathrm{d}x} = -\frac{F'_x}{F'_y}$$

\(F(x,y,z)\Rightarrow z = z(x,y)\)

$$\frac{\partial z}{\partial x} = -\frac{F'_x}{F'_z}$$ $$\frac{\partial z}{\partial y} = -\frac{F'_y}{F'_z}$$

\(\begin{cases}F(x,y,u,v)\\G(x,y,u,v)\end{cases}\Rightarrow u(x,y), v(x,y)\)

$$\begin{cases} F'_x + F'_u\frac{\partial u}{\partial x} + F'_v\frac{\partial v}{\partial x} = 0\\ G'_x + G'_u\frac{\partial u}{\partial x} + G'_v\frac{\partial v}{\partial x} = 0\\ \end{cases}\Rightarrow \frac{\partial u}{\partial x}, \frac{\partial v}{\partial x}$$ $$\begin{cases} F'_y + F'_u\frac{\partial u}{\partial y} + F'_v\frac{\partial v}{\partial y} = 0\\ G'_y + G'_u\frac{\partial u}{\partial y} + G'_v\frac{\partial v}{\partial y} = 0\\ \end{cases}\Rightarrow \frac{\partial u}{\partial y}, \frac{\partial v}{\partial y}$$

多元函数的极值与最值

无约束条件极值

极值

设\(z = f(x,y)\)在点\(P_0(x_0,y_0)\)某邻域内有定义，若对该邻域内的任意点\(P(x,y)\)有

  • \(f(x,y)\leq f(x_0,y_0)\)，称\(f(x_0,y_0)\)为极大值，\((x_0, y_0)\)为极大值点。
  • \(f(x,y)\geq f(x_0,y_0)\)，称\(f(x_0,y_0)\)为极小值，\((x_0, y_0)\)为极小值点。
极值的必要条件

\((x_0,y_0)\)为极值点\(\Rightarrow f'_x(x_0,y_0) = 0, f'_y(x_0,y_0) = 0\)

求极值方法

$$\begin{cases}f'_x = 0\\f'_y = 0\end{cases} \Rightarrow (x_0,y_0)$$

\((x_0, y_0)\)即为驻点.

求\(A = f''_{xx}(x_0,y_0), B = f''_{xy}(x_0,y_0), C = f''_{yy}(x_0,y_0)\)

$$AC - B^2 \gt 0$$ $$A \gt 0$$ \(f(x_0,y_0)\)为极小值
$$A \lt 0$$ \(f(x_0,y_0)\)为极大值
$$AC - B^2 = 0$$ 无法判断\(f(x_0,y_0)\)是否为极值
$$AC - B^2 \lt 0$$ \(f_(x_0,y_0)\)不是极值

条件极值解法

求\(z = f(x.y)\)在条件\(\phi(x,y) = 0\)下的极值

构造拉格朗日函数

$$F(x,y,\lambda) = f(x,y) + \lambda\phi(x,y)$$

解方程

$$\begin{cases} F'_x = f'_x + \lambda \phi'_x = 0\\ F'_y = f'_y + \lambda \phi'_y = 0\\ F'_\lambda = \phi = 0 \end{cases}$$

方程的解\((x_0,y_0)\)即为极值点.

求\(z = f(x,y,z)\)在条件\(\phi_1(x,y,z) = 0, \phi_2(x,y,z) = 0\)下的极值

构造拉格朗日函数

$$F(x,y,z,\lambda_1, \lambda_2) = f(x,y,z) + \lambda_1\phi_1(x,y,z) + \lambda_2\phi_2(x,y,z)$$

解方程

$$\begin{cases} F'_x = f'_x + \lambda_1\phi'_{1x} + \lambda_2\phi'_{2x} = 0\\ F'_y = f'_y + \lambda_1\phi'_{1y} + \lambda_2\phi'_{2y} = 0\\ F'_z = f'_z + \lambda_1\phi'_{1z} + \lambda_2\phi'_{2z} = 0\\ F'_{\lambda_1} = \phi_1 = 0\\ F'_{\lambda_2} = \phi_2 = 0 \end{cases}$$

方程的解\((x_0,y_0,z_0)\)即为极值点.

最大值和最小值

最值

设\(z = f(x,y)\)在区域\(D\)内有定义，点\(P_0(x_0, y_0)\in D\)，对\(\forall (x,y)\in D\)，有

  • \(f(x,y)\geq f(x_0,y_0)\)，称\(f(x_0,y_0)\)为最小值，\((x_0,y_0)\)为最小值点。
  • \(f(x,y)\leq f(x_0,y_0)\)，称\(f(x_0,y_0)\)为最大值，\((x_0,y_0)\)为最大值点。

多元函数最值求法

先求无条件极值，并保留区域\(D\)内的极值点.

再求满足区域\(D\)边界条件的有条件极值.

比较极值大小，得出最大值与最小值.

方向导数

方向导数

设\(l\)为\(xOy\)平面上以\(P_0(x_0,y_0)\)为始点的一条射线，\(\mathbf{e}_l = (\cos\alpha, \cos\beta)\)为与\(l\)同方向的单位向量.

射线\(l\)的参数方程为：

$$\begin{cases} x = x_0 + t\cos\alpha\\ y = y_0 + t\cos\beta \end{cases}(t\geq 0)$$

函数\(f(x,y)\)在点\((x_0, y_0)\)处沿方向\(l\)的方向导数为：

$$\frac{\partial f}{\partial l}\bigg|_{(x_0, y_0)} = \lim\limits_{t\rightarrow 0^+}\frac{f(x_0 + t\cos\alpha, y_0 + t\cos\beta) - f(x_0, y_0)}{t}$$

方向导数\(\frac{\partial f}{\partial l}\bigg|_{(x_0, y_0)}\)就是函数\(f(x,y)\)在点\(P_0(x_0, y_0)\)沿方向\(l\)的变化率.

判断方向导数是否存在及方向导数的计算

若\(f(x,y)\)在点\(P_0(x_0, y_0)\)可微分，则该函数在该点沿任一方向\(l\)的方向导数存在，且

$$\frac{\partial f}{\partial l}\bigg|_{(x_0, y_0)} = f_x(x_0,y_0)\cos\alpha + f_y(x_0,y_0)\cos\beta$$

其中\(\cos\alpha\)与\(\cos\beta\)为方向\(l\)的方向余弦.

梯度 $$\nabla f = \mathbf{grad} f = \frac{\partial f}{\partial x_1}\mathbf{i}_1 + \frac{\partial f}{\partial x_2}\mathbf{i}_2 + \cdots + \frac{\partial f}{\partial x_n}\mathbf{i}_n$$

\(\nabla\)称为Nabla算子（向量微分算子）
