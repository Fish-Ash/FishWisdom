第一章 映射与函数

函数的基本概念

函数

若对于\(x\in D\)，按照一定的法则总有唯一确定的数\(y\)与之对应，则称\(f\)为\(x\)的函数，记作

$$y = f(x),x\in D$$
  • 自变量：\(x\)
  • 因变量：\(y\)
  • 函数定义域：\(D_f = D\)
  • 函数值域：\(R_f = f(D) = \{y|y = f(x),x\in D\}\)
判断两个函数是否为同一函数
  • 对应法则是否相同
  • 定义域是否相同
反函数

设函数\(y = f(x)\)定义域为\(D\)，值域为\(R_y\)。

若对任意\(y\in R_y\)，有唯一确定的\(x\in D\)，使得\(y = f(x)\)，则记\(x = f^{-1}(y)\)为函数\(y = f(x)\)的反函数。

  • 不是每个函数都有反函数。
  • 单调函数一定有反函数，反函数不一定是单调函数。
  • \(f^{-1}[f(x)] = f[f^{-1}(x)] = x\)

常见函数的性质

单调性
  • 单调递增：区间\(D\)内，\(\forall x_1 \lt x_2\)，有\(f(x_1) \lt f_(x_2)\)
  • 单调不减：区间\(D\)内，\(\forall x_1 \lt x_2\)，有\(f(x_1) \leq f_(x_2)\)
  • 单调递减：区间\(D\)内，\(\forall x_1 \lt x_2\)，有\(f(x_1) \gt f_(x_2)\)
  • 单调不增：区间\(D\)内，\(\forall x_1 \lt x_2\)，有\(f(x_1) \geq f_(x_2)\)
奇偶性
  • 奇函数：定义在\((-l, l)\)上的函数，满足\(f(-x) = -f(x)\)
  • 偶函数：定义在\((-l, l)\)上的函数，满足\(f(-x) = f(x)\)
  • 既奇又偶函数：\(f(x) = 0\)
  • 非奇非偶函数
$$ \begin{align} 奇+奇 = 奇 &&偶+偶=偶 &&&奇+偶=?\\ 奇\times奇 = 偶 && 偶\times 偶 = 偶 &&&奇\times 偶 = 奇 \end{align} $$

奇函数\(f(0) = 0\)

周期性

若\(\exists T>0\)，对\(\forall x\)，恒有

$$f(x + T) = f(x)$$

则称\(f(x)\)为以\(T\)为周期的周期函数。

若存在使上式成立的最小正数\(T\)，则其被称为最小正周期。

不是所有的周期函数都有最小正周期，例如常数函数、迪利克雷函数。

若\(f(x)\)周期为\(T\)，则\(f(ax+b)\)周期为\(\frac{T}{|a|}\)

有界性

对于\(\forall x\in D\)

  • 若\(\exists M \gt 0\)， 使得\(|f(x)|\leq M\)，则称\(f(x)\)有界。
  • 若\(\exists M \gt 0\)， 使得\(f(x)\leq M\)，则称\(f(x)\)有上界。
  • 若\(\exists M \gt 0\)， 使得\(f(x)\geq -M\)，则称\(f(x)\)有下界。

\(\frac 1x > -n(n>0) \Rightarrow x\in(-\infty, -\frac 1n)\cup(0, +\infty)\)

第三节 函数的极限

（一）自变量趋于有限值时函数的极限

设函数\(f(x)\)在点\(x_0\)的某去心邻域内有定义。若\(\exists\)常数\(A\)，对于\(\forall\epsilon \gt 0\)，总\(\exists \delta \gt 0\)，使得当\(|x - x_0|\lt \delta\)时，\(|f(x) - A| \lt \epsilon\)

常数\(A\)称为函数\(f(x)\)当\(x\rightarrow x_0\)时的极限，记作\(\mathop{lim}\limits_{x\rightarrow x_0}f(x) = A\)或\(f(x)\rightarrow A(x\rightarrow x_0)\)

  • 左极限：\(\lim\limits_{x\rightarrow x_{0^-}}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists \delta \gt 0, 当x_0 - \delta \lt x \lt x_0时，|f(x) - A|\lt \epsilon\)
  • 右极限：\(\lim\limits_{x\rightarrow x_{0^+}}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists \delta \gt 0, 当x_0 \lt x \lt x_0 + \delta时，|f(x) - A|\lt \epsilon\)
$$\lim\limits_{x\rightarrow x_0}f(x) = A \Leftrightarrow \lim\limits_{x\rightarrow x_{0^-}}f(x) = \lim\limits_{x\rightarrow x_{0^+}}f(x) = A$$ （二）自变量趋于无穷大时函数的极限 $$\lim\limits_{x\rightarrow -\infty}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists X \gt 0, 当x\lt -X时，|f(x) - A|\lt \epsilon$$ $$\lim\limits_{x\rightarrow +\infty}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists X \gt 0, 当x\gt X时，|f(x) - A|\lt \epsilon$$ $$\lim\limits_{x\rightarrow \infty}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists X \gt 0, 当|x|\gt X时，|f(x) - A|\lt \epsilon \Leftrightarrow \lim\limits_{x\rightarrow -\infty}f(x) = \lim\limits_{x\rightarrow +\infty}f(x) = A$$ 常用函数的极限 $$\lim\limits_{x\rightarrow\infty}\sqrt[n]{n} = 1$$ $$\lim\limits_{n\rightarrow \infty}\sqrt[n]{x_1^n + x_2^n + \cdots + x_m^n} = \max\{x_1, x_2, \cdots x_m\}, x_i\gt0$$

不妨设\(0\lt x_1 \leq x_2 \leq \cdots \leq x_m\)

$$x_m^n \leq x_1^n + x_2^n +\cdots + x_m^n \leq mx_m^n$$ $$\sqrt[n]{x_m^n} \leq \sqrt[n]{x_1^n + x_2^n +\cdots + x_m^n} \leq \sqrt[n]{mx_m^n}$$ $$ \begin{cases} \lim\limits_{n\rightarrow \infty}\sqrt[n]{x_m^n} = x_m\\ \lim\limits_{n\rightarrow \infty}\sqrt[n]{mx_m^n} = x_m \end{cases}\Rightarrow \lim\limits_{n\rightarrow \infty}\sqrt[n]{x_1^n + x_2^n + \cdots + x_m^n} = x_m $$
$$\lim\limits_{x\rightarrow 0^+}(1 + \frac{1}{x})^x = \lim\limits_{x\rightarrow +\infty}(1 + x)^{\frac{1}{x}} = 1$$ 极限唯一性

若\(\lim f(x)\)存在，那么此极限唯一

局部有界性

若\(\lim\limits_{x\rightarrow x_0}f(x) = A\)，则\(\exists M\gt 0, \delta \gt 0\)，使得当\(0\lt |x-x_0|\lt \delta\)时，\(|f(x)|\leq M\)

函数极限与无穷小的关系 $$\lim f(x) = A \Leftrightarrow f(x) = A + \alpha(x)$$ 其中\(\alpha(x)\)为高阶无穷小，\(\lim \alpha(x) = 0\)

第四节 极限存在准则

夹逼准则

数列形式

若\(\exists N\in\mathbb{N}_+\)，当\(n \geq N\)时，\(y_n\leq x_n\leq z_n\)，且\(\lim\limits_{n\rightarrow \infty}y_n = \lim\limits_{n\rightarrow\infty}z = a\)

\(\Rightarrow \lim\limits_{n\rightarrow \infty}x_n = a\)

函数形式

当\(0\lt|x-x_0|\lt \delta\)时，\(g(x)\leq f(x)\leq h(x)\)，且\(\lim g(x) = \lim h(x) = A\)

\(\Rightarrow \lim f(x) = A\)

单调有界原理 $$x_{n+1}\geq x_n, x_n\lt M\Rightarrow \exists \lim\limits_{n\rightarrow \infty}x_n$$ $$x_{n+1}\leq x_n, x_n\gt M\Rightarrow \exists \lim\limits_{n\rightarrow \infty}x_n$$

极限的四则运算法则

设\(\lim f(x) = A, \lim g(x) = B\)
$$\lim[f(x) \pm g(x)] = \lim f(x) \pm \lim g(x) = A \pm B$$
$$\lim[f(x)g(x)] = \lim f(x) \cdot \lim g(x) = A \cdot B$$
$$\lim \frac{f(x)}{g(x)} = \frac{\lim f(x)}{\lim g(x)} = \frac AB, B\neq 0$$
推论 $$极限存在+极限存在=极限存在$$ $$极限存在+极限不存在=极限不存在$$ $$极限不存在+极限不存在=未知情况$$ $$\lim f(x) = A \neq 0 \Rightarrow \lim f(x)g(x) = A\lim g(x)$$ $$\lim f(x)g(x) = A, \lim g(x) = \infty \Rightarrow \lim f(x) = 0$$ $$\lim \frac{f(x)}{g(x)} = A, \lim g(x) = 0 \Rightarrow \lim f(x) = 0$$ $$\lim \frac{f(x)}{g(x)} = B \neq 0, \lim f(x) =0 \Rightarrow \lim g(x) = 0$$

保号性

若\(\lim\limits_{x\rightarrow x_0}f(x) = A\)，且\(A\gt 0\)（或\(A \lt 0\)），那么\(\exists \delta\gt 0\)，使得当\(0\lt |x-x_0|\lt \delta\)时，\(|f(x)|\gt 0\)（或\(f(x)\lt 0\)）

无穷小量

若\(\lim\limits_{x\rightarrow x_0}f(x) = 0\)，则称\(f(x)\)为\(x\rightarrow x_0\)时的无穷小量。

无穷小量阶的比较

在相同的自变量变化趋势下，设\(\lim\alpha = 0, \lim\beta = 0\)，且\(\beta(x)\neq 0\)

  • \(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = 0\)，称\(\alpha(x)\)为\(\beta(x)\)的高阶无穷小，记为\(\alpha(x) = o[\beta(x)]\)
  • \(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = \infty\)，称 \(\alpha(x)\)为\(\beta(x)\)的低阶无穷小
  • \(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = c\neq 0\)，称\(\alpha(x)\)为\(\beta(x)\)的同阶无穷小
  • \(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta(x)} = 1\)，称\(\alpha(x)\)与\(\beta(x)\)等价，记为\(\alpha(x)\sim \beta(x)(x\rightarrow x_0)\)
  • \(\lim\limits_{x\rightarrow x_0} \frac{\alpha(x)}{\beta^k(x)} = c\neq 0\)，称\(\alpha(x)\)为\(\beta(x)\)的\(k\)阶无穷小
无穷小的运算 $$o(x^m) + o(x^n) = o(x^k), k = \min\{m,n\}$$ $$o(x^m)\cdot o(x^n) = o(x^{m+n})$$ $$x^m\cdot o(x^n) = o(x^{m+n})$$ $$o(kx^n) = k\cdot o(x^n) = o(x^n), k\neq 0$$ 无穷小的性质

有限个无穷小的和仍是无穷小

有限个无穷小的乘积仍然是无穷小

无穷小与有界量的乘积仍然是无穷小

等价无穷小具有相同的函数值与导数值

常见等价无穷小 $$x - \ln(1 + x) \sim \frac12 x^2$$ $$\ln(1 + x) \sim x$$ $$e^x - 1 \sim x$$ $$a^x - 1 \sim x\ln a$$ $$(1 + x)^\alpha -1 = \alpha x$$

因此无界变量包含无穷大量。

函数连续性

连续

设\(f(x)\)在\(x_0\)某个邻域内有定义，若\(\lim\limits_{x\rightarrow x_0}f(x) = f(x_0)\)，称\(f(x_0)\)在\(x_0\)处连续。

设\(f(x)\)在\(x_0\)某个邻域内有定义，若\(\lim\limits_{\Delta x\rightarrow 0} y = \lim\limits_{\Delta x\rightarrow 0}[f(x_0 + \Delta x) - f(x_0)] = 0\)，称\(f(x_0)\)在\(x_0\)处连续。

  • \(f(x)\)在\(x_0\)处左连续：\(\lim\limits_{x\rightarrow x_0^-}f(x) = f(x_0)\)
  • \(f(x)\)在\(x_0\)处右连续：\(\lim\limits_{x\rightarrow x_0^+}f(x) = f(x_0)\)
$$\lim\limits_{x\rightarrow x_0} \Leftrightarrow \lim\limits_{x\rightarrow x_0^-} = \lim\limits_{x\rightarrow x_0^+} = f(x_0)$$
  • \(f(x)\)在\((a,b)\)内连续：\(f(x)\)在\((a,b)\)内每点都连续。
  • \(f(x)\)在\([a,b]\)内连续：\(f(x)\)在\((a,b)\)内每点都连续，且在\(x=a\)处右连续，在\(x=b\)处左连续。
间断

设\(f(x)\)在\(x_0\)某个去心邻域内有定义，但\(f(x)\)在\(x_0\)处不连续，称\(x_0\)为\(f(x)\)的间断点。

$$\lim\limits_{x\rightarrow x_0}f(x) \neq f(x_0)$$ 间断点
  • 第一类间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x)\)与\(\lim\limits_{x\rightarrow x_0^+}f(x)\)均存在
    • 可去间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x) = \lim\limits_{x\rightarrow x_0^+}f(x) \neq f(x_0)\)
    • 跳跃间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x) \neq \lim\limits_{x\rightarrow x_0^+}f(x)\)
  • 第二类间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x)\)与\(\lim\limits_{x\rightarrow x_0^+}f(x)\)至少一个不存在
    • 无穷间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x) = \infty\)或\(\lim\limits_{x\rightarrow x_0^+}f(x) = \infty\)
    • 振荡间断点：\(\lim\limits_{x\rightarrow x_0}f(x)\)振荡
找间断点方法

找函数定义域内无意义的点。

  • \(\frac1x\)：\(x\neq 0\)
  • \(\ln|x|\)：\(x\neq 0\)

连续性的性质及运算

设函数\(f(x)\)和\(g(x)\)在\(x_0\)处连续，则\(f(x) \pm g(x), f(x)\cdot g(x), \frac{f(x)}{g(x)}(g(x)\neq 0)\)在\(x_0\)处连续。

设\(u = \phi(x)\)在\(x_0\)处连续，且\(\phi(x_0) = u_0\)，而\(y = f(u)\)在点\(u_0\)处连续，则复合函数\(y = f(\phi(x))\)在\(x_0\)处连续。

初等函数在其定义域内均为连续的。

闭区间上连续函数的性质

最值定理：\(f(x)\)在\([a,b]\)上连续\(\Rightarrow\)\(f(x)\)在\([a,b]\)上有最大值与最小值。

有界定理：\(f(x)\)在\([a,b]\)上连续\(\Rightarrow\)\(f(x)\)在\([a,b]\)上有界。

介值定理：\(f(x)\)在\([a,b]\)上连续且\(f(a)\neq f(b)\)，对于\(\forall f(a) \leq c \leq f(b)\)，至少存在一点\(\xi \in (a,b)\)，使得\(f(\xi) = c\)。

介值定理推论：\(f(x)\)在\([a,b]\)上连续，有最大值\(M\)与最小值\(m\)，对于\(\forall m \leq c \leq M\)，至少存在一点\(\xi \in (a,b)\)，使得\(f(\xi) = c\)。

零点定理：\(f(x)\)在\([a,b]\)上连续且\(f(a)\cdot f(b) \lt 0\)，至少存在一点\(\xi \in (a,b)\)，使得\(f(\xi) = 0\)。
