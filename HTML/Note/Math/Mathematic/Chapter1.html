函数的极限

（一）自变量趋于有限值时函数的极限

设函数\(f(x)\)在点\(x_0\)的某去心邻域内有定义。若\(\exists\)常数\(A\)，对于\(\forall\epsilon \gt 0\)，总\(\exists \delta \gt 0\)，使得当\(|x - x_0|\lt \delta\)时，\(|f(x) - A| \lt \epsilon\)

常数\(A\)称为函数\(f(x)\)当\(x\rightarrow x_0\)时的极限，记作\(\mathop{lim}\limits_{x\rightarrow x_0}f(x) = A\)或\(f(x)\rightarrow A(x\rightarrow x_0)\)

  • 左极限：\(\lim\limits_{x\rightarrow x_{0^-}}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists \delta \gt 0, 当x_0 - \delta \lt x \lt x_0时，|f(x) - A|\lt \epsilon\)
  • 右极限：\(\lim\limits_{x\rightarrow x_{0^+}}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists \delta \gt 0, 当x_0 \lt x \lt x_0 + \delta时，|f(x) - A|\lt \epsilon\)
$$\lim\limits_{x\rightarrow x_0}f(x) = A \Leftrightarrow \lim\limits_{x\rightarrow x_{0^-}}f(x) = \lim\limits_{x\rightarrow x_{0^+}}f(x) = A$$ （二）自变量趋于无穷大时函数的极限 $$\lim\limits_{x\rightarrow -\infty}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists X \gt 0, 当x\lt -X时，|f(x) - A|\lt \epsilon$$ $$\lim\limits_{x\rightarrow +\infty}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists X \gt 0, 当x\gt X时，|f(x) - A|\lt \epsilon$$ $$\lim\limits_{x\rightarrow \infty}f(x) = A \Leftrightarrow \forall \epsilon\gt 0, \exists X \gt 0, 当|x|\gt X时，|f(x) - A|\lt \epsilon \Leftrightarrow \lim\limits_{x\rightarrow -\infty}f(x) = \lim\limits_{x\rightarrow +\infty}f(x) = A$$ 常用函数的极限 极限唯一性

若\(\lim f(x)\)存在，那么此极限唯一

局部有界性

若\(\lim\limits_{x\rightarrow x_0}f(x) = A\)，则\(\exists M\gt 0, \delta \gt 0\)，使得当\(0\lt |x-x_0|\lt \delta\)时，\(|f(x)|\leq M\)

函数极限与无穷小的关系 $$\lim f(x) = A \Leftrightarrow f(x) = A + \alpha(x)$$ 其中\(\alpha(x)\)为高阶无穷小，\(\lim \alpha(x) = 0\)

第四节 极限存在准则

夹逼准则

数列形式

若\(\exists N\in\mathbb{N}_+\)，当\(n \geq N\)时，\(y_n\leq x_n\leq z_n\)，且\(\lim\limits_{n\rightarrow \infty}y_n = \lim\limits_{n\rightarrow\infty}z = a\)

\(\Rightarrow \lim\limits_{n\rightarrow \infty}x_n = a\)

函数形式

当\(0\lt|x-x_0|\lt \delta\)时，\(g(x)\leq f(x)\leq h(x)\)，且\(\lim g(x) = \lim h(x) = A\)

\(\Rightarrow \lim f(x) = A\)

单调有界原理 $$x_{n+1}\geq x_n, x_n\lt M\Rightarrow \exists \lim\limits_{n\rightarrow \infty}x_n$$ $$x_{n+1}\leq x_n, x_n\gt M\Rightarrow \exists \lim\limits_{n\rightarrow \infty}x_n$$

极限的四则运算法则

设\(\lim f(x) = A, \lim g(x) = B\)
$$\lim[f(x) \pm g(x)] = \lim f(x) \pm \lim g(x) = A \pm B$$
$$\lim[f(x)g(x)] = \lim f(x) \cdot \lim g(x) = A \cdot B$$
$$\lim \frac{f(x)}{g(x)} = \frac{\lim f(x)}{\lim g(x)} = \frac AB, B\neq 0$$
推论 $$极限存在+极限存在=极限存在$$ $$极限存在+极限不存在=极限不存在$$ $$极限不存在+极限不存在=未知情况$$ $$\lim f(x) = A \neq 0 \Rightarrow \lim f(x)g(x) = A\lim g(x)$$ $$\lim f(x)g(x) = A, \lim g(x) = \infty \Rightarrow \lim f(x) = 0$$ $$\lim \frac{f(x)}{g(x)} = A, \lim g(x) = 0 \Rightarrow \lim f(x) = 0$$ $$\lim \frac{f(x)}{g(x)} = B \neq 0, \lim f(x) =0 \Rightarrow \lim g(x) = 0$$

函数极限的性质

保号性

若\(\lim\limits_{x\rightarrow x_0}f(x) = A\)，且\(A\gt 0\)（或\(A \lt 0\)），那么\(\exists \delta\gt 0\)，使得当\(0\lt |x-x_0|\lt \delta\)时，\(|f(x)|\gt 0\)（或\(f(x)\lt 0\)）

有界性
\(\{x\}\)收敛\(\Rightarrow \{x\}\)有界 \(\lim\limits_{x\rightarrow x_0} f(x)\)存在\(\Rightarrow f(x)\)在\(x_0\)某去心邻域内局部有界
保号性

设\(\lim\limits_{n\rightarrow \infty}x_n = A\)

\(A\gt 0 \Rightarrow \exists N\gt 0\)，\(n\gt N\)时，\(x_n \gt 0\)

\(A\lt 0 \Rightarrow \exists N\gt 0\)，\(n\gt N\)时，\(x_n \lt 0\)

\(\exists N\gt 0\)，\(n\gt N\)时，\(x_n \geq 0 \Rightarrow A\geq 0\)

\(\exists N\gt 0\)，\(n\gt N\)时，\(x_n \leq 0 \Rightarrow A\leq 0\)

函数连续性

连续

设\(f(x)\)在\(x_0\)某个邻域内有定义，若\(\lim\limits_{x\rightarrow x_0}f(x) = f(x_0)\)，称\(f(x_0)\)在\(x_0\)处连续。

设\(f(x)\)在\(x_0\)某个邻域内有定义，若\(\lim\limits_{\Delta x\rightarrow 0} y = \lim\limits_{\Delta x\rightarrow 0}[f(x_0 + \Delta x) - f(x_0)] = 0\)，称\(f(x_0)\)在\(x_0\)处连续。

  • \(f(x)\)在\(x_0\)处左连续：\(\lim\limits_{x\rightarrow x_0^-}f(x) = f(x_0)\)
  • \(f(x)\)在\(x_0\)处右连续：\(\lim\limits_{x\rightarrow x_0^+}f(x) = f(x_0)\)
$$\lim\limits_{x\rightarrow x_0} \Leftrightarrow \lim\limits_{x\rightarrow x_0^-} = \lim\limits_{x\rightarrow x_0^+} = f(x_0)$$
  • \(f(x)\)在\((a,b)\)内连续：\(f(x)\)在\((a,b)\)内每点都连续。
  • \(f(x)\)在\([a,b]\)内连续：\(f(x)\)在\((a,b)\)内每点都连续，且在\(x=a\)处右连续，在\(x=b\)处左连续。

连续性的性质及运算

设函数\(f(x)\)和\(g(x)\)在\(x_0\)处连续，则\(f(x) \pm g(x), f(x)\cdot g(x), \frac{f(x)}{g(x)}(g(x)\neq 0)\)在\(x_0\)处连续。

设\(u = \phi(x)\)在\(x_0\)处连续，且\(\phi(x_0) = u_0\)，而\(y = f(u)\)在点\(u_0\)处连续，则复合函数\(y = f(\phi(x))\)在\(x_0\)处连续。

初等函数在其定义域内均为连续的。

闭区间上连续函数的性质

最值定理：\(f(x)\)在\([a,b]\)上连续\(\Rightarrow\)\(f(x)\)在\([a,b]\)上有最大值与最小值。

有界定理：\(f(x)\)在\([a,b]\)上连续\(\Rightarrow\)\(f(x)\)在\([a,b]\)上有界。

介值定理：\(f(x)\)在\([a,b]\)上连续且\(f(a)\neq f(b)\)，对于\(\forall f(a) \leq c \leq f(b)\)，至少存在一点\(\xi \in (a,b)\)，使得\(f(\xi) = c\)。

介值定理推论：\(f(x)\)在\([a,b]\)上连续，有最大值\(M\)与最小值\(m\)，对于\(\forall m \leq c \leq M\)，至少存在一点\(\xi \in (a,b)\)，使得\(f(\xi) = c\)。

零点定理：\(f(x)\)在\([a,b]\)上连续且\(f(a)\cdot f(b) \lt 0\)，至少存在一点\(\xi \in (a,b)\)，使得\(f(\xi) = 0\)。
