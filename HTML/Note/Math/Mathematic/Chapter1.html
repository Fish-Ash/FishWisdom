第一章 映射与函数

函数的基本概念

函数

若对于\(x\in D\)，按照一定的法则总有唯一确定的数\(y\)与之对应，则称\(f\)为\(x\)的函数，记作

$$y = f(x),x\in D$$
  • 自变量：\(x\)
  • 因变量：\(y\)
  • 函数定义域：\(D_f = D\)
  • 函数值域：\(R_f = f(D) = \{y|y = f(x),x\in D\}\)
判断两个函数是否为同一函数
  • 对应法则是否相同
  • 定义域是否相同
反函数

设函数\(y = f(x)\)定义域为\(D\)，值域为\(R_y\)。

若对任意\(y\in R_y\)，有唯一确定的\(x\in D\)，使得\(y = f(x)\)，则记\(x = f^{-1}(y)\)为函数\(y = f(x)\)的反函数。

  • 不是每个函数都有反函数。
  • 单调函数一定有反函数，反函数不一定是单调函数。
  • \(f^{-1}[f(x)] = f[f^{-1}(x)] = x\)

第二节 数列的极限

数列极限

设\(\{x_n\}\)为一数列，若存在常数\(a\)，对\(\forall\epsilon\gt 0\)，总\(\exists N \in \mathbb{N}_+\)，使得当\(n>N\)时，\(x_n - a \lt \epsilon\)都成立。

则称\(\{x_n\}\)收敛于极限\(a\)，记作\(\mathop{lim}\limits_{n\rightarrow \infty} = a\)或\(x_n \rightarrow a(n\rightarrow \infty)\)

题型1：判断数列\(\{x_n\}\)是否收敛，若收敛求出极限

注意：
  1. 若含\((-1)^n\)项，判断另一个因子是否趋于0

题型2：证明数列\(\{x_n\}\)的极限为\(a\)

  1. 由\(|x_n - a|<\epsilon\)推出\(n > f(\epsilon)\)
  2. \(\forall\)取一大于\(f(\epsilon)\)的正整数\(N\)，当\(n>N>f(\epsilon)\)时，\(|x_n - a|< \epsilon\)一定成立
技巧：
  1. 可以设计一简单的\(g(n) > |x_n - a|\)，此时\(g(n)<\epsilon\)是\(|x_n - a| < \epsilon\)的充分条件

收敛函数的性质
一、唯一性
    若\(\{x_n\}\)收敛，其极限唯一
二、有界性
    \(\{x_n\}\)收敛是\(\{x_n\}\)有界的充分不必要条件
    假命题：有界数列一定收敛\(\Rightarrow\)反例：\(\{(-1)^n\}\)

第三节 函数的极限

函数极限：
    设函数\(f(x)\)在点\(x_0\)的某一去心邻域内有定义，若\(\exists\)常数\(A\)，对\(\forall \epsilon \in \mathbb{N}_+\)，总\(\exists \delta \in \mathbb{N}_+\)，使得当\(x\)满足不等式\(|x - x_0|<\delta\)时，对应函数值满足\(|f(x) - A|<\epsilon\)
    常数\(A\)称为函数\(f(x)\)当\(x\rightarrow x_0\)时的极限，记作\(\mathop{lim}\limits_{x\rightarrow x_0}f(x) = A\)或\(f(x)\rightarrow A(x\rightarrow x_0)\)

\(1^\infty\)型极限 $$ \begin{cases} \lim \alpha(x) = 0\\ \lim \beta(x) = \infty\\ \lim \alpha(x)\beta(x) = k \end{cases}\Rightarrow \lim [1 + \alpha(x)]^{\beta(x)} = e^k $$ 常用函数的极限 $$\lim\limits_{x\rightarrow0}\frac{\sin x}{x} = 1$$

$$\lim\limits_{x\rightarrow0}\frac{\sin x}{x} = \lim\limits_{x\rightarrow0}\frac{\cos x}{1}（洛必达） = 1$$

$$\lim\limits_{x\rightarrow0}(1+x)^{\frac{1}{x}} = e$$ $$\lim\limits_{x\rightarrow\infty}(1+\frac{1}{x})^x = e$$ $$\lim\limits_{x\rightarrow0}\frac{a^x - 1}{x} = \ln a$$

$$\lim\limits_{x\rightarrow0}\frac{a^x - 1}{x} = \lim\limits_{x\rightarrow0}\frac{a^x\ln a}{1}（洛必达） = \ln a$$

$$\lim\limits_{x\rightarrow\infty}\sqrt[n]{n} = 1$$ $$$$
