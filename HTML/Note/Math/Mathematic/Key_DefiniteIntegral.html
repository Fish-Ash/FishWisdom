定积分

牛顿-莱布尼茨公式（简单函数）

对于简单函数\(f(x)\)，直接先用不定积分求\(f(x)\)的原函数\(F(x)\)，再利用牛顿-莱布尼兹公式求解。

$$\int_a^b f(x)\mathrm{d}x = F(b) - F(a)$$

函数奇偶性

\(\int_{-a}^af(x)\mathrm{d}x \Rightarrow\)考虑\(f(x)\)的奇偶性：

  • 若\(f(x)\)为奇函数：\(\int_{-a}^a f(x)\mathrm{d}x = 0\)
  • 若\(f(x)\)为偶函数：\(\int_{-a}^af(x)\mathrm{d}x = 2\int_0^af(x)\mathrm{d}x\)

沃利斯公式（三角函数）

当\(f(x)\)中含有\(\sin x\)或\(\cos x\)时，考虑使用沃利斯公式求解。

$$\int_0^{\frac\pi2}\sin^n x\mathrm{d}x = \int_0^\frac\pi2 \cos^nx \mathrm{d}x = \begin{cases} \frac{\pi}{2}\cdot \frac{1}{2}\cdots\frac{n-3}{n-2}\cdot\frac{n-1}{n}, n为偶数\\ \frac{2}{3}\cdot\frac{4}{5}\cdots\frac{n-3}{n-2}\cdot\frac{n-1}{n},n为奇数\\ \end{cases} $$ $$\int_0^\frac\pi2 f(\sin x)\mathrm{d}x = \int_0^\frac\pi2 f(\cos x)\mathrm{d}x$$ $$\int_0^\pi xf(\sin x)\mathrm{d}x = \frac{\pi}{2}\int_0^\pi f(\sin x)\mathrm{d}x$$

几何意义（根号）

对于\(\int_a^{-a}\sqrt{a^2 - x^2}\mathrm{d}x\)，令\(y = \sqrt{a^2 - x^2}\)，故\(\int_a^{-a}\sqrt{a^2 - x^2}\mathrm{d}x\)为\(x^2 + y^2 = a^2\)的上半圆与\(x\)轴为成的面积，即

$$\int_a^{-a}\sqrt{a^2 - x^2}\mathrm{d}x = \frac12 \pi a^2$$ 推论

对于\(\int_a^b\sqrt{A + Bx - x^2}\mathrm{d}x\)，令\(y = \sqrt{A + Bx - x^2}\)，故\(\int_a^{-a}\sqrt{A + Bx - x^2}\mathrm{d}x\)为\((x - x_0)^2 + y^2 = r^2\)的上半圆在\([a,b]\)上与\(x\)轴为成的面积。

已知\(f'(x)\)

$$\int_a^b f(x) \mathrm{d}x = xf(x)|_a^b - \int_a^bxf'(x)\mathrm{d}x$$
