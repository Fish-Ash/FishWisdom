三角函数专题

弧度制 $$弧度 = 弧长/半径$$

单位：\(rad\)

$$\pi~rad=180°$$

[注]弧度的单位\(rad\)无量纲，可以省略不写.

三角函数及其图像

正弦函数 $$\sin x$$ 余割函数 $$\csc x = \frac{1}{\sin x}$$
余弦函数 $$\cos x$$ 正割函数 $$\sec x = \frac{1}{\cos x}$$
正切函数 $$\tan x = \frac{\sin x}{\cos x}$$ 余切函数 $$\cot x = \frac{\cos x}{\sin x}$$
反正弦函数 $$\arcsin x$$ 反正切函数 $$\arccos x$$
$$(\arcsin x)' = \frac{1}{\sqrt{1 - x^2}}$$

证明：

\(y = f(x) = \arcsin x\)的反函数为\(x = g(y) = \sin y\)

$$\because g'(y) = \cos y$$ $$\begin{align} \therefore f'(x) &= \frac{1}{g'(y)}\\ &= \frac{1}{\cos y}\\ &= \frac{1}{\sqrt{1 - \sin^2y}}\\ &= \frac{1}{\sqrt{1 - \sin^2(\arcsin x)}}\\ &= \frac{1}{\sqrt{1 - x^2}} \end{align} $$
正余互换公式 $$\cot \theta = \tan(\frac{\pi}{2} - \theta)$$
两角和差公式 $$\begin{align} &\sin(\alpha \pm \beta) = \sin\alpha \cos\beta \pm \cos\alpha \sin\beta\\\\ &\cos(\alpha \pm \beta) = \cos\alpha \cos\beta \mp \sin\alpha \sin\beta\\\\ &\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta} \end{align}$$
二倍角公式 $$\begin{align} &\sin2\theta = 2\sin\theta\cos\theta\\\\ &\cos2\theta = \cos^2\theta - \sin^2\theta = 1 - 2\sin^2\theta = 2\cos^2\theta - 1\\\\ &\tan2\theta = \frac{2\tan\theta}{1-\tan^2\theta} \end{align}$$
降幂公式 $$\begin{align} &\sin^2\theta = \frac{1-\cos2\theta}{2}\\\\ &\cos^2\theta = \frac{1 + \cos2\theta}{2} \end{align}$$
积化和差公式 $$\sin\alpha\cdot \cos\beta = \frac{1}{2}[\sin(\alpha + \beta) + \sin(\alpha - \beta)]$$ $$\cos\alpha\cdot \cos\beta = \frac{1}{2}[\cos(\alpha + \beta) + \cos(\alpha - \beta)]$$ $$\sin\alpha\cdot \sin\beta = \frac{1}{2}[\cos(\alpha - \beta) - \cos(\alpha + \beta)]$$
和差化积公式 $$\begin{align} &\sin\alpha + \sin\beta = 2\sin\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}\\\\ &\sin\alpha - \sin\beta = 2\cos\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2}\\\\ &\cos\alpha - \cos\beta = -2\sin\frac{\alpha+\beta}{2}\sin\frac{\alpha-\beta}{2} \end{align}$$
$$\cos\alpha + \cos\beta = 2\cos\frac{\alpha+\beta}{2}\cos\frac{\alpha-\beta}{2}$$
$$\begin{align} \cos\alpha + \cos\beta &= \cos(\frac{\alpha + \beta}{2} + \frac{\alpha - \beta}{2}) + \cos(\frac{\alpha + \beta}{2} - \frac{\alpha - \beta}{2})\\ &= \cos(\frac{\alpha + \beta}{2})\cos(\frac{\alpha - \beta}{2}) - \sin(\frac{\alpha + \beta}{2})\sin(\frac{\alpha - \beta}{2}) + \cos(\frac{\alpha + \beta}{2})\cos(\frac{\alpha - \beta}{2}) + \sin(\frac{\alpha + \beta}{2})\sin(\frac{\alpha - \beta}{2})\\ &= 2\cos(\frac{\alpha + \beta}{2})\cos(\frac{\alpha - \beta}{2}) \end{align}$$
其他公式
$$\arctan \theta + \arctan \frac{1}{\theta} = \frac{\pi}{2}$$

令\(t = \arctan\theta\).

$$\therefore \theta = \tan t \Rightarrow \frac{1}{\theta} = \cot t = \tan(\frac{\pi}{2} - t)$$ $$\therefore \arctan \theta + \arctan \frac{1}{\theta} = t + \frac{\pi}{2} - t = \frac{\pi}{2}$$
$$\frac{1}{1 + \cos x} = \frac{1}{2}\sec^2 \frac{x}{2}$$
$$\sec^2 x = \frac{1}{\cos^2 x} = \frac{2}{1 + \cos 2x}$$ $$\frac{1}{1 + \cos 2x} = \frac{1}{2}\sec^2 x$$ $$\frac{1}{1 + \cos x} = \frac{1}{2}\sec^2 \frac{x}{2}$$
$$\frac{1}{1 - \cos x} = \frac{1}{2}\csc^2 \frac{x}{2}$$
$$\csc^2 x = \frac{1}{\sin^2 x}= \frac{2}{1 - \cos 2x}$$ $$\frac{1}{1 - \cos 2x} = \frac{1}{2}\csc^2 x$$ $$\frac{1}{1 - \cos x} = \frac{1}{2}\csc^2 \frac{x}{2}$$

三角函数带皮亚诺余项的麦克劳林展开式

$$ \begin{align} \sin x &= f(0) + \frac{f'(0)}{1!}x + \frac{f''(0)}{2!}x^2 + \frac{f'''(0)}{3!}x^3 + o(x^3)\\ &= x - \frac16 x^3 + o(x^3)\\ \end{align} \Rightarrow \begin{cases} \sin x \sim x\\ x - \sin x \sim \frac16 x^3 \end{cases} $$
$$ \begin{align} \arcsin x &= f(0) + \frac{f'(0)}{1!}x + \frac{f''(0)}{2!}x^2 + \frac{f'''(0)}{3!}x^3 + o(x^3)\\ &= x + \frac16 x^3 + o(x^3) \end{align} \Rightarrow \begin{cases} \arcsin x \sim x\\ \arcsin x - x \sim \frac16 x^3 \end{cases} $$
$$ \begin{align} \tan x &= f(0) + \frac{f'(0)}{1!}x + \frac{f''(0)}{2!}x^2 + \frac{f'''(0)}{3!}x^3 + o(x^3)\\ &= x + \frac13x^3 + o(x^3) \end{align} \Rightarrow \begin{cases} \tan x \sim x\\ \tan x - x \sim \frac13x^3 \end{cases} $$
$$ \begin{align} \arctan x &= f(0) + \frac{f'(0)}{1!}x + \frac{f''(0)}{2!}x^2 + \frac{f'''(0)}{3!}x^3 + o(x^3)\\ &= x - \frac13x^3 + o(x^3) \end{align} \Rightarrow \begin{cases} \arctan x \sim x\\ x - \arctan x \sim \frac13x^3 \end{cases} $$
$$ \begin{align} \cos x &= f(0) + \frac{f'(0)}{1!}x + \frac{f''(0)}{2!}x^2 + \frac{f'''(0)}{3!}x^3 + o(x^3)\\ &= 1 - \frac{1}{2}x^2 + o(x^2) \end{align} \Rightarrow 1 - \cos x \sim \frac12 x^2 $$
