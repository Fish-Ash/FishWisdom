有理函数

有理函数（有理分式）

两个多项式的商

$$\frac{P(x)}{Q(x)}$$

[注]假定分子多项式\(P(x)\)与分母多项式\(Q(x)\)间无公因式.

真分式

分子多项式次数小于分母多项式次数的有理函数.

假分式

分子多项式次数大于分母多项式次数的有理函数.

利用多项式的除法，总可以将一个假分式化成一个多项式与真分式之和的形式.

对于真分式\(\frac{P(x)}{Q(x)}\)，若分母可分解为两个没有公因式的多项式的乘积

$$Q(x) = Q_1(x)Q_2(x)$$

那么可将真分式分解为两个真分式之和

$$\frac{P(x)}{Q(x)} = \frac{P_1(x)}{Q_1(x)} + \frac{P_2(x)}{Q_2(x)}$$

\(P(x)\)可以用待定系数法求解.

待定系数法分解真分式

$$Q(x) = Q_1(x)Q_2(x)\cdots Q_n(x)$$ $$\frac{P(x)}{Q(x)} = \frac{P_1(x)}{Q_1(x)} + \frac{P_2(x)}{Q_2(x)} + \cdots + \frac{P_n(x)}{Q_n(x)}$$

假设\(Q_i(x)\)的最高次数为\(k\).

设\(P_i(x) = p_{i1}x^{(k-1)} + p_{i2}x^{k-2} + \cdots + p_{i,k-1}x + p_{ik}\)

将\(\frac{P_1(x)}{Q_1(x)} + \frac{P_2(x)}{Q_2(x)} + \cdots + \frac{P_n(x)}{Q_n(x)}\)通分，其分子应等于\(P(x)\)，由此关系解得系数\(p\).
