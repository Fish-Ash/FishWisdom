函数图像

函数单调性

设\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导。

  • 若在\((a,b)\)内\(f'(x)\gt 0\)，则\(f(x)\)在\([a,b]\)内单调递增。
  • 若在\((a,b)\)内\(f'(x)\lt 0\)，则\(f(x)\)在\([a,b]\)内单调递减。

函数极值

定义

若\(f'(x) = 0\)，称\(x = x_0\)为驻点。

设\(f(x)\)在\(x_0\)某个邻域内有定义，对\(\forall x\in U_\delta(x_0)\)

  • 恒有\(f(x)\leq f(x_0)\)，称\(f(x_0)\)为极大值，\(x_0\)为极大值点。
  • 恒有\(f(x)\geq f(x_0)\)，称\(f(x_0)\)为极小值，\(x_0\)为极小值点。
极值的必要条件 $$x = x_0为f(x)的极值点\Rightarrow f'(x_0) = 0$$ 极值的第一充分条件

设\(f'(x_0) = 0\)或\(f(x)\)在\(x_0\)处连续：

$$\begin{cases} f'(x) \gt 0, x\lt x_0\\ f'(x) \lt 0, x\gt x_0 \end{cases}\Rightarrow x = x_0为f(x)的极大值点$$ $$\begin{cases} f'(x) \lt 0, x\lt x_0\\ f'(x) \gt 0, x\gt x_0 \end{cases}\Rightarrow x = x_0为f(x)的极小值点$$ 极值的第二充分条件

设\(f'(x_0)=0\)

$$f''(x) \lt 0 \Rightarrow x = x_0为f(x)的极大值点$$ $$f''(x) \gt 0 \Rightarrow x = x_0为f(x)的极小值点$$ $$f''(x) \neq 0 \nRightarrow x = x_0为f(x)的极值点$$

函数凹凸性

设\(f(x)\)在区间\(U\)上连续，对于\(U\)上任意两点\(x_1,x_2\)

  • 若\(f(\frac{x_1 + x_2}{2})\lt \frac{f(x_1)+f(x_2)}{2}\)，则称\(f(x)\)在区间\(U\)上是凹的。
  • 若\(f(\frac{x_1 + x_2}{2})\gt \frac{f(x_1)+f(x_2)}{2}\)，则称\(f(x)\)在区间\(U\)上是凸的。

设\(f(x)\)在\([a,b]\)内连续，在\((a,b)\)内二阶可导，则

  • 若在\((a,b)\)内有\(f''(x)\gt 0\)，则\(f(x)\)在[a,b]上是凹的。
  • 若在\((a,b)\)内有\(f''(x)\lt 0\)，则\(f(x)\)在[a,b]上是凸的。

函数拐点

定义

连续曲线弧上凹凸分界点。

拐点的必要条件 $$(x_0,f''(x_0))为f(x)的拐点\Rightarrow f''(x_0)=0或f''(x_0)不存在$$ 拐点的第一充分条件

设\(f''(x_0) = 0\)或\(f(x)\)在\(x_0\)处连续：

$$f''(x_0)两侧异号\Rightarrow (x_0, f(x_0))为f(x_0)的拐点$$ $$f''(x_0)两侧同号\Rightarrow (x_0, f(x_0))不是f(x_0)的拐点$$ 拐点的第二充分条件 $$f'''(x_0) \neq 0 \Rightarrow (x_0, f(x_0))为f(x)的拐点$$ $$f'''(x_0) = 0 \nRightarrow (x_0, f(x_0))不为f(x)的拐点$$

函数渐近线

水平渐近线

  • \(\lim\limits_{x\rightarrow +\infty}f(x) = a\)，称\(y = a\)为\(y = f(x)\)右侧水平渐近线
  • \(\lim\limits_{x\rightarrow -\infty}f(x) = b\)，称\(y = b\)为\(y = f(x)\)左侧水平渐近线
  • \(\lim\limits_{x\rightarrow \infty}f(x) = c\)，称\(y = c\)为\(y = f(x)\)水平渐近线

铅直渐近线

\(\lim\limits_{x\rightarrow x_0^+}f(x) = \infty\)或\(\lim\limits_{x\rightarrow x_0^-}f(x) = \infty\)，称\(x=x_0\)为\(y = f(x)\)的铅直渐近线。

斜渐近线

  • \(\lim\limits_{x\rightarrow +\infty}\frac{f(x)}{x} = a, \lim\limits_{x\rightarrow +\infty}[f(x) - ax] = b\)，称\(y = ax + b\)为\(y = f(x)\)右侧斜渐近线。
  • \(\lim\limits_{x\rightarrow -\infty}\frac{f(x)}{x} = a, \lim\limits_{x\rightarrow -\infty}[f(x) - ax] = b\)，称\(y = ax + b\)为\(y = f(x)\)左侧斜渐近线。
  • \(\lim\limits_{x\rightarrow \infty}\frac{f(x)}{x} = a, \lim\limits_{x\rightarrow \infty}[f(x) - ax] = b\)，称\(y = ax + b\)为\(y = f(x)\)斜渐近线。

曲线弧微分与曲率（曲率半径）

弧微分

设\(f(x)\)在\((a,b)\)内有连续导数，则称弧微分\(\mathrm{d}s = \sqrt{1 + f'(x)}\mathrm{d}x\)

曲率 $$\rho = \frac{|f''(x)|}{(1 + f'^2(x))^{\frac32}}$$ 曲率半径 $$R = \frac{1}{\rho} = \frac{(1 + f'^2(x))^{\frac32}}{|f''(x)|}$$

函数零点

通过\(f'(x)\)求解出\(f(x)\)在每个区间内的单调性；求解单调区间内两端点对应的函数值，若异号则存在一根。
