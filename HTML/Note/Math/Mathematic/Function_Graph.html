函数单调性

设\(f(x)\)在\([a,b]\)上连续，在\((a,b)\)内可导。

  • 若在\((a,b)\)内\(f'(x)\gt 0\)，则\(f(x)\)在\([a,b]\)内单调递增。
  • 若在\((a,b)\)内\(f'(x)\lt 0\)，则\(f(x)\)在\([a,b]\)内单调递减。

[注]有限个\(f'(x) = 0\)的点不影响函数单调性.

[注]单调区间需要考虑间断点.

函数极值

定义

若\(f'(x) = 0\)，称\(x = x_0\)为驻点。

设\(f(x)\)在\(x_0\)某个邻域内有定义，对\(\forall x\in U_\delta(x_0)\)

  • 恒有\(f(x)\leq f(x_0)\)，称\(f(x_0)\)为极大值，\(x_0\)为极大值点。
  • 恒有\(f(x)\geq f(x_0)\)，称\(f(x_0)\)为极小值，\(x_0\)为极小值点。
极值的必要条件 $$x = x_0为f(x)的极值点\Rightarrow f'(x_0) = 0$$ 极值的第一充分条件

设\(f'(x_0) = 0\)或\(f(x)\)在\(x_0\)处连续：

$$\begin{cases} f'(x) \gt 0, x\lt x_0\\ f'(x) \lt 0, x\gt x_0 \end{cases}\Rightarrow x = x_0为f(x)的极大值点$$ $$\begin{cases} f'(x) \lt 0, x\lt x_0\\ f'(x) \gt 0, x\gt x_0 \end{cases}\Rightarrow x = x_0为f(x)的极小值点$$ 极值的第二充分条件

设\(f'(x_0)=0\)

$$f''(x) \lt 0 \Rightarrow x = x_0为f(x)的极大值点$$ $$f''(x) \gt 0 \Rightarrow x = x_0为f(x)的极小值点$$ $$f''(x) \neq 0 \nRightarrow x = x_0为f(x)的极值点$$

函数凹凸性

\(f(x)\)在\(U\)上连续，且\(x_1, x_2 \in U\) \(f(x)\)在\([a,b]\)连续，在\((a,b)\)二阶可导
凹函数 $$\frac{f(x_1)+f(x_2)}{2}\gt f(\frac{x_1 + x_2}{2})$$ $$f''(x)\gt 0$$
凸函数 $$\frac{f(x_1)+f(x_2)}{2}\lt f(\frac{x_1 + x_2}{2})$$ $$f''(x) \lt 0$$

函数拐点

拐点

连续曲线弧上凹凸分界点。

$$(x_0,f''(x_0))为f(x)的拐点\Rightarrow f''(x_0)=0或f''(x_0)不存在$$

判断拐点的方法

二阶导数

设\(f''(x_0) = 0\)或\(f(x)\)在\(x_0\)处连续：

$$f''(x_0)两侧异号或f''(x_0)不存在\Rightarrow (x_0, f(x_0))为f(x_0)的拐点$$ $$f''(x_0)两侧同号\Rightarrow (x_0, f(x_0))不是f(x_0)的拐点$$

三阶导数

$$f'''(x_0) \neq 0 \Rightarrow (x_0, f(x_0))为f(x)的拐点$$ $$f'''(x_0) = 0 \nRightarrow (x_0, f(x_0))不为f(x)的拐点$$

函数间断点

间断

设\(f(x)\)在\(x_0\)某个去心邻域内有定义，但\(f(x)\)在\(x_0\)处不连续，称\(x_0\)为\(f(x)\)的间断点。

$$\lim\limits_{x\rightarrow x_0}f(x) \neq f(x_0)$$ 间断点
  • 第一类间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x)\)与\(\lim\limits_{x\rightarrow x_0^+}f(x)\)均存在
    • 可去间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x) = \lim\limits_{x\rightarrow x_0^+}f(x) \neq f(x_0)\)
    • 跳跃间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x) \neq \lim\limits_{x\rightarrow x_0^+}f(x)\)
  • 第二类间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x)\)与\(\lim\limits_{x\rightarrow x_0^+}f(x)\)至少一个不存在
    • 无穷间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x) = \infty\)或\(\lim\limits_{x\rightarrow x_0^+}f(x) = \infty\)
    • 振荡间断点：\(\lim\limits_{x\rightarrow x_0^-}f(x)\)或\(\lim\limits_{x\rightarrow x_0^+}f(x)\)振荡
找间断点方法

找函数定义域内无意义的点。

  • \(\frac1x\)：\(x\neq 0\)
  • \(\ln|x|\)：\(x\neq 0\)

曲线弧微分与曲率（曲率半径）

弧微分

设\(f(x)\)在\((a,b)\)内有连续导数，则称弧微分\(\mathrm{d}s = \sqrt{1 + f'^2(x)}\mathrm{d}x\)

曲率\(\kappa\) $$\kappa = \frac{|f''(x)|}{[1 + f'^2(x)]^{3/2}}$$ 曲率半径\(\rho\) $$\rho = \frac{1}{\kappa} = \frac{[1 + f'^2(x)]^{3/2}}{|f''(x)|}$$ 曲率中心\((\alpha, \beta)\) $$\alpha = x_0 - \frac{f'(x_0)[1 + f'^2(x_0)]}{f''(x_0)}$$ $$\beta = y_0 + \frac{f'^2(x_0)}{f''(x_0)}$$ 曲率圆方程 $$(x - \alpha)^2 + (y - \beta)^2 = \rho^2$$

函数零点

通过\(f'(x)\)求解出\(f(x)\)在每个区间内的单调性；求解单调区间内两端点对应的函数值，若异号则存在一根。
