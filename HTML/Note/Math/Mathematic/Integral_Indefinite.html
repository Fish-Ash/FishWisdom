不定积分

一、不定积分概念与性质

原函数

若\(F'(x) = f(x)\)，称\(F(x)\)为\(f(x)\)的一个原函数.

[注]若\(F(x)\)为\(f(x)\)的原函数，则\(F(x) + C\)也是\(f(x)\)的一个原函数。

不定积分

\(f(x)\)原函数的全体称为\(f(x)\)的不定积分。

$$\int f(x) \mathrm{d}x = F(x) + C$$
原函数存在定理 $$f(x)在区间U内连续\Rightarrow f(x)在区间U上原函数F(x)存在$$ $$f(x)在区间U内存在第一类间断点\Rightarrow f(x)在区间U上原函数F(x)不存在$$
不可由导数公式直接得出的不定积分 $$\begin{align} &\int \frac 1x \mathrm{d}x = \ln |x| + C\\\\ &\int \sec x\mathrm{d}x = \ln|\sec x + \tan x| + C\\\\ &\int \csc x\mathrm{d}x = -\ln|\csc x + \cot x| + C = \ln|\csc x - \cot x| + C\\\\ &\int \frac{1}{\sqrt{a^2 - x^2}}\mathrm{d}x = \arcsin\frac{x}{a} + C\\\\ &\int \frac{1}{a^2 + x^2}\mathrm{d}x = \frac{1}{a}\arctan\frac{x}{a} + C\\\\ \end{align}$$
\(\int \tan x\mathrm{d}x = -\ln|\cos x| + C\)
$$\begin{align} \int \tan x \mathrm{d}x &= -\int \frac{1}{\cos x}\mathrm{d}\cos x\\ &= -\ln |\cos x| + C \end{align}$$
\(\int \cot x\mathrm{d}x = \ln|\sin x| + C\)
不定积分常用结论 $$\int \ln x \mathrm{d}x = x\ln x - x + C$$ $$\int x\sin x \mathrm{d}x = \sin x - x\cos x + C$$ $$\int x\cos x \mathrm{d}x = \cos x + x\sin x + C$$ $$\int \cos^2 x \mathrm{d}x = \frac{1}{2}x + \frac{1}{4}\sin 2x + C$$
$$\int \frac{1}{\sqrt{x^2 + a^2}}\mathrm{d}x = \ln(x + \sqrt{x^2 + a^2}) + C$$

令\(x = a\tan t\)

$$\begin{align} &\int \frac{1}{\sqrt{x^2 + a^2}}\mathrm{d}x\\ =&\int \frac{1}{\sec t}\cdot a\sec^2 t\mathrm{d}t\\ =&a\int \sec t\mathrm{d}t\\ =&a\ln|\sec t + \tan t| + C\\ =&a\ln|\frac{\sqrt{x^2+a^2}}{a} + \frac{x}{a}| + C\\ =&a\ln(\sqrt{x^2+a^2}+x) + C \end{align}$$
$$\int \frac{1}{\sqrt{x^2 - a^2}}\mathrm{d}x = \ln|x + \sqrt{x^2 - a^2}| + C$$
不定积分的性质 $$\begin{align} &\int [f(x) \pm g(x)]\mathrm{d}x = \int f(x)\mathrm{d}x \pm \int g(x)\mathrm{d}x\\\\ &\int kf(x)\mathrm{d}x = k\int f(x)\mathrm{d}x \end{align}$$

分部积分法

$$\mathrm{d}(uv) = u\mathrm{d}v + v\mathrm{d}u$$ $$\int u\mathrm{d}v = uv - \int v\mathrm{d}u $$

\(u \)的优先级：反对幂指三

三角有理化

设\(\tan \frac{x}{2} = t\)，则\(x = 2\arctan t, \mathrm{d}x = \frac{2}{1 + t^2}\)

$$\sin x = \frac{2\sin\frac{x}{2}\cos\frac{x}{2}}{\sin^2\frac{x}{2}+\cos^2\frac{x}{2}} = \frac{2\tan\frac{x}{2}}{\tan^2\frac{x}{2}+1} = \frac{2t}{t^2 + 1}$$
$$\cos x = \frac{\cos^2 \frac{x}{2}-\sin^2 \frac{x}{2}}{\sin^2\frac{x}{2} + \cos^2\frac{x}{2}} = \frac{1 - t^2}{t^2 + 1 }$$
$$\tan x = \frac{\sin x}{\cos x} = \frac{2t}{1-t^2}$$

题型

判断函数是否存在原函数

解题思路

函数连续\(\Rightarrow\)存在原函数

函数不连续\(\Rightarrow\)判断间断点类型

  • 若为第一类间断点\(\Rightarrow\)不存在原函数
  • 若为第二类间断点\(\Rightarrow\)构造出\(F'(x) = f(x)\)

被积函数化简方法

一般分式 按分子拆分
有理函数 部分分式分解
三角函数 积化和差
$$\frac{u^m}{\sqrt{u^n + a}} = u^{m-n}\sqrt{u^n + a} - \frac{au^{m-n}}{\sqrt{u^n + a}}$$

求解不定积分的方法

初等函数

直接利用公式求解

\(\int f[\phi(x)]\phi'(x)\mathrm{d}x\)型——凑微分法

$$\int f[\phi(x)]\phi'(x)\mathrm{d}x = \int f[\phi(x)]\mathrm{d}\phi(x) = F[\phi(x)] + C$$

\(\int f(x)\cdot g(x)\mathrm{d}x\)型——分部积分法

$$\begin{align} \int f(x)g'(x)\mathrm{d}x &= \int f(x) \mathrm{d}g(x)\\ &= f(x)g(x) - \int g(x)\mathrm{d}f(x) \end{align}$$

\(\sqrt{a^2 - x^2},\sqrt{a^2 + x^2}, \sqrt{x^2 - a^2}\)型——三角代换法

\(\sqrt{ax^2 + bx + c}\)型

凑成\(\sqrt{a^2 - x^2},\sqrt{a^2 + x^2}, \sqrt{x^2 - a^2}\)型.

\(f(x)^2 + 1, f(x)^2 - 1\)型

\(f(x)^2 + 1 \Rightarrow\)令\(f(x) = \tan t\)

\(f(x)^2 - 1 \Rightarrow\)令\(f(x) = \sec t\)

\(\frac{1}{ax^2 + bx + c}\)型

$$\begin{align} \int \frac{1}{ax^2 + bx + c} \mathrm{d}x &= A\int \frac{1}{(x + m)^2 + n^2}\mathrm{d}x\\ &= \frac{A}{n}\arctan \frac{x+m}{n} \end{align}$$

\(\frac{a'x + b'}{ax^2 + bx + c}\)型

$$\int\frac{a'x + b'}{ax^2 + bx + c}\mathrm{d}x = A\int\frac{2ax + b}{ax^2 + bx + c}\mathrm{d}x + B\int\frac{1}{ax^2 + bx + c}\mathrm{d}x$$

\(\frac{1}{x^m\sqrt{a + x^n}}\)型——倒代换

令\(x = \frac{1}{u}\)

复杂分式

包括分母项数多或含根式导致不好化简的情况

令分母为\(t\)，注意换\(\mathrm{d}x = x'(t)\mathrm{d}t\)并在计算完成后将\(t\)代换回\(x\).

三角函数

\(\sin x\cdot f(\cos x), \cos x\cdot f(\sin x)\)型 $$\int \sin x\cdot f(\cos x) \mathrm{d}x = -\int f(\cos x)\mathrm{d}(\cos x) $$ $$\int \cos x\cdot f(\sin x) \mathrm{d}x = \int f(\sin x)\mathrm{d}(\sin x) $$

分段函数

由于\(可导\Rightarrow 连续\)，\(F(x) = \int f(x)\mathrm{d}x\)一定连续.

故在分段点\(x_0\)处，由\(\lim\limits_{x\rightarrow x^-_0}F(x) = \lim\limits_{x\rightarrow x^+_0}F(x)\)求解出\(C_1\)与\(C_2\)间的关系.

[注]求不定积分时，结果一定要带上常数\(C\).
