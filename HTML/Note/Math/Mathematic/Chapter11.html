二、向量的数乘

运算规律
结合律 $$\lambda(\mu \mathbf{a}) = \mu(\lambda \mathbf{a}) = (\lambda\mu)\mathbf{a}$$
分配律 $$(\lambda + \mu)\mathbf{a} = \lambda\mathbf{a} + \mu\mathbf{a}$$
$$\lambda(\mathbf{a} + \mathbf{b}) = \lambda \mathbf{a} + \lambda \mathbf{b}$$
$$\mathbf{a} = (a_x, a_y, a_z), \mathbf{b} = (b_x, b_y, b_z)$$
向量的数量积（点乘）
几何表示 $$\mathbf{a}\cdot\mathbf{b} = |\mathbf{a}|\cdot|\mathbf{b}|\cdot\cos\lt\mathbf{a}, \mathbf{b}\gt$$
代数表示 $$\mathbf{a}\cdot \mathbf{b} = a_xb_x + a_yb_y + a_zb_z$$
运算规律 交换律
结合律
分配律
几何应用 向量的模：\(|\mathbf{a}| = \sqrt{a^2}\)
向量夹角：\(\cos \lt\mathbf{a},\mathbf{b}\gt \frac{\mathbf{a}\cdot\mathbf{b}}{|\mathbf{a}|\cdot|\mathbf{b}|}\)
$$\mathbf{a}\perp\mathbf{b} \Rightarrow \mathbf{a}\cdot \mathbf{b} = 0$$

向量的向量积（叉乘）
几何表示 $$\mathbf{a} \times \mathbf{b}$$

大小：\(|\mathbf{a}|\cdot|\mathbf{b}|\cdot\sin\lt\mathbf{a}, \mathbf{b}\gt\)

方向：垂直于\(\mathbf{a}\)与\(\mathbf{b}\)，满足右手法则.
代数表示 $$\mathbf{a}\times \mathbf{b} =\left |\begin{matrix}i & j & k\\a_x & a_y & a_z\\b_x & b_y & b_z\end{matrix}\right |$$
运算规律 反交换律：\(\mathbf{a}\times\mathbf{b} = -\mathbf{b}\times \mathbf{a}\)
分配律
几何应用 同时垂直于\(\mathbf{a}\)和\(\mathbf{b}\)的向量：\(\mathbf{a}\times\mathbf{b}\)
以\(\mathbf{a}\)与\(\mathbf{b}\)为边的平行四边形面积：\(S = |\mathbf{a}\times\mathbf{b}| = |\mathbf{a}|\cdot|\mathbf{b}|\cdot\sin\lt \mathbf{a}, \mathbf{b}\gt\)
$$\mathbf{a} // \mathbf{b} \Rightarrow \mathbf{a}\times\mathbf{b} = 0$$

混合积
几何表示 $$[a,b,c] = (a\times b)\cdot c$$
代数表示 $$[a,b,c] = \left |\begin{matrix}a_x & a_y & a_z\\b_x & b_y & b_z\\c_x & c_y & c_z\end{matrix}\right |$$
运算规律 交换变号：\([a,b,c] = -[b,a,c] = -[a,c,b]\)
几何应用 以\(\mathbf{a}, \mathbf{b}, \mathbf{c}\)为棱的平行六面体的体积：\(V = |[a,b,c]|\)
向量\(\mathbf{a}, \mathbf{b}, \mathbf{c}\)共面：\([a,b,c] = 0\)

常用结论

$$\mathbf{a}\times \mathbf{a} = 0$$
$$\mathbf{a}\times \mathbf{a} = |a||a|\sin 0 = 0$$
$$(\mathbf{a}\times\mathbf{b})\cdot\mathbf{a} = 0$$
$$\mathbf{a}\times\mathbf{b} \perp \mathbf{a} \Rightarrow (\mathbf{a}\times\mathbf{b})\cdot\mathbf{a} = 0$$

空间中的直线方程与平面方程

平面方程

设平面法向量\(\mathbf{n} = (A,B,C)\)，在平面上任取两点\((x,y,z)\)与\((x_0, y_0, z_0)\).故向量\((x-x_0, y-y_0, z-z_0)\)与\(\mathbf{n}\)垂直.

点法式 $$A(x-x_0) + B(y-y_0) + C(z-z_0) = 0$$ 一般式 $$Ax + By + Cz -(Ax_0 +By_0 + Cz_0) = 0$$

令\(Ax_0 + By_0 + Cz_0 = D\)，有

$$Ax + By + Cz + D = 0$$ 截距式

设平面与\(x,y,z\)轴的截距分别为\(a,b,c\)

$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$
特殊平面
过x轴的平面 $$By + Cz = 0 \Rightarrow y + tz = 0$$
过y轴的平面 $$Ax + Cz = 0 \Rightarrow x + tz = 0$$
过z轴的平面 $$Ax + By = 0 \Rightarrow x + ty = 0$$

以过x轴的平面为例：

点\((x,0,0), x\in \mathbb{R}\)均位于该平面内，带入一般式\(Ax + By + Cz + D = 0\)得：

$$Ax + D = 0 \Rightarrow By + Cz = 0$$

直线方程

一般式（两平面交线） $$\begin{cases} A_1x + B_1y + C_1z + D_1 = 0\\ A_2x + B_2y + C_2z + D_2 = 0 \end{cases}$$

设方向向量\(\mathbf{s} = (a,b,c)\)与直线平行，在直线上任取两点\((x,y,z)\)与\((x_0, y_0, z_0)\)构成向量\((x-x_0, y-y_0, z-z_0)\).

对称式 $$\frac{x-x_0}{a} = \frac{y - y_0}{b} = \frac{z - z_0}{c} = t$$ 参数式 $$\begin{cases} x = x_0 + at\\ y = y_0 + bt\\ z = z_0 + ct \end{cases}$$

直线与平面之间的角度

直线与直线间的夹角 $$\cos \theta = \frac{\mathbf{s_1}\cdot \mathbf{s_2}}{|\mathbf{s_1}|\cdot|\mathbf{s_2}|}$$ 平面与平面间的夹角 $$\cos \theta = \frac{\mathbf{n_1}\cdot \mathbf{n_2}}{|\mathbf{n_1}|\cdot|\mathbf{n_2}|}$$ 直线与平面间的夹角 $$\sin \theta = \frac{\mathbf{n}\cdot \mathbf{s}}{|\mathbf{n}|\cdot|\mathbf{s}|}$$ 点到平面的距离

点\((x_0, y_0, z_0)\)到平面\(Ax + By + Cz+ D = 0\)的距离

$$d = \frac{|Ax_0 + By_0 + Cz_0|}{\sqrt{A^2 + B^2 + C^2}}$$ 点到直线的距离

点\((x_0, y_0, z_0)\)到直线\(\frac{x - x_1}{l} = \frac{y - y_1}{m} = \frac{z - z_1}{n}\)的距离

$$d = \frac{|(x_1 - x_0, y_1 - y_0, z_1 - z_0)\times(l,m,n)|}{\sqrt{l^2 + m^2 + n^2 }}$$

空间中的曲面及曲线方程

空间中的曲面

隐函数：\(F(x,y,z) = 0\Rightarrow z = z(x,y)\)

显函数：\(z = f(x,y)\)

空间中的曲线

一般式：

$$\begin{cases}F(x,y,z)=0\\G(x,y,z)=0\end{cases}$$

参数式：

$$\begin{cases}x=x(t)\\y=y(t)\\z=z(t)\end{cases}$$

旋转曲面

设\(L\)为\(yoz\)平面上的一条曲线，其方程为\(\begin{cases}f(y,z)=0\\x=0\end{cases}\)

曲线绕\(y\)轴旋转所得的旋转曲面方程为

$$f(y, \pm\sqrt{x^2+z^2}) = 0$$

曲线绕\(z\)轴旋转所得的旋转曲面方程为

$$f(\pm \sqrt{x^2 + y^2}, z) = 0$$
