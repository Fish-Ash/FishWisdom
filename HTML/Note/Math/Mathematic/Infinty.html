无穷小量

若\(\lim\limits_{x\rightarrow x_0}f(x) = 0\)，则称\(f(x)\)为\(x\rightarrow x_0\)时的无穷小量。

无穷小量阶的比较

\(\lim\alpha = 0, \lim\beta = 0, \beta(x)\neq 0\).

\(\alpha(x)\)为\(\beta(x)\)的：

  • 高阶无穷小：\(\lim \frac{\alpha(x)}{\beta(x)} = 0\)，记作\(\alpha(x) = o[\beta(x)]\)
  • 低阶无穷小：\(\lim \frac{\alpha(x)}{\beta(x)} = \infty\)
  • 同阶无穷小：\(\lim \frac{\alpha(x)}{\beta(x)} = c\neq 0\)
  • 等价无穷小：\(\lim \frac{\alpha(x)}{\beta(x)} = 1\)，记作\(\alpha(x)\sim \beta(x)\)
  • \(k\)阶无穷小：\(\lim \frac{\alpha(x)}{\beta^k(x)} = c\neq 0\)
无穷小的运算 $$o(x^m) + o(x^n) = o(x^k), k = \min\{m,n\}$$ $$o(x^m)\cdot o(x^n) = o(x^{m+n})$$ $$x^m\cdot o(x^n) = o(x^{m+n})$$ $$o(kx^n) = k\cdot o(x^n) = o(x^n), k\neq 0$$
无穷小的性质

有限个无穷小的和仍是无穷小

有限个无穷小的乘积仍然是无穷小

无穷小与有界量的乘积仍然是无穷小

等价无穷小具有相同的函数值与导数值

常见等价无穷小
$$1 - \cos x \sim \sec x - 1 \sim \frac{1}{2}x^2$$ $$\ln(1 + x) \sim e^x - 1 \sim x$$ $$x - \ln(1 + x) \sim \frac12 x^2$$ $$a^x - 1 \sim x\ln a$$ $$(1 + x)^\alpha -1 = \alpha x$$

无穷大量

\(\lim\limits_{x\rightarrow x_0}f(x) = \infty\)，称\(f(x)\)为\(x\rightarrow x_0\)时的无穷大量。

$$对于\forall M \gt 0, \exists \delta \gt 0, 使得当0\lt |x - x_0|\lt \delta时，|f(x)|\gt M$$

\(\lim\limits_{x\rightarrow \infty}f(x) = \infty\)，称\(f(x)\)为\(x\rightarrow \infty\)时的无穷大量。

$$对于\forall M \gt 0, \exists X \gt 0, 使得当|x|\gt X时，|f(x)|\gt M$$ 常见无穷大量的比较

令\(\alpha \gt 0, \beta \gt 0, a\gt 1\)

函数

\(x\rightarrow +\infty\)时：

$$\ln^\alpha x \ll x^\beta \ll a^x$$ 数列

\(n\rightarrow \infty\)时：

$$\ln^\alpha n\ll n^\beta \ll a^n \ll n! \ll n^n$$ 无穷大量的性质

无穷大量包括正无穷与负无穷

两个无穷大量的乘积仍为无穷大量

无穷大量与有界变量之和仍是无穷大量

无穷大量与无界变量的关系

对于数列\(\{x_n\}\)，无穷大量要求\(n\gt N\)后所有项均满足\(|x_n|\gt M\)，无界变量仅要求存在一项满足\(|x_n|\gt M\)。

因此无界变量包含无穷大量。

无穷大量与无穷小量的关系

设\(f(x)\neq 0\)

$$\lim f(x) = 0 \Leftrightarrow \lim \frac{1}{f(x)} = \infty$$
反常积分中常用的等价无穷小与等价无穷大
$$(x^\alpha + x^\beta)^t (\alpha \lt \beta)$$
等价无穷小 $$x^{\alpha t}$$ $$\lim\limits_{x\rightarrow 0}(\frac{x^\alpha + x^\beta}{x^\alpha})^t = \lim\limits_{x\rightarrow 0}(1 + \frac{x^\beta}{x^\alpha})^t = 1$$
等价无穷大 $$x^{\beta t}$$ $$\lim\limits_{x\rightarrow \infty}(\frac{x^\alpha + x^\beta}{x^\beta})^t = \lim\limits_{x\rightarrow 0}(\frac{x^\alpha}{x^\beta} + 1)^t = 1$$
