泰勒公式

泰勒级数 $$\sum\limits_{n=0}^\infty \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n $$

带有皮亚诺余项的\(n\)阶泰勒公式

\(x\rightarrow x_0\)时：

$$f(x) = f(x_0) + \frac{f'(x_0)}{1!}(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + \cdots + \frac{f^{(n)}(x_0)}{n!}(x-x_0)^n + o((x^{x-x_0})^n)$$ 其中，\(o((x-x_0)^n)\)称为皮亚诺余项.

[注]按\((x-x_0)\)的幂展开\(f(x)\)不一定有皮亚诺余项.

麦克劳林级数 $$\sum\limits_{n=0}^\infty \frac{f^{(n)}(0)}{n!}x^n$$

带有阶皮亚诺余项的\(n\)阶麦克劳林展开式

\(x\rightarrow 0\)时：

$$f(x) = f(0) + \frac{f'(0)}{1!}x + \frac{f''(0)}{2!}x^2 + \cdots + \frac{f^{(n)}(0)}{n!}x^n + o(x^n)$$

常用函数的麦克劳林公式及麦克劳林展开式 $$\frac{1}{1-x} = 1 + x + x^2 + \cdots + x^n + o(x^n) = \sum\limits_{n=0}^\infty x^n (-1\lt x\lt 1)$$ $$\frac{1}{1+x} = 1 - x + x^2 + \cdots + (-1)^nx^n + o(x^n) = \sum\limits_{n=0}^\infty (-1)^nx^n (-1\lt x\lt 1)$$ $$e^x = 1 + x + \frac{x^2}{2} + \cdots + \frac{x^n}{n!} + o(x^n) = \sum\limits_{n=0}^\infty\frac{x^n}{n!}(-\infty \lt x\lt +\infty)$$ $$\ln(1 + x) = x -\frac{x^2}{2} + \frac{x^3}{3} + \cdots + (-1)^{n-1}\frac{x^n}{n} + o(x^n) = \sum\limits_{n=1}^\infty(-1)^{n-1}\frac{x^n}{n} (-1\lt x\leq 1)$$ $$(1 + x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!}x^2 + \cdots + \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!}x^n+o(x^n) = 1 + \sum\limits_{n=1}^\infty \frac{\alpha(\alpha-1)\cdots(\alpha-n+1)}{n!}x^n (-1\lt x \lt 1)$$ $$\sin x = x - \frac{x^3}{3!} + \frac{x^5}{5!} + \cdots + (-1)^n\frac{x^{2n+1}}{(2n+1)!} + o(x^{2n+1}) = \sum\limits_{n=0}^\infty (-1)^n\frac{x^{2n+1}}{(2n+1)!}(-\infty \lt x \lt +\infty)$$ $$\cos x = 1 - \frac{x^2}{2!} + \frac{x^4}{4!} + \cdots + (-1)^n\frac{x^{2n}}{(2n)!} + o(x^{2n}) = \sum\limits_{n=0}^\infty (-1)^n\frac{x^{2n}}{(2n)!}(-\infty\lt x \lt +\infty)$$
