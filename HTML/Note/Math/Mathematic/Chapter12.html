第十二章 无穷级数

常数项级数

级数（常数项无穷级数） $$\sum\limits_{n=1}^\infty u_n = u_1 + u_2 + \cdots + u_n + \cdots$$

其中\(u_n\)称为级数的一般项.

无穷级数的前\(n\)项 $$u_1 + u_2 + \cdots + u_n + \cdots$$
级数的部分和 $$s_n = \sum\limits_{k=1}^n u_k = u_1 + u_2 + \cdots + u_n$$ 推导公式 $$u_n = s_n - s_{n-1}$$

级数的敛散性

级数的收敛与发散

若\(\lim\limits_{n\rightarrow\infty}s_n = s\)，称\(\sum\limits_{n=1}^\infty u_n\)收敛.

\(s\)称为级数的和：

$$s = u_1 + u_2 + \cdots +$$

若\(\lim\limits_{n\rightarrow\infty}S_n\)不存在，称\(\sum\limits_{n=1}^\infty u_n\)发散.

级数的性质

\(\sum\limits_{n=1}^\infty u_n\)收敛于\(s \Rightarrow \sum\limits_{n=1}^\infty ku_n\)收敛于\(ks\).

\(\sum\limits_{n=1}^\infty u_n, \sum\limits_{n=1}^\infty v_n\)分别收敛于\(s,t\Rightarrow \sum\limits_{n=1}^\infty(u_n \pm v_n)\)收敛于\(s \pm t\).

\(\sum\limits_{n=1}^\infty u_n\)收敛，\(\sum\limits_{n=1}^\infty v_n\)发散\(\Rightarrow \sum\limits_{n=1}^\infty(u_n \pm v_n)\)发散.

\(\sum\limits_{n=1}^\infty u_n, \sum\limits_{n=1}^\infty v_n\)均发散\(\Rightarrow \sum\limits_{n=1}^\infty(u_n \pm v_n)\)敛散性不确定.

无穷级数中增减或改变有限项，级数敛散性不变

  • 原级数收敛\(\Rightarrow\)加括号后收敛.
  • 级数加括号后收敛\(\Rightarrow\)原级数不一定收敛.
  • 级数加括号后发散\(\Rightarrow\)原级数一定发散.

[注]级数加括号：\((u_1 + u_2) + (u_3 + u_4) + \cdots\)

级数收敛的必要条件

\(\sum\limits_{n=1}^\infty u_n\)收敛\(\Rightarrow \lim\limits_{n\rightarrow \infty}u_n = 0\)

级数审敛法

首先考虑利用等价无穷大或等价无穷小进行化简.

正项级数

$$\sum\limits_{n=1}^\infty u_n(u_n\geq 0)$$

正项级数收敛\(\Leftrightarrow\)部分和\(\{S_n\}\)有上界.

比较判别法

设\(u_n \leq v_n\)

  • \(\sum\limits_{n=1}^\infty v_n\)收敛\(\Rightarrow \sum\limits_{n=1}^\infty u_n\)收敛.
  • \(\sum\limits_{n=1}^\infty u_n\)发散\(\Rightarrow \sum\limits_{n=1}^\infty v_n\)发散.
比较判别法（极限形式）

设\(\lim\limits_{n\rightarrow \infty}\frac{u_n}{v_n} = l(0\leq l \leq +\infty)\)

  • \(0\lt l \lt \infty\)：\(\sum\limits_{n=1}^\infty u_n\)与\(\sum\limits_{n=1}^\infty v_n\)具有相同的敛散性.
  • \(l = 0 \Rightarrow u_n \lt v_n\)：使用比较判别法.
  • \(l = +\infty \Rightarrow u_n \gt v_n\)：使用比较判别法.
比值审敛法——适用于含\(n!\)项、含\(a^n\)项
$$\lim\limits_{n\rightarrow \infty}\frac{u_{n+1}}{u_n} = \rho$$
$$\sum\limits_{n=1}^\infty u_n$$ 收敛 $$\rho \lt 1$$
不确定 $$\rho = 1$$
发散 $$\rho \gt 1$$
根值审敛法（柯西判别法）——适用于仅含\(a^n\)项
$$\lim\limits_{n\rightarrow \infty}\sqrt[n]{u_n} = \rho$$
$$\sum\limits_{n=1}^\infty u_n$$ 收敛 $$\rho \lt 1$$
不确定 $$\rho = 1$$
发散 $$\rho \gt 1$$
积分判别法

设\(f(x)\)是\([1,+\infty)\)上单调递减，非负的连续函数，\(a(n) = f(n)\)，则\(\sum\limits_{n=1}^\infty a_n\)与\(\int_1^{+\infty}f(x)\mathrm{d}x\)具有相同的敛散性.

等比级数（几何级数） $$\sum\limits_{n=1}^\infty aq^n = a + aq + aq^2 + \cdots + aq^n + \cdots,~a\neq0$$

\(q\)称为级数的公比.

\(q \neq 1\)时：

$$S_n = \frac{aq(1-q^n)}{1-q}$$ $$|q| \gt 1 \Rightarrow\lim\limits_{n\rightarrow \infty}S_n = \infty \Rightarrow 级数发散$$ $$|q| \lt 1 \Rightarrow\lim\limits_{n\rightarrow \infty}S_n = \frac{aq}{1-q} \Rightarrow 级数收敛$$ $$q = -1\Rightarrow \lim\limits_{n\rightarrow \infty}S_n不存在\Rightarrow 级数发散$$

\(q=1\)时：

$$S_n = na$$ $$\lim\limits_{n\rightarrow \infty}S_n = \infty \Rightarrow 级数发散$$

综上：

\(|q|\lt1\)时，级数收敛；\(|q|\neq 1\)时，级数发散.

p级数 $$\sum\limits_{n=1}^\infty \frac{1}{n^p}$$

当\(p = 1\)时，\(\sum\limits_{n=1}^\infty \frac{1}{n}\)称为调和级数.

反证法证明调和级数\(\sum\limits_{n=1}^\infty \frac{1}{n}\)发散：

假设\(\sum\limits_{n=1}^\infty \frac{1}{n}\)收敛.

$$\therefore \lim\limits_{n\rightarrow \infty}S_n = \lim\limits_{n\rightarrow \infty}S_{2n} = s \Rightarrow \lim\limits_{n\rightarrow \infty}(S_{2n} - S_n) = 0$$ $$\begin{align}S_{2n} - S_{n} &= (1 + \frac12 + \cdots + \frac1n + \frac{1}{n+1} + \cdots \frac{1}{2n}) - (1 + \frac12 + \cdots + \frac1n)\\ &= \frac{1}{n+1} + \frac{1}{n+2} + \cdots \frac{1}{2n}\end{align}$$ $$\because \frac{1}{n+1}\gt \frac{1}{2n}, \frac{1}{n+2}\gt \frac{1}{2n},\cdots, \frac{1}{2n-1}\gt \frac{1}{2n}$$ $$\therefore \lim\limits_{n\rightarrow \infty}(S_{2n} - S_n) \gt \lim\limits_{n\rightarrow \infty}(n\cdot\frac{1}{2n}) = \frac12 \neq 0$$

故假设不成立，调和级数\(\sum\limits_{n=1}^\infty \frac{1}{n}\)发散.

交错级数

$$\sum\limits_{n=1}^\infty (-1)^nu_n(u_n\gt 0)$$ 莱布尼兹准则

若\(\{u_n\}\)单调递减，\(\lim\limits_{n\rightarrow \infty} u_n = 0 \Rightarrow \sum\limits_{n=1}^\infty(-1)^{n-1}u_n\)收敛.

任意项级数

绝对收敛

若\(\sum\limits_{n=1}^\infty|u_n|\)收敛，且\(\sum\limits_{n=1}^\infty u_n\)收敛.

条件收敛

若\(\sum\limits_{n=1}^\infty u_n\)收敛，但\(\sum\limits_{n=1}^\infty |u_n|\)发散.

结论

\(\sum\limits_{n=1}^\infty u_n\)绝对收敛\(\Rightarrow \sum\limits_{n=1}^\infty \frac{u_n + |u_n|}{2}, \sum\limits_{n=1}^\infty \frac{u_n - |u_n|}{2}\)绝对收敛.

\(\sum\limits_{n=1}^\infty u_n\)条件收敛\(\Rightarrow \sum\limits_{n=1}^\infty \frac{u_n + |u_n|}{2}, \sum\limits_{n=1}^\infty \frac{u_n - |u_n|}{2}\)发散.

幂级数

定义 $$\sum\limits_{n = 0}^\infty a_n x^n = a_0 + a_1x + a_2x^2 + \cdots + a_n x^n + \cdots$$ $$\sum\limits_{n = 0}^\infty a_n(x- x_0)^n = a_0 + a_1(x-x_0) + a_2(x-x_0)^2 + \cdots + a_n(x-x_0)^n + \cdots$$ 阿贝尔定理

若\(\sum\limits_{n=0}^\infty a_nx^n\)在\(x = x_0(x\neq 0)\)处收敛，则当\(|x|\lt |x_0|\)时，\(\sum\limits_{n=1}^\infty a_nx^n\)绝对收敛.

若\(\sum\limits_{n=0}^\infty a_nx^n\)在\(x = x_1\)处发散，则当\(|x|\gt |x_1|\)时，\(\sum\limits_{n=1}^\infty a_nx^n\)发散.

定理

幂级数\(\sum\limits_{n=0}^\infty a_nx^n\)的收敛性有且仅有三种可能：

（1）对于\(\forall x \in (-\infty, +\infty)\)，幂级数都收敛.

（2）仅在\(x=0\)收敛.

（3）存在一个正数\(R\)，当\(|x|\lt R\)时，幂级数绝对收敛，当\(|x|\gt R\)时，幂级数发散.

上述定理中正数\(R\)称为幂级数\(\sum\limits_{n=0}^\infty a_nx^n\)收敛半径，开区间\((-R,R)\)称为其收敛区间.

在幂级数的收敛区间基础上考察\(x=\pm R\)时幂级数的敛散性，得出幂级数全部收敛点的集合，称为收敛域.

[注]若幂级数在点\(x = x_0\)处条件收敛，则点\(x_0\)必为幂级数收敛区间\((-R,R)\)上的一个端点，即\(R=|_0|\)

求解幂级数收敛半径及收敛区间

幂级数的性质

有理运算的性质

设幂级数\(\sum\limits_{n = 0}^\infty a_n x^n\)收敛半径为\(R_1\)，幂级数\(\sum\limits_{n = 0}^\infty b_n x^n\)收敛半径为\(R_2\)，令\(R = \min\{R_1, R_2\}\)，则

$$\sum\limits_{n = 0}^\infty a_n x^n \pm \sum\limits_{n = 0}^\infty b_n x^n = \sum\limits_{n = 0}^\infty (a_n\pm b_n) x^n$$ $$\sum\limits_{n = 0}^\infty a_n x^n\cdot \sum\limits_{n = 0}^\infty b_n x^n$$ $$\frac{\sum\limits_{n = 0}^\infty a_n x^n}{\sum\limits_{n = 0}^\infty b_n x^n}$$ 在\(x \in (-R, R)\)上绝对收敛.

分析性质

设幂级数\(\sum\limits_{n = 0}^\infty a_n x^n\)收敛半径为\(R\)，和函数为\(S(x)\)，则

  • 连续性：\(S(x)\)在收敛域上连续.
  • 可导性：\(S(x)\)在收敛区间\((-R, R)\)上可导且逐项可导. $$\begin{align} S'(x) &= (\sum\limits_{n = 0}^\infty a_n x^n)' = \sum\limits_{n = 0}^\infty (a_n x^n)'\\ &= \sum\limits_{n = 0}^\infty na_n x^{n-1} = \sum\limits_{n = 1}^\infty na_n x^{n-1} \end{align}$$

    该幂级数与原幂级数有相同的收敛半径\((-R, R).\)

  • 可积性：\(S(x)\)在收敛域上可积且逐项可积. $$\begin{align} \int_0^x S(t)\mathrm{d}t &= \int_0^x (\sum\limits_{n=0}^\infty a_nt^n)\mathrm{d}t = \sum\limits_{n=0}^\infty\int_0^x (a_nt^n)\mathrm{d}t\\ &= \sum\limits_{n=0}^\infty \frac{a_n}{n + 1}t^{n+1}|_0^x = \sum\limits_{n=0}^\infty \frac{a_n}{n+1}x^{n+1} \end{align}$$

    该幂级数与原幂级数有相同的收敛半径\((-R, R).\)

和函数

$$S(x) = \lim\limits_{n\rightarrow\infty}S_n$$

幂级数

求解幂级数的收敛半径

不缺项幂级数\(\sum\limits_{n = 0}^\infty a_n x^n\) $$R = \frac{1}{\lim\limits_{n\rightarrow \infty}|\frac{a_{n+1}}{a_n}|} = \frac{1}{\lim\limits_{n\rightarrow \infty}\sqrt[n]{|a_n|}}$$ 缺项幂级数

例如仅含偶数项幂级数\(\sum\limits_{n = 0}^\infty a_{2n} x^{2n}\)或仅含奇数项幂级数\(\sum\limits_{n = 0}^\infty a_{2n+1} x^{2n+1}\)

满足\(\lim\limits_{n\rightarrow \infty}|\frac{u_{n+1}(x)}{u_n(x)}|\lt 1\)的范围即为幂级数的收敛区间.

函数的幂级数展开

设函数\(f(x)\)在开区间\((x_0 - R, x_0 + R)\)内有定义，若

$$f(x) = \sum\limits_{n=0}^\infty a_n(x-x_0)^n$$ 对\(\forall x \in (x_0-R, x_0+R)\)都成立，则称函数\(f(x)\)在区间\((x_0-R, x_0+R)\)上能展开为\(x-x_0\)的幂级数.

函数\(f(x)\)在区间\((x_0-R, x_0+R)\)上能展开为\(x-x_0\)的幂级数\(\Rightarrow f(x)\)在区间\((x,x_0)\)上有任意阶导数，且展开式是唯一的.

$$a_n = \frac{f^{(n)}(x_0)}{n!}, n=0,1,2,\cdots$$

泰勒级数与麦克劳林级数

傅里叶级数

设函数\(f(x)\)是周期为\(2\pi\)的周期函数，且在\([-\pi, \pi]\)上可积，若

$$f(x) \sim \frac{a_0}{2} + \sum\limits_{n=1}^\infty(a_n\cos nx + b_n\sin nx)$$ 称其为\(f(x)\)的傅里叶级数. $$a_n = \frac{1}{\pi} \int_{-\pi}^\pi f(x) \cos nx \mathrm{d}x, n=0,1,2,\cdots$$ $$b_n = \frac{1}{\pi} \int_{-\pi}^\pi f(x) \sin nx \mathrm{d}x, n=1,2,\cdots$$ 称为\(f(x)\)的傅里叶系数.

收敛定理（狄利克雷条件）

设\(f(x)\)为周期函数，若其满足：

  • 在一个周期内连续或只有有限个第一类间断点
  • 在一个周期内至多只有有限个极值点

则\(f(x)\)的傅里叶级数收敛，且：

  • \(x_0\)为\(f(x)\)连续点\(\Rightarrow\)级数收敛于\(f(x_0)\)
  • \(x_0\)为\(f(x)\)间断点\(\Rightarrow\)级数收敛于\(\frac{1}{2}[f(x_0^-) + f(x_0^+)]\)

周期为\(2\pi\)函数\(f(x)\)的展开

奇函数在\([-\pi, \pi\)上的傅里叶级数 $$a_n = 0, n = 0,1,2, \cdots$$ $$b_n = \frac{2}{\pi}\int_0^\pi f(x)\sin nx \mathrm{d}x, n=1,2,\cdots$$ \(f(x)\)为偶函数 $$a_n = \frac{2}{\pi}\int_0^\pi f(x)\cos nx \mathrm{d}x, n=0,1,2\cdots$$ $$b_n = 0, n=1,2,\cdots$$ 在\([0,\pi]\)上函数\(f(x)\)的正弦级数和余弦级数

（1）将\(f(x)\)延拓成\([-\pi, \pi]\)上的奇函数，则

$$a_0, n=0,1,2\cdots$$ $$b_n = \frac{2}{\pi}\int_0^\pi f(x)\sin nx \mathrm{d}x, n=1,2,\cdots$$

\(f(x)\)的正弦级数为：

$$f(x) \sim \sum\limits_{n=1}^\infty b_n\sin nx$$

（2）将\(f(x)\)延拓成\([-\pi, \pi]\)上的偶函数，则

$$a_n = \frac{2}{\pi}\int_0^\pi f(x)\cos nx \mathrm{d}x, n=0,1,2\cdots$$ $$b_n = 0, n=1,2,\cdots$$

\(f(x)\)的余弦级数为：

$$f(x)\sim \frac{a_0}{2} + \sum\limits_{n=1}^\infty a_n\cos nx$$ 周期为\(2l\)的函数的展开

在\([-l,l]\)上函数\(f(x)\)的傅里叶级数为

$$f(x)\sim \frac{a_0}{2} + \sum\limits_{n=1}^\infty(a_n\cos \frac{n\pi}{l}x + b_n\sin\frac{n\pi}{l}x)$$ $$a_n = \frac{1}{l}\int_{-l}^{l}f(x)\cos\frac{n\pi}{l}x\mathrm{d}x, n=0,1,2,\cdots$$ $$b_n = \frac{1}{l}\int_{-l}^lf(x)\sin\frac{n\pi}{l}x\mathrm{d}x, n=1,2,\cdots$$ 在\([-l,l]\)上奇偶函数的傅里叶级数展开

\(f(x)\)为奇函数

$$a_n = 0, n=0,1,2,\cdots$$ $$b_n = \frac{2}{l}\int_0^lf(x)\sin\frac{n\pi}{l}x\mathrm{d}x, n=1,2,\cdots$$ $$f(x)\sim \sum\limits_{n=1}^\infty b_n\sin\frac{n\pi}{l}x$$

\(f(x)\)为偶函数

$$a_n = \frac{2}{l}\int_0^{l}f(x)\cos\frac{n\pi}{l}x\mathrm{d}x, n=0,1,2,\cdots$$ $$b_n = 0, n=1,2,\cdots$$ $$f(x)\sim \frac{a_0}{2} + \sum\limits_{n=1}^\infty a_n\cos \frac{n\pi}{l}x$$ 在\([0,l]\)上\(f(x)\)的正弦级数和余弦级数

（1）将\(f(x)\)延拓成\([-l,l]\)上的奇函数，则

$$a_n = 0, n=0,1,2,\cdots$$ $$b_n = \frac{2}{l}\int_0^lf(x)\sin\frac{n\pi}{l}x\mathrm{d}x, n=1,2,\cdots$$ $$f(x)\sim \sum\limits_{n=1}^\infty b_n\sin\frac{n\pi}{l}x$$

（1）将\(f(x)\)延拓成\([-l,l]\)上的偶函数，则

$$a_n = \frac{2}{l}\int_0^{l}f(x)\cos\frac{n\pi}{l}x\mathrm{d}x, n=0,1,2,\cdots$$ $$b_n = 0, n=1,2,\cdots$$ $$f(x)\sim \frac{a_0}{2} + \sum\limits_{n=1}^\infty a_n\cos \frac{n\pi}{l}x$$
