第四章 不定积分

一、不定积分概念与性质

原函数

若\(F'(x) = f(x)\)，称\(F(x)\)为\(f(x)\)的一个原函数。

若\(F(x)\)为\(f(x)\)的原函数，则\(F(x) + C\)也是\(f(x)\)的一个原函数。

不定积分

\(f(x)\)原函数的全体称为\(f(x)\)的不定积分。

$$\int f(x) \mathrm{d}x = F(x) + C$$ 原函数存在定理 $$f(x)在区间U内连续\Rightarrow f(x)在区间U上原函数F(x)存在$$ $$f(x)在区间U内存在第一类间断点\Rightarrow f(x)在区间U上原函数F(x)不存在$$ 常用函数不定积分 $$\int \frac 1x \mathrm{d}x = \ln |x| + C$$ $$\int \frac{1}{\sqrt{a^2 - x^2}}\mathrm{d}x = \arcsin\frac{x}{a} + C$$ $$\int \frac{1}{a^2 + x^2}\mathrm{d}x = \frac{1}{a}\arctan\frac{x}{a} + C$$ $$\begin{align} \int \frac{1}{x^2 - a^2}\mathrm{d}x &= \int \frac{1}{2a}(\frac{1}{x-a} - \frac{1}{x+a})\mathrm{d}x\\ &= \frac{1}{2a}\ln|\frac{x-a}{x+a}| + C \end{align}$$ $$\int \sec x\mathrm{d}x = \ln|\sec x + \tan x| + C$$ $$\begin{align} \int \csc x\mathrm{d}x &= -\ln|\csc x + \cot x| + C\\ &= \ln|\csc x - \cot x| + C \end{align}$$

三角有理化

设\(\tan \frac{x}{2} = t\)，则\(x = 2\arctan t, \mathrm{d}x = \frac{2}{1 + t^2}\)

$$\sin x = \frac{2\sin\frac{x}{2}\cos\frac{x}{2}}{\sin^2\frac{x}{2}+\cos^2\frac{x}{2}} = \frac{2\tan\frac{x}{2}}{\tan^2\frac{x}{2}+1} = \frac{2t}{t^2 + 1}$$
$$\cos x = \frac{\cos^2 \frac{x}{2}-\sin^2 \frac{x}{2}}{\sin^2\frac{x}{2} + \cos^2\frac{x}{2}} = \frac{1 - t^2}{t^2 + 1 }$$
$$\tan x = \frac{\sin x}{\cos x} = \frac{2t}{1-t^2}$$

解题思路

函数连续\(\Rightarrow\)存在原函数

函数不连续\(\Rightarrow\)判断间断点类型

  • 若为第二类间断点\(\Rightarrow\)构造出\(F'(x) = f(x)\)
