不等式

基本不等式

完全平方式：\((\sqrt{x} - \sqrt{y})^2\geq 0\)

基本不等式：\(x + y \geq 2\sqrt{xy}\quad (x\gt 0, y>0)\)

当\(x = y\)时，基本不等式才能取等号。

考点
  • 和积式（基本）
  • 非齐次式：凑
  • 齐次式：换元\(\rightarrow\)“\(1\)”的代换

常用不等式

$$||a| - |b||\leq |a \pm b| \leq |a| + |b|$$
\(ab\geq 0\)时： $$||a|-|b|| = |a - b| \leq |a + b| = |a| + |b|$$ \(ab \leq 0\)时： $$||a|-|b|| = |a + b| \leq |a - b| = |a| + |b|$$
