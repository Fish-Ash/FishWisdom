第六章 定积分的应用

平面图形面积

设\(D\)是由\(y=f(x),y=g(x),x=a,x=b(f(x)\geq g(x))\)围成的平面区域：

$$S = \int_a^b [f(x) - g(x)]\mathrm{d}x$$

设\(D\)是由\(r=r_1(\theta), r=r_2(\theta), \theta=\alpha, \theta = \beta(r_1(\theta)\geq r_2(\theta))\)围成的平面区域：

$$S = \frac12\int_\alpha^\beta[r_1^2(\theta)-r_2^2(\theta)]\mathrm{d}\theta$$

[注]积分变量可以取\(x\)或\(y\).

[注]原方程较复杂时，可以化为参数方程计算.

[注]有时无需纠结函数的图像形状.

旋转体体积

设\(D\)是由\(y=f(x),y=0,x=a,x=b\)围成的平面区域。

\(D\)绕\(x\)轴旋转一周所得旋转体体积：

$$V = \pi \int_a^b f^2(x)\mathrm{d}x$$

\(D\)绕\(y\)轴旋转一周所得旋转体体积：

$$V = 2\pi\int_a^bxf(x)\mathrm{d}x$$

曲线弧长

曲线\(C\)：\(y = f(x), a\leq x\leq b\)的弧长：

$$L = \int_a^b \sqrt{1+[f'(x)]^2}\mathrm{d}x$$
$$\Delta L = \sqrt{(\Delta x)^2 + (\Delta y)^2} = \sqrt{1 + (\frac{\Delta y}{\Delta x})^2}\Delta x$$ $$L = \lim\limits_{n\rightarrow \infty}\sum\limits_{k=1}^n\sqrt{1 + (\frac{\Delta y}{\Delta x})^2}\Delta x$$

曲线\(C\)：\(\begin{cases}x = x(t)\\y = y(t)\end{cases}, a\leq x\leq b\)的弧长：

$$L = \int_a^b \sqrt{[x'(t)]^2 + [y'(t)]}\mathrm{d}t$$

曲线\(C\)：\(r = r(\theta), \alpha \leq \theta \leq \beta\)的弧长：

$$L = \int_\alpha^\beta \sqrt{r^2(\theta) + [r'(\theta)]^2}\mathrm{d}\theta$$

旋转体的侧面积

由\(y=f(x)\geq 0, x=a, x=b\)以及\(x\)轴围成的区域绕\(x\)轴旋转所得旋转体的侧面积：

$$S = 2\pi\int_a^bf(x)\sqrt{1+[f'(x)]^2}\mathrm{d}x$$
