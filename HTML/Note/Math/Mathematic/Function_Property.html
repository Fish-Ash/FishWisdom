单调性

区间\(D\)内，\(\forall x_1 \lt x_2\)
单调递增 $$f(x_1) \lt f(x_2)$$
单调不减 $$f(x_1) \leq f(x_2)$$
单调递减 $$f(x_1) \gt f(x_2)$$
单调不增 $$f(x_1) \geq f(x_2)$$

奇偶性

  • 奇函数：定义在\((-l, l)\)上的函数，满足\(f(-x) = -f(x)\)
  • 偶函数：定义在\((-l, l)\)上的函数，满足\(f(-x) = f(x)\)
  • 既奇又偶函数：\(f(x) = 0\)
  • 非奇非偶函数
$$ \begin{align} 奇+奇 = 奇 &&偶+偶=偶 &&&奇+偶=?\\ 奇\times奇 = 偶 && 偶\times 偶 = 偶 &&&奇\times 偶 = 奇 \end{align} $$

奇函数\(f(0) = 0\)

周期性

若\(\exists T>0\)，对\(\forall x\)，恒有

$$f(x + T) = f(x)$$

则称\(f(x)\)为以\(T\)为周期的周期函数。

若存在使上式成立的最小正数\(T\)，则其被称为最小正周期。

不是所有的周期函数都有最小正周期，例如常数函数、迪利克雷函数。

若\(f(x)\)周期为\(T\)，则\(f(ax+b)\)周期为\(\frac{T}{|a|}\)

有界性

对于\(\forall x\in D, \exists M \gt 0\)
有界 $$|f(x)|\leq M$$
有上界 $$f(x) \leq M$$
有下界 $$f(x) \geq -M$$
