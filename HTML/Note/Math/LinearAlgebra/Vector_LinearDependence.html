向量组的线性相关性

线性相关与线性无关

设\(\alpha_1, \alpha_2, \cdots, \alpha_s\)为\(s\)个\(m\)维向量，若存在不全为\(0\)的数\(k_1, k_2, \cdots, k_s\)，使得

$$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s = 0$$

称向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)线性相关.

否则，称向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)线性无关.

结论

一个向量\(\alpha\)线性相关\(\Leftrightarrow \alpha=0\).

向量组\(\alpha_1, \alpha_2\)线性相关\(\Leftrightarrow \alpha_1, \alpha_2\)成比例.

含有零向量的向量组必线性相关.

由\(n\)个\(n\)维单位向量

$$\epsilon_1 = (1,0,\cdots,0)^T, \epsilon_2 = (0,1,\cdots,0)^T, \cdots, \epsilon_n = (0,0,\cdots, 1)^T$$

组成的向量组线性无关.

设

$$k_1\epsilon1 + k_2\epsilon_2 + \cdots + k_n\epsilon_n = \mathbf{0}$$ $$\therefore (k_1, k_2, \cdots, k_n)^T = (0,0,\cdots,0)^T$$ $$\therefore k_1 = k_2 = \cdots = k_n = 0$$

故向量组\(\epsilon_1, \epsilon_2, \cdots, \epsilon_n\)线性无关.

向量组\(\alpha_1, \alpha_2, \alpha_3\)线性无关\(\Leftrightarrow\)向量组\(\alpha_1 + \alpha_2, \alpha_1 + \alpha_3, \alpha_2 + \alpha_3\)线性无关.

向量组线性相关性的判定

由齐次线性方程组解的情况判定向量组的线性相关性

对于向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)

$$\mathbf{A} = (\alpha_1, \alpha_2, \cdots, \alpha_n)$$

以\(\mathbf{A}\)为系数矩阵的齐次方程组存在非零解时，向量组线性相关.

以\(\mathbf{A}\)为系数矩阵的齐次方程组仅有零解时，只有\(0\alpha_1 + 0\alpha_2 + \cdots + 0\alpha_n = 0\)成立，向量组线性无关.

向量组线性相关性的判定

由齐次线性方程组解的情况判定

线性相关\(\Leftrightarrow\)有非零解.

线性无关\(\Leftrightarrow\)仅有零解.

对于向量数大于\(1\)的向量组：

向量组线性相关\(\Leftrightarrow\)向量组中至少存在一个向量可由其他向量线性表示.

向量组线性无关\(\Leftrightarrow\)向量组中任一向量均不能由其他向量线性表示.

设向量组为\(\alpha_1, \alpha_2, \cdots, \alpha_n\).

充分性：

设\(\alpha_t\)可由其他向量线性表示

$$\alpha_t = k_1\alpha_1 + k_2\alpha_2 + \cdots + k_{t-1}\alpha_{t-1} + k_{t+1}\alpha_{t+1} + \cdots k_n\alpha_n$$ $$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_{t-1}\alpha_{t-1} + k_{t+1}\alpha_{t+1} + \cdots k_n\alpha_n - \alpha_t = 0$$

\(-1\neq 0\)即\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性相关.

必要性：

\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性相关

$$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_{t-1}\alpha_{t-1} + k_t\alpha_t + k_{t+1}\alpha_{t+1} + \cdots k_n\alpha_n = 0$$

设\(k_t \neq 0\)

$$\alpha_t = -\frac{k_1}{k_t}\alpha_1 -\frac{k_2}{k_t}\alpha_2 - \cdots - \frac{k_{t-1}}{k_t}\alpha_{t-1} - \frac{k_{t+1}}{k_t}\alpha_{t+1} - \cdots -\frac{k_n}{k_t}\alpha_n $$

即\(\alpha_t\)可由其他向量线性表示.

定理

向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性无关而\(\alpha_1, \alpha_2, \cdots, \alpha_n, \beta\)线性相关\(\Leftrightarrow\beta\)可由\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性表示，且表达式唯一。

\(\because \alpha_1, \alpha_2, \cdots, \alpha_n, \beta\)线性相关.

\(\therefore\)存在一组不全为为\(0\)的数，使得

$$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_n\alpha_n + k\beta = 0$$

要证\(\beta\)可由\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性表示，即证\(k\neq 0\).

反证法：假设\(k = 0\).

则存在不全为\(0\)的一组\(k_1, k_2, \cdots, k_n\)，使得\(k_1\alpha_1 + k_2\alpha_2 + \cdots + k_n\alpha_n = 0\).

这与“向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性无关”的条件矛盾，反证法不成立，即\(k\neq 0\).

$$\therefore \beta = -\frac{k_1}{k}\alpha_1 -\frac{k_2}{k}\alpha_2 - \cdots -\frac{k_n}{k}\alpha_n$$

故\(\beta\)可由\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性表示.


设\(\beta\)有两个表达式

$$\beta = \lambda_1\alpha_1 + \lambda_2\alpha_2 + \cdots + \lambda_n\alpha_n$$ $$\beta = \mu_1\alpha_1 + \mu_2\alpha_2 + \cdots + \mu_n\alpha_n$$

两式相减得

$$(\lambda_1-\mu_1)\alpha_1 + (\lambda_2-\mu_2)\alpha_2 + \cdots + (\lambda_n-\mu_n)\alpha_n = 0$$

\(\because\)向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性无关

$$\therefore \lambda_i = \mu_i(i=1,2,\cdots,n)$$

故表达式唯一.

定理

一个向量组中的某个子向量组线性相关\(\Rightarrow\)该向量组线性相关.

向量组线性无关\(\Rightarrow\)其任一子向量组也线性无关.

定理

\(r\)维向量组

$$ \begin{align} &\alpha_1 = (a_{11}, a_{12}, \cdots, a_{1r})^T\\ &\alpha_2 = (a_{12}, a_{22}, \cdots, a_{2r})^T\\ &\cdots\cdots\\ &\alpha_n = (a_{1n}, a_{2n}, \cdots, a_{nr})^T \end{align} $$ 中每个向量添加\(m-r\)个分量得到 $$ \begin{align} &\beta_1 = (a_{11}, a_{12}, \cdots, a_{1r}, a_{1,r+1}, \cdots, a_{1m})^T\\ &\beta_2 = (a_{12}, a_{22}, \cdots, a_{2r}, a_{2,r+1}, \cdots, a_{2m})^T\\ &\cdots\cdots\\ &\beta_n = (a_{1n}, a_{2n}, \cdots, a_{nr}, a_{n,r+1}, \cdots, a_{nm})^T \end{align} $$

\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性无关\(\Rightarrow \beta_1, \beta_2, \cdots, \beta_n\)线性无关.

\(\beta_1, \beta_2, \cdots, \beta_n\)线性相关\(\Rightarrow \alpha_1, \alpha_2, \cdots, \alpha_n\)线性相关.

定理

\(n\)维向量组\(\alpha_1, \alpha_2, \cdots, \alpha_r\)可由\(n\)维向量组\(\beta_1, \beta_2, \cdots, \beta_s\)线性表示，且\(r\gt s\)，则向量组\(\alpha_1, \alpha_2, \cdots, \alpha_r\)线性相关.

推论1

\(n\)维向量组\(\alpha_1, \alpha_2, \cdots, \alpha_r\)可由\(n\)维向量组\(\beta_1, \beta_2, \cdots, \beta_s\)线性表示，且向量组\(\alpha_1, \alpha_2, \cdots, \alpha_r\)线性无关，则\(r\leq s\).

推论2

向量组\(\alpha_1, \alpha_2, \cdots, \alpha_r\)与向量组\(\beta_1, \beta_2, \cdots, \beta_s\)等价且均线性无关，则\(r = s\).
