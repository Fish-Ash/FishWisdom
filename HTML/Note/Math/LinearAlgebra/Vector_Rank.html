向量组的秩

极大无关组

从向量组中选取一个子向量组，若子向量组线性无关，且原向量组中的每个向量都可由子向量组线性表示，则成子向量组为原向量组的一个极大线性无关向量组，简称极大无关组.

[注]向量组的极大无关组不唯一.

[注]只含零向量的向量组没有极大无关组.

[注]线性无关向量组的极大无关组就是它本身.

定理

任一向量组与其极大无关组等价.

推论

向量组中任意两极大无关组等价，从而这两个极大无关组中所含向量个数相等.

向量组的秩

向量组的极大无关组所含向量的个数，记作\(R(\alpha_1, \alpha_2, \cdots, \alpha_n)\)

[注]若一个向量仅含零向量，规定其秩为零.

定理

向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)可由向量组\(\beta_1, \beta_2, \cdots, \beta_m\)线性表示\(\Leftrightarrow R(\alpha_1, \alpha_2, \cdots, \alpha_n)\leq R(\beta_1, \beta_2, \cdots, \beta_m)\)

推论

若向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)与向量组\(\beta_1, \beta_2, \cdots, \beta_m\)等价\(\Leftrightarrow R(\alpha_1, \alpha_2, \cdots, \alpha_n) = R(\beta_1, \beta_2, \cdots, \beta_m)\)

向量组秩与矩阵秩的关系

矩阵\(\mathbf{A} = (a_{ij})_{m\times n}\)按行、列分块为

$$\mathbf{A} = \left(\begin{matrix}\alpha_1^T\\\alpha_2^T\\\vdots\\\alpha_m^T\end{matrix}\right) = (\beta_1, \beta_2, \cdots, \beta_n)$$

\(\mathbf{A}\)的行秩：\(R(\alpha_1^T, \alpha_2^T, \cdots, \alpha_m^T)\)

\(\mathbf{A}\)的列秩：\(R(\beta_1, \beta_2, \cdots, \beta_n)\)

$$R(\mathbf{A}) = R(\alpha_1^T, \alpha_2^T, \cdots, \alpha_m^T) = R(\beta_1, \beta_2, \cdots, \beta_n)$$

求向量组的秩、极大无关组并将其他向量用极大无关组表示的方法

对矩阵施以初等行变换，不改变矩阵列向量组的线性关系.

对于给定的向量组，将其组合成矩阵，而后化为行最简型矩阵的形式.

向量组的秩

向量组的秩即为行最简型矩阵的非零行数即矩阵的秩.

向量组的一个极大无关组

含非零首元列对应的向量组即为极大无关组.

用极大无关组表示其余向量

行最简型行列式中，极大无关组对应的列均为单位列向量，非极大无关组对应列用单位列向量表示的方式，即为极大无关组表示向量组中其他向量的方式.
