题型

求\(\mathbf{A}^n\)

题目条件：\(\mathbf{A} = \mathbf{\alpha}\mathbf{\beta}\)

思路：\(\mathbf{A}^n = (\mathbf{\alpha}\mathbf{\beta})^n = \mathbf{\alpha}(\mathbf{\beta}\mathbf{\alpha})^{n-1}\mathbf{\beta}\)

题目条件：\(\left(\begin{matrix}\mathbf{X_1}\\&\mathbf{X_2}\\&&\ddots\\&&&\mathbf{X}_t\end{matrix}\right)\)

思路：\(\mathbf{A}^n = \left(\begin{matrix}\mathbf{X_1}^n\\&\mathbf{X_2}^n\\&&\ddots\\&&&\mathbf{X}_t^n\end{matrix}\right)\)

已知矩阵方程\(f(\mathbf{A})\)，求\(g(\mathbf{A})\)的逆矩阵

思路：将\(f(\mathbf{A})\)化为\(g(\mathbf{A})\phi(\mathbf{A}) = \mathbf{E}\)的形式，\(\phi(\mathbf{A}) = g^{-1}(\mathbf{A})\)

要点：\(\mathbf{A} = \mathbf{A}\mathbf{E}\)

已知\(R\)，求未知数\(k\)

思路：通过基本行/列变换，化简矩阵，发现有\(t\)行/列无法依靠不含参数列继续化简.

此时参数的值影响这些列能否被化简为0.
