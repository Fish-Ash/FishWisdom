向量

\(n\)维向量

由\(n\)个数\(a_1, a_2, \cdots a_n\)组成的有序数组

$$(a_1, a_2, \cdots, a_n)~or~ \left(\begin{matrix}a_1\\a_2\\\vdots\\a_n\end{matrix}\right)$$

其中第\(i\)个数\(a_i(i=1,2,\cdots,n)\)称为\(n\)维向量的第\(i\)个分量.

\(n\)维实向量 分量全为实数的向量
\(n\)维复向量 含有复数的向量
\(n\)维行向量 \(1\times n\)矩阵
\(n\)维列向量 \(n\times1\)矩阵

[注]行（列）向量的转置即为列（行）向量.

[注]\(\alpha\)与\(\alpha^T\)常被视为不同向量.

零向量 $$\mathbf{0} = (0,0,\cdots 0)^T$$ 负向量 $$-\alpha = (-a_1, -a_2, \cdots, -a_n)^T$$

\(n\)维向量的线性运算

$$\alpha = (a_1, a_2, \cdots, a_n)^T$$ $$\beta = (b_1, b_2, \cdots, b_n)^T$$ 向量加法 $$\alpha + \beta = (a_1 + b_1, a_2 + b_2, \cdots, a_n + b_n)^T$$ 运算规律

交换律：\(\alpha + \beta = \beta + \alpha\)

结合律：\(\alpha + (\beta + \gamma) = (\alpha + \beta) + \gamma\)

\(\alpha + \mathbf{0} = \mathbf{0} + \alpha = \alpha\)

\(\alpha + (-\alpha) = 0\)

向量数乘 $$k\alpha = (ka_1, ka_2, \cdots, ka_n)^T$$ 运算规律

结合律：\(k(l\alpha) = (kl)\alpha\)

分配律：\(k(\alpha + \beta) = k\alpha + k\beta\)

分配律：\((k+l)\alpha = k\alpha + l\alpha\)

向量组的线性组合

给定一组向量\(\alpha_1, \alpha_2, \cdots, \alpha_s\)以及一个向量\(\beta\)，若存在一组数\(\lambda_1, \lambda_2, \cdots, \lambda_s\)，使得

$$\beta = \lambda_1\alpha_1 + \lambda_2\alpha_2 + \cdots + \lambda_s\alpha_s$$

称\(\beta\)可表示为\(\alpha_1, \alpha_2, \cdots, \alpha_s\)的线性组合.

或称\(\beta\)可由向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)线性表示.

其中，\(\lambda_1, \lambda_2, \cdots, \lambda_s\)称为组合系数.

[注]零向量可以表示为任一向量组的线性组合，因为可以取其组合系数均为\(0\).

定理

向量\(\beta\)可由向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性表示\(\Leftrightarrow \mathbf{A} = (\alpha_1, \alpha_2, \cdots, \alpha_n)\)的秩与\(\bar{\mathbf{A}} = (\alpha_1, \alpha_2, \cdots, \alpha_n, \beta)\)的秩相等.

任一\(n\)维向量\(\alpha = (a_1, a_2, \cdots, a_n)^T\)是\(n\)维向量组\(\epsilon_1 = (1,0,\cdots,0)^T, \epsilon_2 = (0,1,\cdots,0)^T, \cdots, \epsilon_n = (0,0,\cdots, 1)^T\)的线性组合.

$$\alpha = a_1\epsilon_1 + a_2\epsilon_2 + \cdots + a_n\epsilon_n$$

向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)中每个向量\(\alpha_i(i=1,2,\cdots,s)\)可由该向量组线性表示.

$$\alpha_i = 0\alpha_1 + \cdots + 0\alpha_{i-1} + 1\alpha_i + 0\alpha_{i+1} + \cdots + 0\alpha_s$$

题型

判断向量\(\beta\)是否为\(\alpha_1, \alpha_2, \cdots, \alpha_n\)的线性组合，若是，写出表达式.

对矩阵\(\bar{\mathbf{A}} = (\alpha_1, \cdots, \alpha_n, \beta)\)实施初等行变换，化为行最简形矩阵.

若\(R(\mathbf{A}) \neq R(\bar{\mathbf{A}})\)，则向量\(\beta\)不是\(\alpha_1, \alpha_2, \cdots, \alpha_n\)的线性组合.

若\(R(\mathbf{A}) = R(\bar{\mathbf{A}})\)，则增广矩阵的解即为系数.

向量组的等价

设有\(n\)维向量组

$$(Ⅰ):\alpha_1, \alpha_2, \cdots, \alpha_r$$ $$(Ⅱ):\beta_1, \beta_2, \cdots, \beta_s$$

若向量组\((Ⅰ)\)中的每个向量\(\alpha_i(i=1,2,\cdots, r)\)均可有向量组\((Ⅱ)\)线性表示，即

$$ \begin{align} &\alpha_1 = p_{11}\beta_1 + p_{21}\beta_2 + \cdots + p_{s1}\beta_s\\ &\alpha_2 = p_{12}\beta_1 + p_{22}\beta_2 + \cdots + p_{s2}\beta_s\\ &\cdots\cdots\\ &\alpha_r = p_{1r}\beta_1 + p_{2r}\beta_2 + \cdots + p_{sr}\beta_s\\ \end{align} $$

称向量组\((Ⅱ)\)可由向量组\(Ⅰ\)线性表示/

矩阵表示为

$$ (\alpha_1, \alpha_2, \cdots, \alpha_r) = (\beta_1, \beta_2, \cdots, \beta_s)\left(\begin{matrix}p_{11}&p_{12}&\cdots&p_{1r}\\p_{21}&p_{22}&\cdots&p_{2r}\\\vdots&\vdots&&\vdots\\p_{s1}&p_{s2}&\cdots&p_{sr}\end{matrix}\right) $$

记

$$\mathbf{P} = \left(\begin{matrix}p_{11}&p_{12}&\cdots&p_{1r}\\p_{21}&p_{22}&\cdots&p_{2r}\\\vdots&\vdots&&\vdots\\p_{s1}&p_{s2}&\cdots&p_{sr}\end{matrix}\right)$$ 则 $$(\alpha_1, \alpha_2, \cdots, \alpha_r) = (\beta_1, \beta_2, \cdots, \beta_s)\mathbf{P}$$

称\(\mathbf{P}\)为向量组\(\alpha_1, \alpha_2, \cdots, \alpha_r\)关于向量组\(\beta_1, \beta_2, \cdots, \beta_s\)的表示矩阵.

若向量组\((Ⅰ)\)可由向量组\((Ⅱ)\)线性表示，且向量组\((Ⅱ)\)也可由向量组\((Ⅱ)\)线性表示，那么称向量组\(Ⅰ\)与向量组\(Ⅱ\)等价.

向量组的线性相关性

设\(\alpha_1, \alpha_2, \cdots, \alpha_s\)为\(s\)个\(m\)维向量，若存在不全为\(0\)的数\(k_1, k_2, \cdots, k_s\)，使得

$$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_s\alpha_s = 0$$

称向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)线性相关.

否则，称向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)线性无关.

结论

一个向量\(\alpha\)线性相关\(\Leftrightarrow \alpha=0\).

两个向量构成的向量组线性相关\(\Leftrightarrow\)其对应分量成比例.

含有零向量的向量组必线性相关.

由\(n\)个\(n\)维单位向量

$$\epsilon_1 = (1,0,\cdots,0)^T, \epsilon_2 = (0,1,\cdots,0)^T, \cdots, \epsilon_n = (0,0,\cdots, 1)^T$$

组成的向量组线性无关.

向量组\(\alpha_1, \alpha_2, \alpha_3\)线性无关\(\Leftrightarrow\)向量组\(\alpha_1 + \alpha_2, \alpha_1 + \alpha_3, \alpha_2 + \alpha_3\)线性无关.

向量组线性相关性的判定

对于向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)

$$\mathbf{A} = (\alpha_1, \alpha_2, \cdots, \alpha_n)$$

\(R(\mathbf{A}) = n\Leftrightarrow\)向量组线性无关

\(R(\mathbf{A}) \lt n\Leftrightarrow\)向量组线性相关

\(R(\mathbf{A}) = n\)，即以\(\mathbf{A}\)为系数矩阵的齐次方程组仅有零解，即只有\(0\alpha_1 + 0\alpha_2 + \cdots + 0\alpha_n = 0\)成立，向量组线性无关.

\(R(\mathbf{A}) lt n\)，即以\(\mathbf{A}\)为系数矩阵的齐次方程组存在非零解，向量组线性相关.

推论1

\(n\)个\(n\)维列向量

$$ \begin{align} &\alpha_1 = (a_{11}, a_{21}, \cdots, a_{n1})^T\\ &\alpha_2 = (a_{12}, a_{22}, \cdots, a_{n2})^T\\ &\cdots\cdots\\ &\alpha_n = (a_{1n}, a_{2n}, \cdots, a_{nn})^T \end{align} $$

线性相关\(\Leftrightarrow |\mathbf{A}| = |\alpha_1, \alpha_2, \cdots, \alpha_n| = 0\)

线性无关\(\Leftrightarrow |\mathbf{A}| = |\alpha_1, \alpha_2, \cdots, \alpha_n| \neq 0\)

推论2

若向量组中向量的个数大于向量的维数，则该向量组必线性相关.

定理

向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性相关\(\Leftrightarrow\)向量组中至少存在一个向量可由其他向量线性表示.

充分性：

设\(\alpha_t\)可由其他向量线性表示

$$\alpha_t = k_1\alpha_1 + k_2\alpha_2 + \cdots + k_{t-1}\alpha_{t-1} + k_{t+1}\alpha_{t+1} + \cdots k_n\alpha_n$$ $$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_{t-1}\alpha_{t-1} + k_{t+1}\alpha_{t+1} + \cdots k_n\alpha_n - \alpha_t = 0$$

\(-1\neq 0\)即\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性相关.

必要性：

\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性相关

$$k_1\alpha_1 + k_2\alpha_2 + \cdots + k_{t-1}\alpha_{t-1} + k_t\alpha_t + k_{t+1}\alpha_{t+1} + \cdots k_n\alpha_n = 0$$

设\(k_t \neq 0\)

$$\alpha_t = -\frac{k_1}{k_t}\alpha_1 -\frac{k_2}{k_t}\alpha_2 - \cdots - \frac{k_{t-1}}{k_t}\alpha_{t-1} - \frac{k_{t+1}}{k_t}\alpha_{t+1} - \cdots -\frac{k_n}{k_t}\alpha_n $$

即\(\alpha_t\)可由其他向量线性表示.

定理

向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性无关而\(\alpha_1, \alpha_2, \cdots, \alpha_n, \beta\)线性相关\(\Leftrightarrow\beta\)可由\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性表示，且表达式唯一

定理

一个向量组中的某个子向量组线性相关\(\Rightarrow\)该向量组线性相关.

定理

\(r\)维向量组

$$ \begin{align} &\alpha_1 = (a_{11}, a_{12}, \cdots, a_{1r})^T\\ &\alpha_2 = (a_{12}, a_{22}, \cdots, a_{2r})^T\\ &\cdots\cdots\\ &\alpha_n = (a_{1n}, a_{2n}, \cdots, a_{nr})^T \end{align} $$ 中每个向量添加\(m-r\)个分量得到 $$ \begin{align} &\beta_1 = (a_{11}, a_{12}, \cdots, a_{1r}, a_{1,r+1}, \cdots, a_{1m})^T\\ &\beta_2 = (a_{12}, a_{22}, \cdots, a_{2r}, a_{2,r+1}, \cdots, a_{2m})^T\\ &\cdots\cdots\\ &\beta_n = (a_{1n}, a_{2n}, \cdots, a_{nr}, a_{n,r+1}, \cdots, a_{nm})^T \end{align} $$

\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性无关\(\Leftrightarrow \beta_1, \beta_2, \cdots, \beta_n\)线性无关.

\(\beta_1, \beta_2, \cdots, \beta_n\)线性相关\(\Leftrightarrow \alpha_1, \alpha_2, \cdots, \alpha_n\)线性相关.
