向量

\(n\)维向量

由\(n\)个数\(a_1, a_2, \cdots a_n\)组成的有序数组

$$(a_1, a_2, \cdots, a_n)~or~ \left(\begin{matrix}a_1\\a_2\\\vdots\\a_n\end{matrix}\right)$$

其中第\(i\)个数\(a_i(i=1,2,\cdots,n)\)称为\(n\)维向量的第\(i\)个分量.

\(n\)维实向量 分量全为实数的向量
\(n\)维复向量 含有复数的向量
\(n\)维行向量 \(1\times n\)矩阵
\(n\)维列向量 \(n\times1\)矩阵

[注]行（列）向量的转置即为列（行）向量.

[注]\(\alpha\)与\(\alpha^T\)常被视为不同向量.

零向量 $$\mathbf{0} = (0,0,\cdots 0)^T$$ 负向量 $$-\alpha = (-a_1, -a_2, \cdots, -a_n)^T$$

\(n\)维向量的线性运算

$$\alpha = (a_1, a_2, \cdots, a_n)^T$$ $$\beta = (b_1, b_2, \cdots, b_n)^T$$ 向量加法 $$\alpha + \beta = (a_1 + b_1, a_2 + b_2, \cdots, a_n + b_n)^T$$ 运算规律

交换律：\(\alpha + \beta = \beta + \alpha\)

结合律：\(\alpha + (\beta + \gamma) = (\alpha + \beta) + \gamma\)

\(\alpha + \mathbf{0} = \mathbf{0} + \alpha = \alpha\)

\(\alpha + (-\alpha) = 0\)

向量数乘 $$k\alpha = (ka_1, ka_2, \cdots, ka_n)^T$$ 运算规律

结合律：\(k(l\alpha) = (kl)\alpha\)

分配律：\(k(\alpha + \beta) = k\alpha + k\beta\)

分配律：\((k+l)\alpha = k\alpha + l\alpha\)

向量组的线性组合

给定一组向量\(\alpha_1, \alpha_2, \cdots, \alpha_s\)以及一个向量\(\beta\)，若存在一组数\(\lambda_1, \lambda_2, \cdots, \lambda_s\)，使得

$$\beta = \lambda_1\alpha_1 + \lambda_2\alpha_2 + \cdots + \lambda_s\alpha_s$$

称\(\beta\)可表示为\(\alpha_1, \alpha_2, \cdots, \alpha_s\)的线性组合.

或称\(\beta\)可由向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)线性表示.

其中，\(\lambda_1, \lambda_2, \cdots, \lambda_s\)称为组合系数.

[注]零向量可以表示为任一向量组的线性组合，因为可以取其组合系数均为\(0\).

定理

向量\(\beta\)可由向量组\(\alpha_1, \alpha_2, \cdots, \alpha_n\)线性表示\(\Leftrightarrow \mathbf{A} = (\alpha_1, \alpha_2, \cdots, \alpha_n)\)的秩与\(\bar{\mathbf{A}} = (\alpha_1, \alpha_2, \cdots, \alpha_n, \beta)\)的秩相等.

任一\(n\)维向量\(\alpha = (a_1, a_2, \cdots, a_n)^T\)是\(n\)维向量组\(\epsilon_1 = (1,0,\cdots,0)^T, \epsilon_2 = (0,1,\cdots,0)^T, \cdots, \epsilon_n = (0,0,\cdots, 1)^T\)的线性组合.

$$\alpha = a_1\epsilon_1 + a_2\epsilon_2 + \cdots + a_n\epsilon_n$$

向量组\(\alpha_1, \alpha_2, \cdots, \alpha_s\)中每个向量\(\alpha_i(i=1,2,\cdots,s)\)可由该向量组线性表示.

$$\alpha_i = 0\alpha_1 + \cdots + 0\alpha_{i-1} + 1\alpha_i + 0\alpha_{i+1} + \cdots + 0\alpha_s$$

题型

判断向量\(\beta\)是否为\(\alpha_1, \alpha_2, \cdots, \alpha_n\)的线性组合，若是，写出表达式.

对矩阵\(\bar{\mathbf{A}} = (\alpha_1, \cdots, \alpha_n, \beta)\)实施初等行变换，化为行最简形矩阵.

若\(R(\mathbf{A}) \neq R(\bar{\mathbf{A}})\)，则向量\(\beta\)不是\(\alpha_1, \alpha_2, \cdots, \alpha_n\)的线性组合.

若\(R(\mathbf{A}) = R(\bar{\mathbf{A}})\)，则增广矩阵的解即为系数.

向量组的等价

设有\(n\)维向量组

$$(Ⅰ):\alpha_1, \alpha_2, \cdots, \alpha_r$$ $$(Ⅱ):\beta_1, \beta_2, \cdots, \beta_s$$

若向量组\((Ⅰ)\)中的每个向量\(\alpha_i(i=1,2,\cdots, r)\)均可有向量组\((Ⅱ)\)线性表示，即

$$ \begin{align} &\alpha_1 = p_{11}\beta_1 + p_{21}\beta_2 + \cdots + p_{s1}\beta_s\\ &\alpha_2 = p_{12}\beta_1 + p_{22}\beta_2 + \cdots + p_{s2}\beta_s\\ &\cdots\cdots\\ &\alpha_r = p_{1r}\beta_1 + p_{2r}\beta_2 + \cdots + p_{sr}\beta_s\\ \end{align} $$

称向量组\((Ⅱ)\)可由向量组\(Ⅰ\)线性表示

矩阵表示为

$$ (\alpha_1, \alpha_2, \cdots, \alpha_r) = (\beta_1, \beta_2, \cdots, \beta_s)\left(\begin{matrix}p_{11}&p_{12}&\cdots&p_{1r}\\p_{21}&p_{22}&\cdots&p_{2r}\\\vdots&\vdots&&\vdots\\p_{s1}&p_{s2}&\cdots&p_{sr}\end{matrix}\right) $$

记

$$\mathbf{P} = \left(\begin{matrix}p_{11}&p_{12}&\cdots&p_{1r}\\p_{21}&p_{22}&\cdots&p_{2r}\\\vdots&\vdots&&\vdots\\p_{s1}&p_{s2}&\cdots&p_{sr}\end{matrix}\right)$$ 则 $$(\alpha_1, \alpha_2, \cdots, \alpha_r) = (\beta_1, \beta_2, \cdots, \beta_s)\mathbf{P}$$

称\(\mathbf{P}\)为向量组\(\alpha_1, \alpha_2, \cdots, \alpha_r\)关于向量组\(\beta_1, \beta_2, \cdots, \beta_s\)的表示矩阵.

等价

若向量组Ⅰ可由向量组Ⅱ线性表示，且向量组Ⅱ也可由向量Ⅰ线性表示，那么称向量组Ⅰ与向量组Ⅱ等价.

反身性

每一个向量组与自身等价.

对称性

向量组Ⅰ与向量组Ⅱ等价\(\Leftrightarrow\)向量组Ⅱ与向量组Ⅰ等价.

传递性

向量组Ⅰ与向量组Ⅱ等价，向量组Ⅱ与向量组Ⅲ等价\(\Leftrightarrow\)向量组Ⅰ与向量组Ⅲ等价.

向量的内积

向量的内积 $$\alpha = (x_1, x_2, \cdots, x_n)^T$$ $$\beta = (y_1, y_2, \cdots, y_n)^T$$ $$(\alpha, \beta) = \alpha^T\beta = x_1y_1 + x_2y_2 + \cdots + x_ny_n$$

称\((\alpha, \beta)\)为向量\(\alpha\)与向量\(\beta\)的内积.

$$(\alpha, \beta) = (\beta, \alpha)$$ $$(\lambda\alpha, \beta) = \lambda(\alpha, \beta)$$ $$(\alpha + \beta, \gamma) = (\alpha, \gamma) + (\beta, \gamma)$$ $$\alpha=\mathbf{0}\Rightarrow(\alpha, \alpha) = 0, \alpha\neq \mathbf{0} \Rightarrow (\alpha, \alpha)\gt 0$$
柯西-施瓦茨不等式 $$(\alpha, \beta)^2 \leq (\alpha, \alpha)(\beta, \beta)$$

向量的长度和夹角

向量的长度 $$\|\alpha\| = \sqrt{(\alpha, \alpha)} = \sqrt{x_1^2 + x_2^2 + \cdots + x_n^2}$$
单位向量

\(\|\alpha\| = 1\)的向量\(\alpha\).

向量标准化（单位化）

对于非零向量\(\alpha\)，\(e = \frac{\alpha}{\|a\|}\)为单位向量.

$$\|\lambda \alpha\| = |\lambda|\|\alpha\|$$
向量的夹角

\(\alpha \neq \mathbf{0}, \beta \neq \mathbf{0}\)时

$$\cos \theta = \frac{(\alpha, \beta)}{\|\alpha\|\|\beta\|}(0\leq \theta\leq \pi)$$

称\(\theta\)为\(\alpha\)与\(\beta\)的夹角.

正交

\((\alpha,\beta) = 0\)时，称\(\alpha\)与\(\beta\)正交，记作\(\alpha \perp \beta\).

[注]对于任意向量\(\alpha\)，总有\((\alpha, \mathbf{0}) = \mathbf{0}\)，故零向量与任何向量都正交.

正交向量组

若\(\alpha_1, \alpha_2, \cdots, \alpha_n\)为一个非零向量组，且其中的向量两两正交，则称其为正交向量组.

正交规范向量组（标准正交向量组）

所有向量均为单位向量的正交向量组.

定理

正交向量组线性无关.

特征值与特征向量

设\(\mathbf{A}\)为\(n\)阶矩阵，若数\(\lambda\)和\(n\)维非零列向量\(\alpha\)满足

$$\mathbf{A}\alpha = \lambda\alpha$$

称数\(\lambda\)为方阵\(\mathbf{A}\)的特征值.

非零列向量\(\alpha\)为\(\mathbf{A}\)对应于特征值\(\lambda\)的特征向量.

上式也可写成

$$(\lambda\mathbf{E} - \mathbf{A})\alpha = \mathbf{0}$$

这说明\(\alpha\)是齐次线性方程组\((\lambda\mathbf{E} - \mathbf{A})\mathbf{x} = \mathbf{0}\)的非零解.

方程组\((\lambda\mathbf{E} - \mathbf{A})\mathbf{x} = \mathbf{0}\)有非零解的充分必要条件为\(|\lambda\mathbf{E} - \mathbf{A}| = 0\)

矩阵\(\mathbf{A}\)的特征多项式 $$f(\lambda) = |\lambda\mathbf{E} = \mathbf{A}|$$ 矩阵\(\mathbf{A}\)的特征方程 $$f(\lambda) = |\lambda\mathbf{E} = \mathbf{A}| = 0$$

求矩阵特征值与特征向量的方法

特征值 $$f(\lambda) = |\lambda \mathbf{E} - \mathbf{A}| = 0 \Rightarrow \lambda_i$$ 特征向量 $$(\lambda_i \mathbf{E} - \mathbf{A})\alpha = 0 \Rightarrow \alpha$$

[注]\(\lambda\)需标注（\(k\)重），一重可以省略不写.

[注]\(\alpha\)的通解中\(k_i\)不同时为\(0\).

方阵的迹

方阵主对角线上的元素之和

$$Tr(\mathbf{A}) = a_{11} + a_{22} + \cdots + a_{nn}$$
方阵特征值的性质

\(n\)阶方阵\(\mathbf{A}\)的特征值为\(\lambda_1, \lambda_2, \cdots, \lambda_n\)（包含重根）.

$$\sum\limits_{i=1}^n\lambda_i = Tr(\mathbf{A})$$ $$\lambda_1\lambda_2\cdots\lambda_n = |\mathbf{A}|$$

\(n\)阶方阵可逆\(\Leftrightarrow \lambda_i\neq 0(i=1,2,\cdots, n )\)

\(n\)阶方阵\(\mathbf{A}\)与\(\mathbf{A}^T\)具有相同的特征值.

\(\lambda\)为\(n\)阶方阵\(\mathbf{A}\)的特征值，\(k,m\in \mathbb{N}^+\)，则

\(\lambda^k\)为\(\mathbf{A}^k\)的特征值.

\(f(\lambda) = a_0\lambda^m + a_1\lambda^{m-1} + \cdots + a_{m-1}\lambda + a_m\)是\(f(\mathbf{A}) = a_0\mathbf{A}^m + a_1\mathbf{A}^{m-1}+\cdots+a_{m-1}\mathbf{A} + a_m\mathbf{E}\)的特征值.

\(\lambda\)为可逆方阵\(\mathbf{A}\)的特征值，则

\(1/\lambda\)为\(\mathbf{A}^{-1}\)的特征值.

\(|\mathbf{A}|/\lambda\)为\(\mathbf{A}^*\)的特征值.

若\(\mathbf{p}_1, \mathbf{p}_2, \cdots, \mathbf{p}_m\)是\(n\)阶方阵\(\mathbf{A}\)属于互不相同特征值\(\lambda_1, \lambda_2, \cdots, \lambda_m\)的特征向量，则\(\mathbf{p}_1, \mathbf{p}_2, \cdots, \mathbf{p}_m\)线性无关.
