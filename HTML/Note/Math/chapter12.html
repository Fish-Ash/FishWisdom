第一节 常数项级数的概念与性质

一、常数项级数的概念

（常数项）级数：\(\mathop{\sum}\limits_{i=1}^\infty u_i = u_1 + u_2 +\cdots + u_n + \cdots\)，其中第n项\(u_n\)叫做级数的一般项。

部分和：\(s_n = \mathop{\sum}\limits_{i=1}^n u_i = u_1 + u_2 + \cdots + u_n\)

常见级数：

  • 等比级数（几何级数）： $$ \mathop{\sum}\limits_{i=0}^\infty aq^i = a + aq + aq^2 + \cdots +aq^n + \cdots\\ \begin{cases} |q|< 1时：收敛，s = \frac{a}{1-q}\\ |q| \geq 1 时：发散 \end{cases} $$

    $$s_n = a + aq + \cdots + aq^{n-1}\\ \begin{align} (1)~&|q| = 1时:\\ ①~&q = 1时:\\ &s_n = na\\ &\mathop{lim}\limits_{n\rightarrow \infty}s_n = \mathop{lim}\limits_{n\rightarrow \infty}a_n = \infty\\ &\therefore \mathop{\sum}\limits_{i=0}^\infty aq^i发散（q=1）\\ ②~&q = -1 时：\\ &s_n = \begin{cases} 0， n为偶数\\ a， n为奇数 \end{cases}\\ &\therefore \mathop{\sum}\limits_{i=0}^\infty aq^i发散（q=-1）\\ (2)~&|q|\neq 1时：\\ &s_n = \frac{a(1-q^n)}{1-q}\\ ①~&|q|>1时：\\ &\mathop{lim}\limits_{n\rightarrow \infty}s_n = \mathop{lim}\limits_{n\rightarrow \infty}\frac{a(1-q^n)}{1-q} = \infty\\ &\therefore \mathop{\sum}\limits_{i=0}^\infty aq^i发散（|q|> 1）\\ ②~&|q|< 1时：\\ &\mathop{lim}\limits_{n\rightarrow \infty}s_n = \mathop{lim}\limits_{n\rightarrow \infty}\frac{a(1-q^n)}{1-q} = \frac{a}{1-q}\\ &\therefore \mathop{\sum}\limits_{i=0}^\infty aq^i收敛（|q|< 1）\\ \end{align}\\ 综上所述：\\ \begin{cases} |q|< 1时：收敛，s = \frac{a}{1-q}\\ |q| \geq 1 时：发散 \end{cases} $$
