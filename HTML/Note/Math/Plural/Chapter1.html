复数

复数

设\(x,y\in \mathbb{R}\)，则形如：

$$z = x + iy$$

的数称为复数.

\(i\)为虚单位，有性质\(i^2 = -1\).

\(x\)为复数的实部，记作\(x = \mathrm{Re}z\).

\(y\)为复数的虚部，记作\(x = \mathrm{Im}z\).

[注]一般情况下，复数无法比较大小.

实数

虚部为\(0\)的复数，即\(\mathbb{R}\subset \mathbb{C}\).

纯虚数

实部为\(0\)且虚部不为\(0\)的复数.

共轭复数

实部相等，虚部互为相反数的两个复数称为共轭复数.

若一个复数记为\(z\)，则其共轭复数记为\(\overline{z}\)

$$\overline{x + iy} = x - iy$$
共轭复数的性质 $$\overline{\overline{z}} = z$$ $$\overline{z_1 \pm z_2} = \overline{z_1} \pm \overline{z_2}$$ $$\overline{z_1\cdot z_2} = \overline{z_1}\cdot \overline{z_2}$$ $$\overline{\left(\frac{z_1}{z_2}\right)} = \frac{\overline{z_1}}{\overline{z_2}},~z_2\neq0$$

复数的几何表示

对于\(\forall z = x+iy\)，总有平面直角坐标系上一点\(P(x,y)\)与之对应.

欧拉公式 $$e^{ix} = \cos{x} + i\sin{x}$$ 推导式
$$\cos x = \frac{e^{ix} + e^{-ix}}{2}$$
$$\begin{align} \frac{e^{ix} + e^{-ix}}{2} &= \frac{\cos x + i\sin x + \cos x - i\sin x}{2}\\ &= \cos x \end{align}$$
$$\sin x = \frac{e^{ix} - e^{-ix}}{2i}$$
$$\begin{align} \frac{e^{ix} - e^{-ix}}{2i} &= \frac{\cos x + i\sin x - \cos x + i\sin x}{2i}\\ &= \sin x \end{align}$$
