傅里叶变换

\(f(x)\)的傅里叶变换式

设函数\(f(t)\)满足傅里叶积分定理条件，则表达式：

$$F(\omega) = \int_{-\infty}^{+\infty}f(t)e^{-i\omega t}\mathrm{d}t$$

为\(f(t)\)的傅里叶变换式，记为\(F(\omega) = F[f(t)]\)，称函数\(F(\omega)\)为\(f(t)\)的傅里叶变换.

\(F(\omega)\)的傅里叶逆变换式 $$f(t) = \frac{1}{2\pi}\int_{-\infty}^{+\infty}F(\omega)e^{i\omega t}\mathrm{d}\omega$$

该表达式为\(F(\omega)\)的傅里叶逆变换式，记作\(f(t) = F^{-1}[F(\omega)]\)，称函数\(f(t)\)为\(F(\omega)\)的傅里叶逆变换.

傅里叶变换的性质

线性性质

$$F[k_1f_1(t) + k_2f_2(t)] = k_1F[f_1(t)] + k_2F[f_2(t)]$$ $$F^{-1}[k_1F_1(\omega) + k_2F_2(\omega)] = k_1F^{-1}[F_1(\omega)] + k_2F^{-1}[F_2(\omega)]$$

对称性质

$$F(\omega) = F[f(t)]\Rightarrow F[F(t)] = 2\pi f(-\omega)$$
$$\because F[f(t)] = \int_{-\infty}^{+\infty}f(t)e^{-i\omega t}\mathrm{d}t$$ $$\therefore F[F(t)] = \int_{-\infty}^{+\infty}F(t)e^{i\omega t}\mathrm{d}t$$ $$\because f(t) = \int_{-\infty}^{+\infty}F(\omega)e^{i\omega t}\mathrm{d}\omega$$ $$\begin{align} \therefore f(-\omega) &= \frac{1}{2\pi}\int_{-\infty}^{+\infty}F(x)e^{ix(-\omega)}\mathrm{d}x\\ &= \frac{1}{2\pi}\int_{-\infty}^{+\infty}F(t)e^{-i\omega t}\mathrm{d}t\\ &= \frac{1}{2\pi}F[F(t)] \end{align}$$ $$\therefore F[F(t)] = 2\pi f(-\omega)$$
卷积

给定定义在\((-\infty, +\infty)\)上的函数\(f_1(t)\)与\(f_2(t)\)，称

$$g(t) = \int_{-\infty}^{+\infty}f_1(\tau)f_2(t-\tau)\mathrm{d}\tau$$

为函数\(f_1(t)\)与\(f_2(t)\)的卷积，记作

$$g(t) = f_1(t) * f_2(t)$$

卷积的性质

交换性质

$$f_1(t)*f_2(t) = f_2(t)*f_1(t)$$

结合性质

$$f_1(t)*[f_2(t)*f_3(t)] = [f_1(t)*f_2(t)]*f_3(t)$$

线性性质

$$[k_1f_1(t) + k_2f_2(t)]*g(t) = k_1f_1(t)g(t) + k_2f_2(t)g(t),~k_1,k_2\in \mathbb{C}$$

平移不变性质

$$f_1(t)*f_2(t) = g(t) \Rightarrow f_1(t-\alpha)*f_2(t-\beta) = g(t-\alpha-\beta)$$

坐标放缩性质

$$f_1(t)*f_2(t) = g(t) \Rightarrow f_1(at)*f_2(at) = \frac{1}{|a|}g(at),~a\neq 0$$
