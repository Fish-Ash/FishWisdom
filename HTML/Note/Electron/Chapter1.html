Electron入门

环境配置

Node.js

一个基于Chrome V8引擎的JavaScript运行环境.

开发者使用Node.js创建Electron项目.

下载地址

下载完成后，若VSCode无法调用npm指令，需要使用管理员身份打开VSCode.

若无法执行 npm install 指令，将node.js所在文件夹的权限修改为不需要管理员身份.

yarn

依赖包管理工具.

下载方式：


            # 命令
            npm install -g yarn
创建第一个Electron应用 

            # 命令
            yarn init

该命令执行完成后，会有一系列提示，要求用户输入项目名称、项目版本、作者等信息，若按默认设置全部回车即可.

执行这条命令将会生成一个package.js文件.

安装Electron依赖包

下载地址


            yarn config set ELECTRON_MIRROR https://github.com/electron/electron/releases
