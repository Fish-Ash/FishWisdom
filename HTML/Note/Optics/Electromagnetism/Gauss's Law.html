Gauss's Law

公式 $$\oint_S\mathbf{D}\mathrm{d}\mathbf{S} = Q$$ \(\mathbf{D}\) 电位移矢量
\(\mathbf{S}\) 闭合曲面
\(Q\) 闭合面内自由电荷代数和
描述 静电场中，通过任意闭合曲面的电位移通量等于闭合面内自由电荷的代数和。

证明

球面内点电荷



以点电荷\(q\)为中心，取任意长度\(r\)为半径做闭合球面\(S\)。在\(S\)上求面元d\(S\)，其法线\(\mathbf{n}\)与面元处的电场强度\(\mathbf{E}\)方向相同，故通过\(\mathrm{d}S\)的电通量为：

$$\mathrm{d}\Phi_e = \mathbf{E}\mathrm{d}\mathbf{S} = E\cos0\mathrm{d}S = \frac{1}{4\pi\epsilon_0}\frac{q}{r^2}\mathrm{d}S$$

通过整个闭合球面\(S\)的电通量为：

$$\Phi_e = \oint_S\mathrm{d}\Phi_e = \oint_S \frac{1}{4\pi\epsilon_0}\frac{q}{r^2}\mathrm{d}S = \frac{1}{4\pi\epsilon_0}\frac{q}{r^2}\cdot 4\pi r^2 = \frac{q}{\epsilon_0}$$

任意闭合曲面内点电荷

作一任意闭合曲面\(S'\)包围点电荷，易知穿过\(S'\)的电场线根数与穿过\(S\)的电场线根数相等，即：

$$\Phi_e = \frac{q}{\epsilon_0}$$

任意闭合曲面外点电荷

在点电荷外作一任意闭合曲面\(S''\)，其外部的电荷发出的电场线进入该闭合曲面的根数与穿过闭合曲面的根数相等，故闭合曲面外的点电荷对该闭合曲面上的电场强度无贡献，即：

$$\Phi_e = \frac{q}{\epsilon_0} = 0$$

任意带电系统的电场

根据场强叠加原理，有：

$$\Phi_e = \oint_S\mathbf{E}\cdot\mathrm{d}\mathbf{S} = \sum\limits_{i=1}^n\oint_S\mathbf{E}_i\cdot\mathrm{d}\mathbf{S} = \frac{\sum q_i}{\epsilon_0}$$

电介质中

电介质中，总电场\(\mathbf{E}\)是自由电荷电场强度\(\mathbf{E}_0\)和极化电荷电场强度\(\mathbf{E}'\)的叠加。

$$\oint_S\mathbf{E}\mathrm{d}\mathbf{S} = \frac{1}{\epsilon_0}(\sum q_i + \sum q_i')$$ $$\because \oint_S\mathbf{P}\mathrm{d}\mathbf{S} = -\sum q_i'$$极化强度与极化电荷的关系式 $$\therefore \oint_S\mathbf{D}\mathrm{d}\mathbf{S} = \oint_S \epsilon_0\mathbf{E} + \mathbf{P}\mathrm{d}\mathbf{S} = \sum q_i = Q$$

没有介质存在时，\(\mathbf{D} = \epsilon_0\mathbf{E_0}\)，公式依然成立。
