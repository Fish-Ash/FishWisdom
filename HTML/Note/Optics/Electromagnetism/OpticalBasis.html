光学基础

光的波粒二象性

光既是电磁波（波动性），又是粒子流（粒子性）。

  • 光在传播过程中主要表现为波动性。
  • 光与物质发生能量交换时表现为粒子性。

光的电磁波性质

电磁波

变化的电磁场。

电磁矢量以速度\(v\)按波动形式在介质中传播。

$$v = 1/\sqrt{\mu\epsilon}$$

光本质上是一种电磁波。

光速\(c\)

真空中电磁波的传播速度

$$c = 1/\sqrt{\mu_0\epsilon_0}\approx2.9979\times 10^8m/s$$ 式中，\(\mu_0\)为真空中的磁导率,\(\epsilon_0\)为真空中的介电常数

光在介质中的传播速度都小于\(c\)。

介质的折射率\(n\)

电磁波在真空中的速度\(c\)与在不导电的均匀介质中的速度\(v\)之比

$$n = \frac{c}{v}=\frac{\sqrt{\mu\epsilon}}{\sqrt{\mu_0\epsilon_0}} = \sqrt{\mu_r\epsilon_r}$$ 式中，相对磁导率\(\mu_r = \mu/\mu_0\)

光的粒子性

光子能量公式：\(E = h\nu\)

光子动量公式：\(P = h\nu/c = h/\lambda\)

  • \(h = 6.626\times 10^{-34}\mathrm{J}\cdot \mathrm{S}\)：普朗克常量

光波的分类

紫外光 可见光 红外光
波段 \(\lambda\lt 380nm\) \(380nm \lt \lambda \lt 780nm\) \(\lambda \gt 780nm\)

可见光随波长不同而引起人眼不同的颜色感觉。

单色光 具有单一波长的光，理想光源，现实中不存在。 激光单色性良好，可近似看作单色光。
复色光 由不同单色光混合而成的光。 太阳光由无限多种单色光组成，可见光范围内可分解为红、橙、黄、绿、青、蓝、紫七种颜色的光。
