典型光学系统

人眼

分类
正常眼
反常眼

近视眼：远点位于有限距离处，在视网膜前形成焦点

度量：100度 = 1屈光度

校正：在近视眼前放一\(f' = lr\)的负透镜（光焦度为\(-\frac{度数}{100}D\)）

远视眼：远点位于眼后有限距离

校正：在远视眼前放一\(f' = lr\)的正透镜
老花眼
散光眼
弱视
斜视
眼睛的调节

眼睛对任意距离物体自动调焦的过程。

眼睛能清晰调焦的距离
  • 远点距离（\(lr\)）
  • 近点距离（\(lp\)）
发散度（会聚度）：单位为屈光度\(D\)，\(1D = 1m^{-1}\)
  • 远点发散度（视度）：\(R = 1/lr\)
  • 近点发散度：\(P = 1/lp\)
眼睛的调节范围（\(\bar A\)）: $$\bar A = R - P$$ 眼睛的明视距离：
    正常照明条件下（光照度为\(50lx\)），正常人眼最习惯的观察距离为\(250mm\)

照相系统

组成
  • 照相镜头：对物体成像
  • 可变光阑（光圈）：改变成像光束的宽度，调节光能量以适应不同的照明条件
  • 接收器件：记录物体的像
光阑设置
  • 孔径光阑：专设可变光阑，通常设置在镜头中间，可以认为与镜头重合
  • 视场光阑：与底片重合，感光底片边框为视场光阑

显微系统

分类
  • 低倍显微镜
  • 高倍显微镜
  • 测量显微镜
组成
  • 物镜：一次实放大
  • 目镜：二次虚放大
目镜放大倍率 $$\Gamma = 250/f_e'$$ 出瞳直径 $$D' = \frac{500NA}{\Gamma}$$ 数值孔径 $$NA = n\sin u$$ 参数：
  • 视觉放大倍率：
        人眼借助光学仪器观察物体时视网膜上的像高与用人眼直接观察物体时视网膜上的像高之比。 $$\Gamma = \frac{y_i'}{y_e'}$$
光阑设置：
  • 孔径光阑：
    • 低倍率显微镜：物镜组简单，孔径光阑为物镜框
    • 高倍率显微镜：物镜组复杂，孔径光阑为最后一组的镜框，或特设一个光阑
    • 测量显微镜：孔径光阑在物镜像方焦平面上
  • 视场光阑：分划板框
参数：
  • 共轭距

    显微物镜从物平面到像平面之间的距离。

  • 机械筒长

    把显微镜物镜和目镜取下后所剩的镜筒长度。

  • 光学筒长

    物镜像方焦点到目镜像方焦点间的距离。

显微镜物镜
  • 消色差物镜
  • 复消色差物镜
  • 平像场物镜
  • 反射式物镜
  • 折反射物镜

显微镜照明方式

  • 透射光亮视场照明
    • 临界照明
    • 柯勒照明
  • 反射光亮视场照明
  • 透射光暗视场照片
  • 反射光暗视场照明
临界照明

将光源的像成在物平面上，故光源表面亮度的不均匀性会影响显微镜的观察效果。

望远系统

分类：

  • 开普勒望远系统
  • 伽利略望远系统
开普勒望远系统：
特点：
  • 物镜与目镜均为正透镜光组
  • 成倒立虚像
  • 有一个中间实像面，可以设置分划板
光阑设置：
  • 孔径光阑
  • 视场光阑：分划板框
伽利略望远系统：
特点：
  • 物镜为正透镜光组，目镜为负透镜光组
  • 成正立虚像
  • 系统无中间实像面，不能设置分化板测量
光阑设置：
  • 孔径光阑：人眼瞳孔，同时也是出瞳
  • 视场光阑：物镜框，同时也是入窗
参数：
1.分辨率：
    望远系统的分辨率用极限分辨角\(\phi\)表示，对于\(Lm\)远处相距\(lm\)的两点： $$\phi = \frac lL$$ 2.视觉放大倍率： $$\Gamma = \frac{\tan \omega'}{\tan \omega} = \gamma = -f_o'/f_e' = -D/D' = 1/\beta$$     满足分辨要求的最小视觉放大率为有效放大率（正常放大率） $$\Gamma = 1'/\phi = D/2.3$$     眼睛处于分辨极限条件下观察物像时会使眼睛感到疲劳，故在设计望远镜时，工作放大率一般取视觉放大率的2.3倍 $$\Gamma = D$$ 3.机械筒长： $$L = f_o' + f_e'$$
