几何光学的基本定律

光的直线传播定律

各向同性均匀介质中，光沿直线传播.

光的独立传播定律

不同光源发出的光在空间某点相遇时，彼此互不影响，各光束独立传播.

折射定律（斯涅尔定律） $$n_1\sin\theta_1 = n_2\sin\theta_2$$ 反射定律 $$\theta' = -\theta$$ 费马原理
马吕斯定律

光在各向同性均匀介质中传播时，始终保持着与波面的正交性，且入射波面与出射波面对应点之间的光程均为定值.

折反射定律、费马原理、马吕斯定律中的任意一个，均可视作几何光学的一个基本定律，而另外两个视为该定律的推论。

一、光波与光线

在各向同性均匀介质中，光线为光波的法线，光束对应波面的法线束

波阵面（波面）：发光点发出的光波向四周传播时，某一时刻振动位相相同的点构成的等相面。

光束：与波面对应的所有光束的集合。

波面分类：

  • 平面光束：与平面波对应的相互平行的光束。
  • 同心光束：与球面波对应的相交于球面波球心的光束
    • 会聚光束：会聚于同一点的光束。
    • 发散光束：由同一点发出的光束。
  • 像散光束：由像差引起的与非球面波（任意曲面波）对应的光束。

第二节 成像的基本概念与完善成像条件

一、光学系统与成像概念

光学系统

由若干（球面、平面或非球面）光学元件（透镜、棱镜、反射镜和分划板等）组成的系统.

现代光学元件还有光栅、光纤、干涉/衍射元件、二元光学元件等。
光学系统 共轴光学系统 组成光学系统的各个光学元件表面曲率中心都在同一个直线上.
非共轴光学系统
球面光学系统 光学元件全为球面的光学系统.
非球面光学系统 含有非球面元件的光学系统.

光学系统的主要用途之一就是对物体成像.

物

发出入射光的物质.

可以是光源、被光源照明的物体或其他光学系统的像.
像 由出射光形成的物质.

[注]物像的概念不是绝对的.

物空间 物所在的空间，即光学系统成像前的整个空间.
像空间 像所在的空间，即光学系统成像后的整个空间.

[注]物空间和像空间没有严格的位置界限，其范围均为\((-\infty, +\infty)\)

实物（像）点 由实际光线相交所形成的点。既能被人眼观察，又能用屏幕、胶片或光电成像器件（数码相机中的CCD、CMOS等）记录.
虚物（像）点 由光线延长线相交所形成的点。只能被人眼观察，不能被记录.

完善像点：如果一球面波经过一光学系统以后仍为一球面波，那么对应的光束仍为同心光束，则称该同心光束的中心为物点经过光学系统所成的完善像点。

完善像：物体上每个点经过光学系统后所成完善像的集合。

二、完善成像条件

完善成像的条件：入射波面为球面波时，出射球面也为球面波。

实际光线光路计算

光线经单个折射球面的光路计算，是指在给定单个折射球面的结构参量\(n,n',r\)时，由已知入射光线参数\(L\)与\(U\)计算其他参数的过程.

入射角\(I\)
$$\sin I = \frac{L-r}{r}\sin U$$

在\(\triangle AEC\)中，由正弦定理：

$$\frac{\sin(-U)}{r} = \frac{\sin(\pi - I)}{r - L}$$ $$\frac{-\sin U}{r} = \frac{\sin I}{r-L}$$ $$\sin I = \frac{L-r}{r}\sin U$$
折射角\(I'\)
$$\sin I' = \frac{n}{n'}\sin I$$

由折射定律：

$$n\sin I = n'\sin I'$$ $$\sin I' = \frac{n}{n'}\sin I$$
像方孔径角\(U'\)
$$U' = U + I - I'$$

在\(\triangle AEC\)中：

$$I = -U + \phi$$

在\(\triangle A'EC\)中：

$$\phi = U' + I'$$ $$\therefore U' = \phi - I' = U + I - I'$$
像方截距\(L'\)
$$L' = r(1 + \frac{\sin I'}{\sin U'})$$

在\(\triangle A'EC\)中，由正弦定理：

$$\frac{\sin U'}{r} = \frac{\sin I'}{L' - r}$$ $$L' = r(1 + \frac{\sin I'}{\sin U'})$$

单个折射球面近轴区光路计算公式

$$\begin{cases} i = \frac{l-r}{r}u\\ i' = \frac{n}{n'}i\\ u' = u + i - i'\\ l' = r(1 + \frac{i'}{u'}) \end{cases}$$

近轴区：当U很小时，I、I'、U'都很小。光线在光轴附近很小的区域内，这个区域即为近轴区。近轴区内的光线即为近轴光线。
此时有：\(\sin\theta=\theta-\frac{\theta^3}{3!}+\frac{\theta^5}{5!}-\frac{\theta^7}{7!}+\dots\)，
故\(sin\theta\approx b\)

近轴区的划定：所谓近轴区并没有明确的界限，其受所允许的相对误差大小制约，误差不同近轴区的界定值也不同。 $$ \frac{\sin U-U}{\sin U} \leq \pm 0.001 \Rightarrow U\leq \pm 5^\circ $$ $$ \frac{\sin U-U}{\sin U} \leq \pm 0.0001 \Rightarrow U\leq \pm 1.5^\circ $$

单个折射球面近轴区成像

物像公式

$$\frac{n'}{l'} - \frac{n}{l} = \frac{n'-n}{r}$$
阿贝不变量\(Q\) $$Q = n'(\frac{1}{r}-\frac{1}{l'}) = n(\frac{1}{r}-\frac{1}{l})$$

第四节 球面光学成像系统

单个折射面成像：

垂轴放大率（横向放大率）\(\beta\)

像的大小与物的大小之比.

$$\beta = \frac{y'}{y} = \frac{nl'}{n'l}$$
$$\because\triangle ABC \sim \triangle A'B'C$$ $$\therefore\frac{-y'}{y} = \frac{l'-r}{r-l}$$ $$\because Q = n'(\frac{1}{r}-\frac{1}{l'}) = n(\frac{1}{r}-\frac{1}{l})$$ $$\frac{n'(l'-r)}{l'} = \frac{n(l-r)}{l}$$ $$\frac{l'-r}{l-r} = \frac{nl'}{n'l}$$ $$\beta = \frac{y'}{y} = \frac{l'-r}{l-r} = \frac{nl'}{n'l}$$
$$\beta\gt0$$ $$\beta\lt0$$
成像正倒 正像 倒像
成像虚实 物像虚实相反 物像虚实相同

$$\beta|\gt1$$ $$|\beta|=1$$ $$|\beta|\lt1$$
成像大小 成放大β倍的像 成等大的像 成缩小1/β倍的像
轴向放大率\(\alpha\)

物点沿光轴做微小移动\(\mathrm{d}l\)时，所引起的像点移动量\(\mathrm{d}l'\)与物点移动量\(\mathrm{d}l\)之比.

$$\alpha = \frac{\mathrm{d}l'}{\mathrm{d}l}$$

引入阿贝不变量Q，有： $$ Q = n'(\frac{1}{r}-\frac{1}{l'}) = n(\frac{1}{r}-\frac{1}{l}) $$ 两边求微分，有： $$ \frac{n'}{l'^2}dl' = \frac{n}{l^2}dl $$ $$ \alpha = \frac{dl'}{dl} = \frac{nl'^2}{n'l^2} = \frac{n'}{n}\beta^2 $$

  1. \(\alpha\propto\beta^2\)，表明\(\alpha\)恒为正值，即物像移动方向相同
  2. 由于\(\alpha\)通常不等于\(\beta\)，故立体的物体经过系统成像后往往不能得到立体的像
  3. 此公式仅适用于\(dl\)很小的情况

角放大率\(\gamma\)：一对共轭光线的像方孔径角与物方孔径角的正切值之比

$$ \gamma = \frac{u'}{u} $$

近轴区内： $$\begin{cases} \tan u \approx u = \frac{h}{l}\\ \tan u' \approx u' = \frac{h}{l'} \end{cases}\Rightarrow ul = u'l' $$ $$ \gamma = \frac{u'}{u} = \frac{l}{l'} = \frac{n}{n'} \frac{1}{\beta} $$

  1. 表征了折射球面将光束变宽或变细的能力。
  2. 仅适用于物体尺寸或角度较小的情况；当物体较大时，角放大率将随物点偏离光轴的程度而产生一定变化。
  3. 仅与共轭点位置有关，而与光线孔径角无关。

垂轴放大率、轴向放大率与角放大率之间的关系：\(\alpha \gamma = \beta\)

拉赫不变量（拉格朗日-赫姆霍兹不变量）\(J\) $$J = nuy = n'u'y'$$ $$\because \begin{cases} \beta = \frac{y'}{y} = \frac{nl'}{n'l}\\ \gamma = \frac{u'}{u} = \frac{l}{l'} \end{cases}$$ $$\therefore \frac{y'}{y} = \frac{nl'}{n'l} = \frac{nu}{n'u'}$$ $$\therefore nuy = n'u'y'$$

上式称为拉格朗日-赫姆霍兹恒等式.


由$$ \beta = \frac{y'}{y} = \frac{nu}{nu'y'} $$ 得 拉赫不变量仅适用于光学系统在近轴区成像时
