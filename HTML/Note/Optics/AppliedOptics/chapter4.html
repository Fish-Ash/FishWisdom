第四章 光学系统中的光阑与光束限制

第一节 光阑

光学系统的成像要求：

  1. 对给定的物体在给定的共轭距离上按要求的倍率成像
  2. 具有给定的成像范围：决定于系统的视场
  3. 具有一定的光度水准（有一定亮度）
  4. 具有一定的分辨本领，反映物体的一定细节
要求解析：
  • 要求1决定了系统的轴向尺寸
  • 要求2决定于系统的视场
  • 要求3、4决定于成像光束的立体角或孔径角
  • 要求2、3、4说明光学系统应对成像范围以内的物点以要求孔径角的光束成像。

人眼的分辨率：
    人眼能分辨两个邻近点的能力。
人眼的极限分辨角（\(\epsilon\)）：
    恰能分开的两个点对人眼物方节点的张角，单位为\(rad\)
分辨率与极限分辨角成反比。

  • \(\eta'\)：视觉细胞的直径，\(\eta'\approx 0.006mm\)
  • \(x_j'\)：像方节点到视网膜间的距离，\(x_j' = f\)，当眼睛处于放松状态时\(f = -17.1mm\)
$$\eta' = x_j' \cdot \tan \epsilon \Rightarrow \epsilon \approx 70''$$

光学系统的光束限制：
    进入系统光线的多少取决于构成光学系统的各种元件通光口径的大小及各元件的相对位置。

光阑：

限制成像光束和成像范围的光学元件，不具备成像特性。

  • 孔径光阑（有效光阑）
  • 视场光阑：
  • 渐晕光阑
  • 消杂光光阑

一、孔径光阑

定义：
    光学系统中限制轴上物点成像光束大小的光阑。

孔径：
    光学系统中描述成像光束大小的参量，物体在有限远时用孔径角\(U\)表示，在无穷远时用孔径高度\(h\)表示。

    孔径光阑实际上限制的是成像光束立体角的大小，如果在子午面内进行分析，孔径光阑决定了轴上点发出的最大孔径角的大小，多为圆形。

特点：

  • 参与成像的光束的孔径角大小既与光阑的大小有关又与光阑的位置有关。
  • 通过适当选择光阑的位置，就能对轴外物点的成像光束进行选择，从而把偏离于理想成像的、质量较差的部分光拦截掉，最终改善成像质量。
  • 在保证成像质量的前提下，合理选取光阑的位置，能够减小系统的横向尺寸，使结构匀称。

入射光瞳（入瞳）：
    孔径光阑经过其前面的镜组在系统物空间所成的像。限制了轴上物点最大物方孔径角大小，且入瞳越大，进入系统参与成像的能量越多。

出射光瞳（出瞳）：
    孔径光阑经过其后面的镜组在系统像空间所成的像。限制了轴上像点最大像方孔径角大小。

    孔径光阑既与入瞳相共轭，又与出瞳相共轭，我们常将出瞳看作是入瞳经过整个系统所成的像。
    通过入瞳中心的光线必通过孔径光阑，也必通过出瞳，这条光线被称为主光线。
    入瞳、出瞳、孔径光阑对光束的限制作用是等价的。
    光学元件通常由许多光学元件构成，每个光学元件的外框都有可能起到限制光束的作用。 孔径光阑的判断方法：
  • 当物在有限远时：将光学系统中所有光学元件的通光口径分别通过其前面的镜组成像到整个系统的物空间中，根据各像的位置和大小求出对轴上物点的张角，对轴上物点张角最小的像为光学系统的入瞳。与入瞳相共轭的元件即为孔径光阑。
  • 当物在无限远时：将光学系统中所有光学元件的通光口径分别通过其前面的镜组成像到整个系统的物空间中，则直径最小的像就是系统的入瞳。与入瞳相共轭的元件即为孔径光阑

二、视场光阑

定义：
    安置在物平面或像平面用以限制成像范围的光阑。
    光学系统只能有一个视场光阑，视场光阑的位置与孔径将直接决定物面与像面上的成像范围。很多光学系统都是在最终的实像平面或中间过程的实像平面上专门设置视场光阑。

视场：
    描述成像光学系统物、像平面（或空间）上的成像范围大小。
度量视场的方法：

  • 线视场：系统对近距离物体成像时，用长度来度量视场
    • 物方（全）线视场：\(2y\)
    • 像方（全）线视场：\(2y'\)
  • 视场角：系统对远距离物体成像时，用角度来度量视场
    • 物方（全）视场角：\(2\omega\)
    • 像方（全）视场角：\(2\omega'\)
    半视场角\(\omega\)：由光轴转向光线的夹角。

入射窗：
    视场光阑经过前面的镜组在物空间所成的像。入射窗将限制物面的成像范围（像方视场的大小），入射窗与视场光阑相共轭。
出射窗：
    视场光阑经过后面的镜组在像空间所成的像。出射窗将限制像面的成像范围（像方视场的大小），出射窗与视场光阑相共轭。
可以把出射窗看作是入射窗经过整个系统所成的像。

判断视场光阑的方法：将光学系统中所有光学元件的通光口径分别通过其前面的镜组成像到整个系统的物空间中，根据各像的位置和大小求出对入瞳中心的张角，张角最小的像即为光学系统的入射窗。与入射窗相共轭的元件即为视场光阑。

三、渐晕光阑

定义：
    能产生渐晕现象的光阑

渐晕：
    轴外点发出的充满入瞳的光束受到透镜通光孔径的限制而部分被遮拦，随视场增大像逐渐变暗的现象。

四、消杂光光阑

定义：
    能够消除因系统反射或散射产生的杂光的光阑

第三节 望远系统中成像光束的选择

  • 望远系统的孔径光阑大致在物镜左右，具体位置可根据尽量减少光学零件的尺寸和体积的考虑去设定。

第四节 显微系统中的光束限制与分析

显微系统：

  • 孔径光阑：系统中成像光束的口径往往由物镜框限制。
  • 视场光阑：位于目镜物方焦平面上的圆孔光阑或分划板框限制了系统的成像范围。

远心光路

  • 物方远心光路：

    现象：
        物方主光线平行于系统光轴，主光线的会聚中心位于物方无限远处。

    解决方法：
        将孔径光阑放置在系统的像方焦平面处
  • 像方远心光路：

    应用：
        测距光学仪器

    解决方法：
        将孔径光阑放置在系统的像方焦平面处

    现象：
        像方主光线平行于系统光轴，主光线的会聚中心位于像方无限远处。

由于系统中存在景深，使得仪器在测量过程无法确保物体像平面与分划板的刻尺面完全相重合，即存在视差（像平面与刻尺面不重合的现象），由于视差的存在，将产生一定的测量误差。

测量精度在很大程度上取决于像平面与刻尺面的不重合度，不重合程度越大其测量精度越低。

场镜：

定义：
    位于像面或像面附近的透镜（通常为正透镜），以改变斜光束方向，但不影响成像性质。
作用：

  • 减少后续光学系统口径
  • 增大出瞳距

第五节 光学系统的景深

景深\(\Delta\)：
    能在景像平面上获得清晰像的物空间深度。
    产生原因：像面处接收器本身不完善。

景像平面共轭平面为对准平面

按照理想光学系统的共线成像理论，景像平面上的空间像，严格来讲除了对准平面上的物点能成清晰像外，其他位置的空间点在景像平面上只能成一个弥散斑。

当弥散斑对人眼的张角小于人眼的极限分辨角（约为1'）时，则人眼对图像将无不清晰的感觉，即在一定空间范围内的空间点在景像平面上可成清晰像。

弥散斑的大小与光学系统入瞳大小、空间点距对准平面的距离有关。

  • 远景平面：能成清晰像的最远物平面
  • 近景平面：能成清晰像的最近物平面

  • 远景深度\(\Delta_1\)：远景平面与对准平面间的距离
  • 近景深度\(\Delta_2\)：近景平面与对准平面间的距离
$$\Delta = \Delta_1 + \Delta_2$$

远景平面、对准平面、近景平面到入瞳的距离分别以\(p_1、p、p_2\)表示，以入瞳中心点P为坐标原点。像空间对应的共轭面到出射光瞳的距离分别以\(p_1'、p'、p_2'\)，以出射光瞳中心P'为坐标原点。

相对孔径：

入瞳直径与系统焦距之比，\(\frac{D_{入}}{f'}\)

光圈数（光圈系数、F数、\( F^{\#} \)）：

相对孔径的倒数，\(F^{\#} = \frac{f'}{D_{入}}\)。中国国家标准中要求用\(f/F^{\#}\)表示。

光圈数值越大，光圈孔径越小，镜头通光量越小。
