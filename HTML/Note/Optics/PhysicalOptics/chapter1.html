第一章 光的电磁理论基础

第一节 光的电磁波性质

电磁波

变化的电磁场。

电磁矢量以速度\(v\)按波动形式在介质中传播。

$$v = 1/\sqrt{\mu\epsilon}$$

光本质上是一种电磁波。

光速\(c\)

真空中电磁波的传播速度

$$c = 1/\sqrt{\mu_0\epsilon_0}\approx2.9979\times 10^8m/s$$ 式中，\(\mu_0\)为真空中的磁导率,\(\epsilon_0\)为真空中的介电常数

介质的折射率\(n\)

电磁波在真空中的速度\(c\)与在不导电的均匀介质中的速度\(v\)之比

$$n = \frac{c}{v}=\frac{\sqrt{\mu\epsilon}}{\sqrt{\mu_0\epsilon_0}} = \sqrt{\mu_r\epsilon_r}$$ 式中，相对磁导率\(\mu_r = \mu/\mu_0\)

物质方程

静止的各向同性介质中，有：

$$\mathbf{j} = \sigma\mathbf{E}$$ $$\mathbf{D} = \epsilon\mathbf{E}$$ $$\mathbf{B} = \mu\mathbf{H}$$
  • \(\sigma\)：电导率，各向同性均匀介质中\(\sigma = 0\)
  • \(\mu\)：磁导率，各项同性均匀介质中为常数。\(\mu_0 = 4\pi\times 10^{-7}N\cdot s^2/C^2\)。非磁性物质\(\mu = \mu_0\)


等相面（波面）

某一时刻相位相同的点的空间位置

波矢量\(\mathbf{k}\) $$\mathbf{k} = k\mathbf{k_0}$$

\(\mathbf{k}_0\)为单位矢量，沿波的传播方向，即等相面的法线方向。

\(k = 2\pi/\lambda = \omega/\nu\)为空间角频率或波数。

平面波的表达式

$$\mathbf{E} = \mathbf{E_0}\exp[i\omega(t - \frac{r}{v})] = \mathbf{E_0}\exp[i(\omega t + \alpha)]$$ $$\mathbf{H} = \mathbf{H_0}\exp[i\omega(t - \frac{r}{v})] = \mathbf{H_0}\exp[i(\omega t + \beta)]$$ 其中，\(\alpha\)、\(\beta\)可以分别看作是\(\mathbf{E}\)与\(\mathbf{H}\)的初相。

第二节 光在电介质面上的反射与折射

电磁场的连续条件

在没有传导电流和自由电荷的介质中，磁感强度\(\mathbf{B}\)与电感强度\(\mathbf{D}\)的法向分量在界面上连续；电场强度\(\mathbf{E}\)和磁场强度\(\mathbf{H}\)的切向分量在界面上连续。

$$ \begin{align} \begin{cases} B_{1n} = B_{2n}\\ D_{1n} = D_{2n}\\ H_{1t} = H_{2t}\\ E_{1t} = E_{2t} \end{cases} \end{align} $$ 任一方位振动的光矢量\(\mathbf{E}\)，都可分解为互相垂直的两个分量
  • \(p\)偏振波（\(E_p\)）：\(\mathbf{E}\)平行于入射面振动的分量，又称TM波（横磁波\(\mathbf{H}\)沿界面切线方向）
  • \(s\)偏振波（\(E_s\)）：\(\mathbf{E}\)垂直于入射面振动的分量，又称TE波（横电波\(\mathbf{E}\)沿界面切线方向）

菲涅尔公式

全偏振现象与Brewster角

光从光疏介质入射到光密介质时，由菲涅尔公式：

  • \(\theta_1 = 0°\)即垂直入射时，\(r_p \gt 0 \gt r_s\)、\(t_s\gt 0\)、\(t_p\gt 0\)，此时存在反射波与折射波。
  • \(\theta_1 = 90°\)即掠入射时，\(r_s = r_p = -1\)，\(t_s = t_p = 0\)，此时仅有反射光。
全偏振现象

反射光的振动方向只沿一个平面内的现象。

由于\(r_p\gt 0\)的范围及其单调特性，可知当光从光疏介质进入到光密介质时，\(r_p\)可能会等于零。

\(r_p=0\)意味着此时反射光中只有\(s\)方向偏振的电场，即反射光为全偏振光。

\(r_p = \frac{\tan(\theta_1 - \theta_2)}{\tan(\theta_1 + \theta_2)} = 0\)，由于\(\theta_1 \neq \theta_2\)，故\(\tan(\theta_1 - \theta_2)\neq 0\)，只能是\(\tan(\theta_1 + \theta_2) = 0\)即\(\theta_1 + \theta_2 = \pi/2\)。称此时的入射角为起偏振角或Brewster角（\(\theta_B\)）。

坡印廷矢量\(\mathbf{S}\)（辐射强度矢量、能流密度）

用于描述电磁能量的传播。

其方向表示能量流动的方向。各向同性介质中，亦为波的传播方向。

其大小等于单位时间内垂直通过单位面积的能量。

$$\mathbf{S} = \mathbf{E} \times \mathbf{H} = \frac{1}{\mu}\mathbf{E}\times\mathbf{B}$$ 光强度\(I\)

介质中某点坡印廷矢量的瞬时值是忽大忽小的，但在一个周期内其平均值是一个定值，将其定义为光强度。

$$I = \frac{1}{T}\int_0^TS\mathrm{d}t = \frac{1}{2}\sqrt{\frac{\epsilon}{\mu}}A^2$$

取\(\mathbf{E}\)与\(\mathbf{H}\)的实数部分，有：

$$\mathbf{E} = \mathbf{E_0}\cos(\omega t + \alpha)$$ $$\mathbf{H} = \mathbf{H_0}\cos(\omega t + \beta)$$ $$\begin{align} I &= \frac{1}{T}\int_0^{T}S\mathrm{d}t\\ &= \frac{1}{T}\int_0^{T}|\mathbf{E}||\mathbf{H}|\sin\frac{\pi}{2}\mathrm{d}t\\ &= \frac{1}{T}\int_{0}^TE_0H_0\cos(\omega t + \alpha)\cos(\omega t + \beta)\mathrm{d}t\\ &= \frac{1}{2}E_0H_0\cos(\alpha - \beta) \end{align}$$ $$\begin{align} \because Re(EH^*) &= Re\{E_0\exp[{i(\omega t + \alpha)}]\cdot H_0\exp[{-i(\omega t + \beta)}]\}\\ &= E_0H_0\cos(\alpha - \beta) \end{align}$$ $$\therefore I = \frac{1}{2} Re(EH^*)$$

入射波、反射波与透射波的能量关系


通过单位面积的光能为：

入射波：\(W_1 = I_1\cos\theta_1 = \frac{1}{2}\sqrt{\frac{\epsilon_1}{\mu_1}}A_1^2\cos\theta_1\)

反射波：\(W'_1 = I'_1\cos\theta_1 = \frac{1}{2}\sqrt{\frac{\epsilon_1}{\mu_1}}A_1'^2\cos\theta_1\)

透射波：\(W_2 = I_2\cos\theta_2 = \frac{1}{2}\sqrt{\frac{\epsilon_2}{\mu_2}}A_2^2\cos\theta_2\)
定义 公式
反射比\(\rho\) 反射波与入射波能量之比 $$\rho = \frac{W_1'}{W_1} = \frac{A_1^2}{A_1'^2}$$
透射比\(\tau\) 透射波与入射波能量之比 $$\tau = \frac{W_2}{W_1} = \frac{\sqrt{\epsilon_2\mu_1}\cos\theta_2}{\sqrt{\epsilon_1\mu_2}\cos\theta_1}\frac{A_2^2}{A_1^2} = \frac{n_2\cos\theta_2}{n_1\cos\theta_1}\frac{A_2^2}{A_1^2}$$（非磁性物质\(\mu_1 = \mu_2\)）
