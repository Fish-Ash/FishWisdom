光的电磁理论基础

物质方程

静止的各向同性（物质每一点的物理性质不随方向而改变）介质中，有：

$$\mathbf{j} = \sigma\mathbf{E}$$ $$\mathbf{D} = \epsilon\mathbf{E}$$ $$\mathbf{B} = \mu\mathbf{H}$$
  • \(\sigma\)：电导率，各向同性均匀介质中\(\sigma = 0\)
  • \(\mu\)：磁导率，各项同性均匀介质中为常数。\(\mu_0 = 4\pi\times 10^{-7}N\cdot s^2/C^2\)。非磁性物质\(\mu = \mu_0\)


等相面（等相位面，波面）

某一时刻相位相同的点的空间位置.

相邻等相面间的相位差为\(2\pi\).

波矢量\(\mathbf{k}\) $$\mathbf{k} = k\mathbf{k_0}$$

\(\mathbf{k}_0\)为单位矢量，沿波的传播方向，即等相面的法线方向。

\(k = \sqrt{k_x^2 + k_y^2 + k_z^2} = 2\pi/\lambda = \omega/\nu\)为空间角频率或波数。

\(k = 0 \Leftrightarrow\)沿\(\mathbf{k}\)方向相位不发生改变.

基本公式 $$c = 1/\sqrt{\mu_0\epsilon_0}\approx2.9979\times 10^8m/s$$ $$n = c/v$$ $$\mu_r = \mu/\mu_0$$ $$\epsilon_r = \epsilon/\epsilon_0$$ $$\omega = 2\pi\nu = 2\pi/T$$ $$\lambda = vT$$ $$k = 2\pi/\lambda$$ $$f = \frac{1}{\lambda}$$ 衍生公式 $$\nu = 1/T$$
$$n = \frac{\sqrt{\mu\epsilon}}{\sqrt{\mu_0\epsilon_0}} = \sqrt{\mu_r\epsilon_r}$$
$$n = \frac{c}{v} = \frac{\sqrt{\mu\epsilon}}{\sqrt{\mu_0\epsilon_0}} = \sqrt{\mu_r\epsilon_r}$$
$$k = \frac{2\pi}{\lambda} = \frac{2\pi}{vT} = \frac{2\pi\nu}{v} = \frac{\omega}{v}$$
$$\lambda = \lambda_0/n$$
$$\lambda = vT = cT/n = \lambda_0/n$$
$$f = \frac{k}{2\pi}$$
$$f = \frac{1}{\lambda}$$ $$k = \frac{2\pi}{\lambda} \Rightarrow \lambda = \frac{2\pi}{k}$$ $$f = \frac{k}{2\pi}$$
参数 符号
光在真空中的传播速度 $$c$$
光在介质中的传播速度 $$v$$
介质的折射率 $$n$$
磁导率 $$\mu$$
真空中的磁导率 $$\mu_0$$
振动频率 $$\nu$$
振动周期 $$T$$
介质中的光波长 $$\lambda$$
真空中的光波长 $$\lambda_0$$

单色平面电磁波

表达式

$$\mathbf{E} = \mathbf{E_0}\exp[i\omega(t - \frac{r}{v})] = \mathbf{E_0}\exp[i(\omega t + \alpha)]$$ 其中，\(\alpha\)可以看作是\(\mathbf{E}\)的初相。

[注]光波中包含电场矢量\(\mathbf{E}\)与磁场矢量\(\mathbf{H}\)，从波的传播特性来看，其具有相同的地位；但从光与介质的相互作用来看，磁场的作用远比电场弱，甚至不起作用.因此，在讨论光的波动特性时，常常只考虑电场矢量\(\mathbf{E}\).通常称\(\mathbf{E}\)为光矢量，\(\mathbf{E}\)的振动称为光振动.

三角函数表示

设平面波沿直角坐标系\(xyz\)的\(z\)方向传播.

$$\mathbf{E} = \mathbf{E}_0\cos(\omega t - kz) = \mathbf{E}_0\cos[\omega(t - \frac{z}{v})] = \mathbf{E}_0\cos[2\pi(\frac{t}{T} - \frac{z}{\lambda})]$$
$$ \begin{align} \mathbf{E} &= \mathbf{E}_0\cos(\omega t - kz)\\ &= \mathbf{E}_0\cos[\omega(t - \frac{k}{w}z)]\\\\ &\because k = \frac{2\pi}{\lambda}, \omega = \frac{2\pi}{T}\\\\ \mathbf{E} &= \mathbf{E}_0\cos[\omega(t - \frac{T}{\lambda}z)]\\ &= \mathbf{E}_0\cos[\omega(t - \frac{z}{v})]\\ &= \mathbf{E}_0\cos[2\pi(\frac{t}{T} - \frac{z}{\lambda})] \end{align} $$
复数形式

复数的虚部无物理意义，为了便于计算，引入单色平面波的复数形式.

$$\mathbf{E} = \mathbf{E}_0 e^{-i(\omega t - kz)}$$

由欧拉方程

$$\begin{align} \mathbf{E} &= \mathbf{E}_0 e^{-i(\omega t - kz)}\\ &= \mathbf{E}_0 [\cos(\omega t - kz) - i\sin(\omega t - kz)] \end{align}$$

由于虚部无物理意义，故

$$\mathbf{E} = \mathbf{E}_0 \cos(\omega t - kz)$$
复振幅

复振幅反映了光场振动的振幅和相位随空间的变化.

$$\mathbf{E} = \mathbf{E}_0 e^{-i(\omega t - kz)} = \mathbf{E}_0 e^{ikz}e^{-i\omega t}$$

\(\tilde{E} = \mathbf{E}_0 e^{ikz}\)称为复振幅.

沿任意波矢量\(\mathbf{k}\)方向传播的平面简谐波 $$\mathbf{E} = \mathbf{E}_0\cos(\omega t - \mathbf{k}\cdot \mathbf{r} + \phi_0) = \mathbf{E}_0 e^{-i(\omega t - \mathbf{k}\cdot \mathbf{r} + \phi_0)}$$

复振幅为

$$\tilde{\mathbf{E}} =\mathbf{E}_0 e^{i(\mathbf{k}\cdot \mathbf{r} - \phi_0)}$$

性质

平面电磁波是横波 $$\mathbf{k}\mathbf{E} = 0$$ $$\mathbf{k}\mathbf{B} = 0$$

电矢量与磁矢量方向均垂直于波传播方向，平面电磁波为横波

\(\mathbf{E}, \mathbf{B}, \mathbf{k}_0\)互成右手螺旋系

\(\mathbf{E}, \mathbf{B}\)同相位
单色平面电磁波的一般方程 $$E = E_0\cos(\omega t - kr + \phi_0) = E_0e^{-i(\omega t - kr+ + \phi_0)}$$
  • 振动方向：\(\mathbf{E}_0\)的方向
  • 传播方向：\(\mathbf{k}_0\)的方向
  • 相位速度：\(v = \frac{\omega}{k}\)，单位为\(m/s\)，方向沿\(\mathbf{k}_0\)
  • 振幅：\(E_0\)，单位为\(V/m\)
  • 频率：\(\nu = 2\pi/\omega\)，单位为\(Hz\)
  • 波长：\(\lambda = 2\pi/k\)，单位为\(m\)

球面波

最简单的简谐球面光波——单色球面光波的波函数 $$E = \frac{A_1}{r}\cos(\omega t - kr)$$

复数形式

$$E = \frac{A_1}{r}e^{-i(\omega t - kr)}$$

复振幅

$$\tilde{E} = \frac{A_1}{r}e^{ikr}$$

\(A_1\)为离开点光源单位距离外的振幅值.

光波场的空间频率域表示

在空间域内，波数\(k\)可称为空间圆频率，波长\(\lambda\)可称为光波场的空间周期.

光波场在光波传播方向上的空间频率\(f_k\)

表示光波场沿波矢\(\mathbf{k}\)方向每增加单位长度，光波场增加的周期数.

$$f_k = \frac{1}{\lambda}$$

[注]光波场的空间周期与空间频率随观察方向不同而不同.

三维空间表示式 $$E = E_0 e^{i(k_xx + k_yy + k_zz + \phi_0)}$$

[注]三维空间表示式即为省略时间相位因子的表示式.

光波的速度

相速度

等相位面的传播速度.

群速度（包络速度）

等振幅面的传播速度.

单色光波的速度

单色光波的能量传播速度（能流速度） $$v = \omega/k = c/n$$ 单色光波的相速度

单色光波的相速度等于单色光波的能量传播速度.

复色光波的速度

简单起见，以二色光波为例：

$$E = E_{01}\cos(\omega_1t - k_1z) + E_{02}\cos(\omega_2t - k_2z)$$

设\(E_{0} = E_{01} = E_{02}\)，由和差化积公式

$$\begin{align} E &= E_0[\cos(\omega_1t - k_1z) + \cos(\omega_2t - k_2z)]\\ &= 2E_0\cos\frac{\omega_1t-k_1z + \omega_2t - k_2z}{2}\cos\frac{\omega_1t-k_1z - \omega_2t + k_2z}{2}\\ &= 2E_0\cos(\frac{\omega_1 + \omega_2}{2}t - \frac{k_1+k_2}{2}z)\cos(\frac{\omega_1 - \omega_2}{2}t - \frac{k_1 - k_2}{2}z)\\ &= 2E_0\cos(\omega_mt - k_mz)\cos(\bar{\omega}t - \bar{k}z) \end{align}$$ $$\begin{align} &\omega_m = \frac{1}{2}(\omega_1 - \omega_2) = \frac{1}{2}\Delta\omega\\\\ &k_m = \frac{1}{2}(k_1 - k_2) = \frac{1}{2}\Delta k\\\\ &\bar{\omega} = \frac{1}{2}(\omega_1 + \omega_2)\\\\ &\bar{k} = \frac{1}{2}(k_1 + k_2) \end{align}$$ 复色光波的相速度 $$v = \frac{\bar{\omega}}{\bar{k}}$$ 复色光波的群速度 $$v_g = \frac{\omega_m}{k_m} = \frac{\Delta \omega}{\Delta k}$$

当\(\Delta \omega\)很小时

$$v_g = \frac{\mathrm{d}\omega}{\mathrm{d}k} = = v(1 + \frac{\lambda}{n}\frac{\mathrm{d}n}{\mathrm{d}\lambda})$$
$$\because \omega = kv$$ $$v_g = \frac{\mathrm{d}\omega}{\mathrm{d}k} = \frac{\mathrm{d}(kv)}{\mathrm{d}k} = \frac{k\mathrm{d}v + v\mathrm{d}k}{\mathrm{d}k} = v + k\frac{\mathrm{d}v}{\mathrm{d}k}$$ $$\because v = c/n, k = 2\pi/\lambda$$ $$\therefore \mathrm{d}v = -\frac{c}{n^2}\mathrm{d}n, \mathrm{d}k = -\frac{2\pi}{\lambda^2}\mathrm{d}\lambda$$ $$v_g = v + \frac{2\pi}{\lambda}\frac{-\frac{c}{n^2}\mathrm{d}n}{-\frac{2\pi}{\lambda^2}\mathrm{d}\lambda} = v + v\frac{\lambda}{n}\frac{\mathrm{d}n}{\mathrm{d}\lambda} = v(1 + \frac{\lambda}{n}\frac{\mathrm{d}n}{\mathrm{d}\lambda})$$

在折射率\(n\)随波长\(\lambda\)变化的色散介质中，复色光波的相速度不等于群速度：

  • 正常色散介质（\(\mathrm{d}n/\mathrm{d}\lambda \lt 0\)）：\(v \gt v_g\)
  • 反常色散介质（\(\mathrm{d}n/\mathrm{d}\lambda \gt 0\)）：\(v \lt v_g\)

无色散介质（\(\mathrm{d}n/\mathrm{d}\lambda = 0\)）中，复色光波的群速度等于相速度，实际上只有真空才属于这种情况.

平面光波的偏振特性

光波的偏振特性是横波区别于纵波的最明显标志.

设光波沿\(z\)方向传播，电场矢量为

$$\mathbf{E} = \mathbf{E}_0 \cos(\omega t - kz + \phi_0)$$

为表征光波的偏振特性，将其表示为沿\(x, y\)方向上振动的两个独立分量的线性组合

$$\mathbf{E} = \mathbf{i}\mathbf{E}_x + \mathbf{j}\mathbf{E}_y$$ $$E_x = E_{0x}\cos(\omega t - kz + \phi_x) = E_{0x}e^{-i(\omega t - kz + \phi_x)}$$ $$E_y = E_{0y}\cos(\omega t - kz + \phi_y) = E_{0y}e^{-i(\omega t - kz + \phi_y)}$$
$$(\frac{E_x}{E_{0x}})^2 + (\frac{E_y}{E_{0y}})^2 - 2(\frac{E_x}{E_{0x}})(\frac{E_y}{E_{0y}})\cos \phi = \sin^2 \phi(\phi = \phi_y - \phi_x)$$

令\(\phi = \phi_y - \phi_x, \omega t - kz + \phi_x = \theta\)

$$E_x = E_{0x}\cos\theta \Rightarrow \cos\theta = \frac{E_x}{E_{0x}}$$ $$\sin\theta = \sqrt{1 - (\frac{E_x}{E_{0x}})^2}$$ $$E_y = E_{0y}\cos(\theta + \phi) = E_{0y}(\cos\theta\cos\phi - \sin\theta\sin\phi)$$ $$(\frac{E_y}{E_{0y}}) = \frac{E_x}{E_{0x}}\cos\phi - \sqrt{1 - (\frac{E_x}{E_{0x}})^2}\sin \phi$$ $$[1 - (\frac{E_x}{E_{0x}})]\sin^2\phi = (\frac{E_x}{E_{0x}})^2\cos^2\phi - 2(\frac{E_x}{E_{0x}})(\frac{E_y}{E_{0y}})\cos\phi + (\frac{E_y}{E_{0y}})^2$$ $$\sin^2\phi = (\frac{E_x}{E_{0x}})^2 - 2(\frac{E_x}{E_{0x}})(\frac{E_y}{E_{0y}})\cos\phi + (\frac{E_y}{E_{0y}})^2$$
线偏振光

条件：\(\phi = m\pi(m = 0, \pm 1, \pm 2, \cdots)\)

$$\frac{1}{E_{0x}}E_x \pm \frac{1}{E_{0y}}E_y = 0$$
$$\phi = m\pi(m = 0, \pm 1, \pm 2, \cdots) \Rightarrow \begin{cases}\sin \phi = 0\\ \cos\phi = \pm 1\end{cases}$$ $$(\frac{E_x}{E_{0x}})^2 + (\frac{E_y}{E_{0y}})^2 - 2(\frac{E_x}{E_{0x}})(\frac{E_y}{E_{0y}})\cos \phi = \sin^2 \phi$$ $$(\frac{E_x}{E_{0x}})^2 \pm 2(\frac{E_x}{E_{0x}})(\frac{E_y}{E_{0y}}) + (\frac{E_y}{E_{0y}})^2 = 0$$ $$(\frac{E_x}{E_{0x}} \pm \frac{E_y}{E_{0y}})^2 = 0$$ $$\frac{1}{E_{0x}}E_x \pm \frac{1}{E_{0y}}E_y = 0$$
$$\frac{E_y}{E_x} = \frac{E_{0y}}{E_{0x}}e^{-i\phi} = \frac{E_{0y}}{E_{0x}}e^{-im\pi}$$

当\(m\)为偶数时，光场在\(Ⅰ,Ⅲ\)象限内振动.

当\(m\)为奇数时，光场在\(Ⅱ,Ⅳ\)象限内振动.

圆偏振光

条件：\(E_{0x} = E_{0y} = E'_0, \phi = m\pi/2(m = \pm 1, \pm 3, \pm 5\cdots)\)

$$E^2_x + E^2_y = E'^2_0$$
$$\phi = m\pi/2(m = \pm 1, \pm 3, \pm 5\cdots) Rightarrow \begin{cases}\sin \phi = \pm 1 \\ \cos\phi = 0\end{cases}$$ $$E_{0x} = E_{0y} = E'_0$$ $$(\frac{E_x}{E_{0x}})^2 + (\frac{E_y}{E_{0y}})^2 - 2(\frac{E_x}{E_{0x}})(\frac{E_y}{E_{0y}})\cos \phi = \sin^2 \phi$$ $$(\frac{E_x}{E'_0})^2 + (\frac{E_y}{E'_0})^2 = 1$$ $$E^2_x + E^2_y = E'^2_0$$
$$\frac{E_y}{E_x} = \frac{E_{0y}}{E_{0x}}e^{-i\phi} = e^{\mp i\frac{\pi}{2}} = \mp i$$

规定：逆着光传播方向看，\(\mathbf{E}\)顺时针旋转，称为右旋偏振光；\(\mathbf{E}\)逆时针旋转，称为左旋偏振光.

\(\phi = \frac{\pi}{2}\)时，为右旋偏振光；\(\phi=-\frac{\pi}{2}\)时，为左旋偏振光.

椭圆偏振光

\(2m\pi \lt \phi \lt (2m+1)\pi\)：右旋椭圆偏振光.

\((2m-1)\pi \lt \phi \lt 2m\pi\)：左旋椭圆偏振光.

琼斯矩阵法表示偏振态

琼斯矩阵 $$\left[\begin{matrix}E_x\\E_y\end{matrix}\right] = \left[\begin{matrix}E_{0x}e^{-i\phi_x}\\E_{0y}e^{-i\phi_y}\end{matrix}\right]$$ \(Ⅰ，Ⅲ\)象限内的线偏振光

此时\(\phi_x = \phi_y = \phi_0\)

$$\left[\begin{matrix}E_x\\E_y\end{matrix}\right] = \left[\begin{matrix}E_{0x}\\E_{0y}\end{matrix}\right]e^{-i\phi_0}$$ 右旋、左旋圆偏振光

此时\(\phi_y - \phi_x = \pm \frac{\pi}{2}, E_{0x} = E_{0y} = E_0\)

$$\left[\begin{matrix}E_x\\E_y\end{matrix}\right] = \left[\begin{matrix}1\\\mp i\end{matrix}\right]E_0e^{-i\phi_x}$$
