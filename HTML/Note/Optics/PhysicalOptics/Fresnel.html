菲涅尔公式

表示反射波、折射波与入射波的振幅和相位关系。

一般式 两种介质都是电介质时，考虑一般的非磁性物质，即\(\mu_1 = \mu_2\) 斜入射时，考虑折射定律\(n_1\sin\theta_1 = n_2\sin\theta_2\)
s波 振幅反射系数\(r_s\) $$r_s = \frac{E_{0rs}}{E_{0is}} = \frac{\frac{n_1}{\mu_1}\cos\theta_1 - \frac{n_2}{\mu_2}\cos\theta_2}{\frac{n_1}{\mu_1}\cos\theta_1 + \frac{n_2}{\mu_2}\cos\theta_2}$$ $$r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2}$$
$$r_s = \frac{\tan\theta_2 - \tan\theta_1}{\tan\theta_2 + \tan\theta_1} = \frac{\sin(\theta_2 - \theta_1)}{\sin(\theta_2 + \theta_1)}$$
$$n_1\sin\theta_1 = n_2\sin\theta_2\Rightarrow n = \frac{n_2}{n_1} = \frac{\sin\theta_1}{\sin\theta_2}$$ $$\begin{align} r_s &= \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2}\\ &= \frac{\cos\theta_1 - \frac{\sin\theta_1}{\sin\theta_2}\cos\theta_2}{\cos\theta_1 + \cos\theta_2}\\ &= \frac{\sin\theta_2\cos\theta_1 - \sin\theta_1\cos\theta_2}{\sin\theta_2\cos\theta_1 + \sin\theta_1\cos\theta_2}\\ &= \frac{\tan\theta_2 - \tan\theta_1}{\tan\theta_2 + \tan\theta_1}\\ &= \frac{\sin(\theta_2 - \theta_1)}{\sin(\theta_2 + \theta_1)} \end{align}$$
振幅透射系数\(t_s\) $$t_s = \frac{E_{0ts}}{E_{0is}} = \frac{2\frac{n_1}{\mu_1}\cos\theta_1}{\frac{n_1}{\mu_1}\cos\theta_1 + \frac{n_2}{\mu_2}\cos\theta_2}$$ $$t_s = \frac{2n_1\cos\theta_1}{n_1\cos\theta_1 + n_2\cos\theta_2}$$ $$t_s = \frac{2\cos\theta_1\sin\theta_2}{\sin(\theta_1 + \theta_2)}$$
p波 振幅反射系数\(r_p\) $$r_p = \frac{E_{0rp}}{E_{0ip}} = \frac{\frac{n_2}{\mu_2}\cos\theta_1 - \frac{n_1}{\mu_1}\cos\theta_2}{\frac{n_2}{\mu_2}\cos\theta_1 + \frac{n_1}{\mu_1}\cos\theta_2}$$ $$r_p = \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2}$$
$$r_p = \frac{\sin 2\theta_1 - \sin 2\theta_2}{\sin 2\theta_1 + \sin 2\theta_2} = \frac{\tan(\theta_1-\theta_2)}{\tan(\theta_1 + \theta_2)}$$
$$n_1\sin\theta_1 = n_2\sin\theta_2\Rightarrow n = \frac{n_2}{n_1} = \frac{\sin\theta_1}{\sin\theta_2}$$ $$\begin{align} r_p &= \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2}\\ &= \frac{\frac{\sin\theta_1}{\sin\theta_2}\cos\theta_1 - \cos\theta_2}{\frac{\sin\theta_1}{\sin\theta_2}\cos\theta_1 + \cos\theta_2}\\ &= \frac{\sin\theta_1 \cos\theta_1 - \sin\theta_2 \cos\theta_2}{\sin\theta_1\cos\theta_1 + \sin\theta_2\cos\theta_2}\\ &= \frac{\sin2\theta_1 - \sin2\theta_2}{\sin2\theta_1 + \sin2\theta_2} \end{align}$$

由和差化积公式

$$\begin{align} r_p &= \frac{\sin2\theta_1 - \sin2\theta_2}{\sin2\theta_1 + \sin2\theta_2}\\ &= \frac{2\cos\frac{2\theta_1 + 2\theta_2}{2}\sin\frac{2\theta_1 - 2\theta_2}{2}}{2\sin\frac{2\theta_1 + \theta_2}{2}\cos\frac{2\theta_1 - 2\theta_2}{2}}\\ &= \frac{\cos(\theta_1 + \theta_2)\sin(\theta_1 - \theta_2)}{\sin(\theta_1+\theta_2) \cos(\theta_1-\theta_2)}\\ &= \frac{\tan(\theta_1 - \theta_2)}{\tan(\theta_1 + \theta_2)} \end{align}$$
振幅透射系数\(t_p\) $$t_p = \frac{E_{0tp}}{E_{0ip}} = \frac{2\frac{n_1}{\mu_1}\cos\theta_1}{\frac{n_2}{\mu_2}\cos\theta_2+\frac{n_1}{\mu_1}\cos\theta_2}$$ $$t_p = \frac{2n_1\cos\theta_1}{n_2\cos\theta_1 + n_1\cos\theta_2}$$ $$t_p = \frac{2\cos\theta_1\sin\theta_2}{\sin(\theta_1 + \theta_2)\cos(\theta_1 - \theta_2)}$$

由菲涅尔公式得出的结论

光从光疏介质入射到光密介质

$$ n_1 \lt n_2 \Rightarrow n = n_2/n_1 \gt 1 $$
垂直入射 略入射
$$r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2} = \frac{1 - n}{1 + n}$$ $$-1 \lt r_s \lt 0$$ $$r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2} = -\frac{n_2\cos\theta_2}{n_2\cos\theta_2} = -1$$
$$t_s = \frac{2n_1\cos\theta_1}{n_1\cos\theta_1 + n_2\cos\theta_2} = \frac{2}{1+n}$$ $$0 \lt t_s \lt 1$$ $$t_s = \frac{2n_1\cos\theta_1}{n_1\cos\theta_1 + n_2\cos\theta_2} = 0$$
$$r_p = \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2} = \frac{n-1}{n+1}$$ $$0 \lt r_p \lt 1$$ $$r_p = \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2} = -\frac{n_1\cos\theta_2}{n_1\cos\theta_2} = -1$$
$$t_p = \frac{2n_1\cos\theta_1}{n_2\cos\theta_1 + n_1\cos\theta_2} = \frac{2}{n+1}$$ $$0 \lt t_p \lt 1$$ $$t_p = \frac{2n_1\cos\theta_1}{n_2\cos\theta_1 + n_1\cos\theta_2} = 0$$

光从光密介质入射到光疏介质

$$ n_1 \gt n_2 \Rightarrow n = n_2/n_1 \lt 1 $$
垂直入射
$$r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2} = \frac{1 - n}{1 + n}$$ $$0 \lt r_s \lt 1$$
$$t_s = \frac{2n_1\cos\theta_1}{n_1\cos\theta_1 + n_2\cos\theta_2} = \frac{2}{1+n}$$ $$1 \lt t_s \lt 2$$
$$r_p = \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2} = \frac{n-1}{n+1}$$ $$-1 \lt r_p \lt 0$$
$$t_p = \frac{2n_1\cos\theta_1}{n_2\cos\theta_1 + n_1\cos\theta_2} = \frac{2}{n+1}$$ $$1 \lt t_p \lt 2$$

关于菲涅尔公式单调性的讨论

\(n \gt 1\)：\(r_s, r_p, t_s, t_p\)单调递减.

\(n \lt 1\)：\(r_s, r_p, t_s, t_p\)单调递增.

由折射定律： $$ n_1\sin\theta_1 = n_2\sin\theta_2(0\leq \theta_1 \leq \frac{\pi}{2})\\ \sin\theta_2 = \frac1n\sin\theta_1\\ \cos\theta_2 = \sqrt{1 - \frac{1}{n^2}\sin^2\theta_1} $$
对于\(r_s\)： $$ \begin{align} r_s &= \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2}\\ &= 1 - \frac{2n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2}\\ &= 1 - \frac{2}{\frac{1}{n}\frac{\cos\theta_1}{\cos\theta_2} + 1} \end{align} $$ $$ \begin{align} \frac{\cos\theta_1}{\cos\theta_2} &= \sqrt{\frac{\cos^2\theta_1}{1 - \frac{1}{n^2}\sin^2\theta_1}}\\ &= \sqrt{\frac{n^2\cos^2\theta_1}{(n^2 - 1) + \cos^2\theta_1}}\\ &= \sqrt{\frac{n^2}{(n^2-1)\frac{1}{\cos^2\theta_1}+1}} \end{align} $$
  • \(n\gt 1\)光疏入光密时：\(r_s\downarrow\)
  • \(n\lt 1\)光密入光疏时：\(r_s\uparrow\)
