第二章 激光的物理学基础

第一节 光波的模式和光子的量子状态

光子的性质

  • 光子的能量\(\epsilon\)与光波频率\(\nu\)对应 $$\epsilon = h\nu$$ 式中，普朗克常数\(h=6.626\times10^{-34}\)J·s

  • 光子具有运动质量\(m\) $$m = \frac{\epsilon}{c^2}$$ 光子的静止质量为0

  • 光子的动量\(\mathbf{P}\)与单色平面光波的波矢\(\mathbf{k}\)对应 $$\mathbf{P} = mc\mathbf{n_0} = \frac{h\nu}{c}\mathbf{n_0} = \frac{h}{2\pi}\times\frac{2\pi}{\lambda}\mathbf{n_0} = \hbar \mathbf{k}$$ 式中，约化普朗克常数\(\hbar = \frac{h}{2\pi}\)
    \(\mathbf{k} = \frac{2\pi}{\lambda}\mathbf{n_0}\)
    \(\mathbf{n_0}\)为光子运动方向（平面光波传播方向）上的单位矢量。

  • 光子具有两种可能的独立偏振状态，对应于光波场的两个独立偏振方向。
  • 光子具有自旋，且自旋量子数为整数，属于玻色子。
玻色子

自旋为整数的粒子

玻色子服从玻色-爱因斯坦统计规律

三、光子的相干性

Definition
光的相干性

不同空间点上、不同时刻的光波场某些特性的相关性

相干体积\(V_c\)

光波场都具有明显相干性的空间体积。

$$V_c = A_c L_c = A_c \tau_c c$$ 相干面积\(A_c\)

垂直于光传播方向的截面

相干长度\(L_c\)

沿光的传播方向

相干时间\(\tau_c\)

光沿传播方向通过相干长度\(L_c\)所需的时间。

Conclusion

相格空间体积以及一个光波模或光子态占有的空间体积都等于相干体积。

属于同一状态的光子或同一模式的光波是相干的，不同状态的光子或不同模式的光波是不相干的。

四、光子简并度

光子简并度\(\bar n\)
  • 处于同一相格中的光子数
  • 处于一个模式中的光子数
  • 处于相干体积内的光子数
  • 处于同一光子态的光子数

第二节 光的受激辐射理论

一、黑体辐射的普朗克公式

处于某一温度\(T\)下的物体能够发出和吸收电磁辐射，发出的辐射称为热辐射或温度辐射。

黑体（绝对黑体）

能够完全吸收任何波长电磁辐射的物质。

典型的理想黑体为带有小孔的空腔。

黑体辐射（平衡辐射）

当黑体处于某一温度的热平衡状态下时，其吸收的辐射能量应该等于其发出的辐射能量，即黑体与辐射场之间应该处于能量（热）平衡状态。这种平衡状态导致空腔内存在完全确定的辐射场，即黑体辐射。

黑体辐射是黑体温度\(T\)和辐射场频率\(\nu\)的函数，并用单色能量密度\(\rho_\nu\)表示。

单色能量密度\(\rho_\nu\)

单位体积内，频率处于\(\nu\)附近的单位频率间隔中的电磁辐射能量。

单位\(J\cdot m^{-3}\cdot s\)

玻尔兹曼统计分布

描述了在温度为\(T\)的热平衡状态下，粒子在不同能级上的分布情况

$$n_i = n\frac{e^{-E_i/k_bT}}{Z}$$ 符号解释
符号 解释
$$n_i$$ 某能级上的粒子数
$$n$$ 总粒子数
黑体辐射的普朗克公式 $$\rho_\nu = \frac{8\pi h\nu^3}{c^3}\frac{1}{e^{\frac{h\nu}{k_bT}}-1}$$

式中，玻尔兹曼常数\(k_b = 1.38062\times 10^{-23}\)J/K

能级

处于不同状态的原子具有分立的能量

基态

原子处于最低能级的状态

激发态

比基态能量高的状态


热平衡状态下，绝大多数原子都处于基态；处于基态的原子从外界吸收能量后，将跃迁至能量较高的激发态。

当原子被激发到高能级时，其在高能级上是不稳定的，将迅速跃迁至低能级；处于高能级上的原子，即使在没有任何外界作用的情况下，也可能从高能级跃迁至低能级。

原子从高能级跃迁至低能级时会释放能量，在没有外界作用的情况下原子从高能级向低能级跃迁的方式有自发辐射跃迁和无辐射跃迁。

无辐射跃迁

跃迁过程中，能量以热量的形式放出。

自发辐射跃迁

跃迁过程中，能量以光辐射的形式放出，辐射出的光子能量为\(h\nu_{21}\)，且满足玻尔条件。

$$h\nu_{21} = E_2 - E_1$$ 激发能级的平均寿命\(\tau_{21}\) $$\tau_{21} = \frac{1}{A_{21}}$$

受激吸收

当原子系统受到外来能量为\(h\nu_{21} = E_2 - E_1\)的光子作用时，处于低能级\(E_1\)上的原子由于吸收此光子而受到激发跃迁到高能级\(E_2\)上。

\(B_{12}\)只与原子性质有关

受激吸收跃迁的反过程，表现为处于高能级上的原子在频率为\(\nu\)的辐射场作用下跃迁至低能级并辐射出一个能量为\(h\nu_12\)的光子。

\(A_{21}\)、\(B_{21}\)、\(B_{12}\)的相互关系

在温度为\(T\)的热平衡态下，忽略无辐射跃迁，黑体吸收的辐射能量应等于发出的辐射能量，才能保持能量密度不变。

$$n_{2}A_{21} + n_2B_{21}\rho_\nu = n_1B_{12}\rho_\nu$$
