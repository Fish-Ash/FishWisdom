第二章 激光的物理学基础

第一节 光波的模式和光子的量子状态

一、光子的性质

光的量子学说（光子说）认为，光是一种以光速\(c\)运动的光子流.

光子的粒子属性：
光子的能量\(\epsilon\) $$\epsilon = h\nu$$

\(h=6.626\times10^{-34}J\cdot s\)为普朗克常数.

光子的运动质量\(m\) $$m = \frac{\epsilon}{c^2} = \frac{h\nu}{c^2}$$

光子的静止质量为\(0\).

光子的动量\(\mathbf{P}\)

光子运动方向即平面光波传播方向.

$$\mathbf{P} = mc\mathbf{k}_0 = \hbar \mathbf{k}$$
$$\mathbf{P} = mc\mathbf{k}_0$$ $$\because m = \frac{h\nu}{c^2}$$ $$\mathbf{P} = \frac{h\nu}{c}\mathbf{k}_0$$ $$\because \begin{cases} \nu = \frac{1}{T}\\ c = \frac{\lambda}{T} \end{cases}$$ $$\begin{align} \mathbf{P} &= \frac{h}{\lambda}\mathbf{k}_0\\ &= \frac{h}{2\pi}\frac{2\pi}{\lambda}\mathbf{k}_0\\ &= \hbar\mathbf{k} \end{align}$$

\(\hbar = \frac{h}{2\pi}\)称为约化普朗克常数.

\(\mathbf{k}\)为波矢量.

光子具有两种可能的独立偏振状态，对应于光波场的两个独立偏振方向。

光子属于玻色子，处于同一状态的光子数目是没有限制的。

粒子属性 表达公式 波动属性
能量 $$\epsilon = h \nu$$ 频率
动量 $$\mathbf{P} = \hbar \mathbf{k}$$ 波矢

量子电动力学从理论上将光的电磁（波动）理论和光子（微粒）理论在电磁场量子化描述的基础上统一起来，在理论上阐明了光的波粒二象性。

玻色子

自旋为整数的粒子

玻色子服从玻色-爱因斯坦统计规律

二、光波模式和光子状态

任意电磁波可看作是一系列单色平面电磁波（以\(\mathbf{k}\)波矢为标志）的线性叠加，或一系列电磁波的本征模式（或本征状态）的叠加。

每个本征模式所具有的能量是量子化的，是基元能量的整数倍\(h\nu_l\)；本征模式的动量是量子化的，是基元动量\(\hbar \mathbf{k}\)的整数倍。

具有基元能量和基元动量的物质单元称为属于第\(l\)个本征模式（或状态）的光子。具有相同能量和动量的光子彼此间不可区分，因而处于同一模式（或状态）。每个模式内的光子数目是没有限制的。

按量子电动力学概念，光波的模式与光子的状态是等价的.

四、光子简并度

光子简并度\(\bar n\)
  • 处于同一相格中的光子数
  • 处于一个模式中的光子数
  • 处于相干体积内的光子数
  • 处于同一光子态的光子数

第二节 光的受激辐射理论

一、黑体辐射的普朗克公式

处于某一温度\(T\)下的物体能够发出和吸收电磁辐射，发出的辐射称为热辐射或温度辐射。

黑体（绝对黑体）

能够完全吸收任何波长电磁辐射的物质。

典型的理想黑体为带有小孔的空腔。

黑体辐射（平衡辐射）

当黑体处于某一温度的热平衡状态下时，其吸收的辐射能量应该等于其发出的辐射能量，即黑体与辐射场之间应该处于能量（热）平衡状态。这种平衡状态导致空腔内存在完全确定的辐射场，即黑体辐射。

黑体辐射是黑体温度\(T\)和辐射场频率\(\nu\)的函数，并用单色能量密度\(\rho_\nu\)表示。

单色能量密度\(\rho_\nu\)

单位体积内，频率处于\(\nu\)附近的单位频率间隔中的电磁辐射能量。

单位\(J\cdot m^{-3}\cdot s\)

玻尔兹曼统计分布

描述了在温度为\(T\)的热平衡状态下，粒子在不同能级上的分布情况

$$n_i = n\frac{e^{-E_i/k_bT}}{Z}$$ 符号解释
符号 解释
$$n_i$$ 某能级上的粒子数
$$n$$ 总粒子数
黑体辐射的普朗克公式 $$\rho_\nu = \frac{8\pi h\nu^3}{c^3}\frac{1}{e^{\frac{h\nu}{k_bT}}-1}$$

式中，玻尔兹曼常数\(k_b = 1.38062\times 10^{-23}\)J/K

二、光（辐射）与物质（原子）的相互作用

光与物质的相互作用过程 跃迁概率 爱因斯坦系数 跃迁速率
自发辐射跃迁 $$A_{21} = (\frac{\mathrm{d}n_{21}}{\mathrm{d}t})_\mathrm{sp}\frac{1}{n_2}$$ $$A_{21}$$ $$n_2A_{21}$$
受激吸收跃迁 $$W_{12} = (\frac{\mathrm{d}n_{12}}{\mathrm{d}t}_\mathrm{st})\frac{1}{n_1}$$ $$B_{12} = \frac{W_{12}}{\rho_\nu}$$ $$n_1 W_{12} = n_1B_{12}\rho_\nu$$
受激辐射跃迁 $$W_{21} = (\frac{\mathrm{d}n_{21}}{\mathrm{d}t})_\mathrm{st}\frac{1}{n_2}$$ $$B_{21} = \frac{W_{21}}{\rho_\nu}$$ $$n_2 W_{21} = n_2 B_{21} \rho_\nu$$

这三种物理过程是同时存在的.

sp表示自发跃迁，st表示辐射跃迁.

  • \(n\)：能级上单位体积内的原子总数
  • \(\mathrm{d}n\)：\(\mathrm{d}t\)时间间隔内单位体积中发生跃迁的原子总数

自发辐射跃迁

自发跃迁

处于高能级\(E_2\)的一个原子自发向低能级\(E_1\)跃迁，并发射出一个能量为\(h\nu\)的光子的过程.

自发辐射

由原子自发跃迁发出的光波.

能级

处于不同状态的原子具有分立的能量

基态

原子处于最低能级的状态

激发态

比基态能量高的状态


热平衡状态下，绝大多数原子都处于基态；处于基态的原子从外界吸收能量后，将跃迁至能量较高的激发态。

当原子被激发到高能级时，其在高能级上是不稳定的，将迅速跃迁至低能级；处于高能级上的原子，即使在没有任何外界作用的情况下，也可能从高能级跃迁至低能级。

原子从高能级跃迁至低能级时会释放能量，在没有外界作用的情况下原子从高能级向低能级跃迁的方式有自发辐射跃迁和无辐射跃迁。

无辐射跃迁

跃迁过程中，能量以热量的形式放出。

自发辐射跃迁

跃迁过程中，能量以光辐射的形式放出，辐射出的光子能量为\(h\nu_{21}\)，且满足玻尔条件。

$$h\nu_{21} = E_2 - E_1$$ 激发能级的平均寿命\(\tau_{21}\) $$\tau_{21} = \frac{1}{A_{21}}$$

受激吸收

当原子系统受到外来能量为\(h\nu_{21} = E_2 - E_1\)的光子作用时，处于低能级\(E_1\)上的原子由于吸收此光子而受到激发跃迁到高能级\(E_2\)上。

\(B_{12}\)只与原子性质有关

受激辐射

受激辐射跃迁

受激吸收跃迁的反过程，表现为处于高能级上的原子在频率为\(\nu\)的辐射场作用下跃迁至低能级并辐射出一个能量为\(h\nu_12\)的光子。

受激辐射

受激辐射跃迁发出的光波。

\(A_{21}\)、\(B_{21}\)、\(B_{12}\)的相互关系

在温度为\(T\)的热平衡态下，忽略无辐射跃迁，黑体吸收的辐射能量应等于发出的辐射能量，才能保持能量密度不变。

$$n_{2}A_{21} + n_2B_{21}\rho_\nu = n_1B_{12}\rho_\nu$$

腔内黑体辐射场与物质原子相互作用的结果应维持黑体处于温度为\(T\)的热平衡状态，这种热平衡状态的标志是：

1、腔内存在由黑体的普朗克公式表示的热平衡黑体辐射.

2、腔内物质原子数按能级分布应服从热平衡状态下的波尔兹曼分布：

$$\frac{n_2}{n_1} = \frac{f_2}{f_1}e^{-\frac{E_2 - E_1}{k_bT}}$$

其中\(f_2\)与\(f_1\)分别为能级\(E_2\)和\(E_1\)的统计权重.
