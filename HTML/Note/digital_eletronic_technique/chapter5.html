第五章 锁存器和触发器

第一节 基本双稳态电路

双稳态电路（双稳态存储电路）：
    一旦进入0、1某种逻辑状态就能长期保持不变的单元电路。

  • 锁存器
  • 触发器

第二节 锁存器

锁存器
    对脉冲电平敏感的双稳态电路
SR锁存器：
    是一种具有最简单控制功能的双稳态电路

  • \(S=R=0\)：保持
  • \(S=0、R=1\)：置0
  • \(S=1、R=0\)：置1
  • \(S=R=1\)：处于既非0又非1的未定义状态
    为使锁存器始终工作于定义状态，输入信号应遵守\(SR=0\)的约束条件。

触发
    对时钟脉冲边沿敏感的状态更新
触发器
    具有触发工作特性的存储单元
    \(CP\)表示以上升沿触发的时钟信号
    \(\overline{CP}\)表示以下降沿触发的时钟信号

触发器 特性方程 功能
JK触发器 \(Q^{n+1} = J\overline{Q^n} + \overline KQ^n\) 计数（翻转）、保持、置0、置1
D触发器 \(Q^{n+1} = D\) 置0、置1
SR触发器 \(\begin{cases}Q^{n+1} = S+\overline RQ^n\\SR=0（约束条件）\end{cases}\) 保持、置0、置1
T触发器 \(Q^{n+1} = T\overline{Q^n} + \overline TQ^n\) 计数（翻转）、保持
T'触发器 \(Q^{n+1} = \overline{Q^n}\) 计数（翻转）

JK触发器

  • 当\(J=K=0\)时，JK触发器保持不变（\(Q^{n+1} = Q^n\)）
  • 当\(J=0,K=1\)时，JK触发器的下一状态将被置0（\(Q^{n+1} = 0\)）
  • 当\(J=1,K=0\)时，JK触发器的下一状态将被置1（\(Q^{n+1} = 1\)）
  • 当\(J=K=1\)时，JK触发器的下一状态将被翻转（\(Q^{n+1} = \bar{Q^n}\)）

D触发器

  • 构成JK触发器：\(D = J\overline{Q^n}+\overline{K}Q^n\)
  • 构成T触发器：\(D = T\oplus Q^n\)
  • 构成T'触发器：\(D = \overline{Q^n}\)
