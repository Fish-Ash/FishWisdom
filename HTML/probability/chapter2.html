第二节 离散型随机变量及其概率分布

一、离散型随机变量及其概率分布

离散型随机变量：可能的取值只有有限个或可列无限多个的随机变量\(X\)，

离散型随机变量\(X\)的概率分布（概率函数、分布律（列））：\(X\)取所有可能值的概率，即\(P\{X = x_k\} = p_k,~k =1,2,\cdots\)

二、常见的离散型随机变量的分布

  • 二项分布：n重伯努利试验中，事件A发生的概率为\(p\)，\(X\)表示n重伯努利试验中事件A发生的次数，则称\(X\)服从参数为\(n,p\)的二项分布，记作\(X\sim b(n,p)\) $$ P\{X = k\} = C_n^kp^k(1-p)^{n-k} $$
  • 几何分布：若事件A发生的概率为\(p\)，\(X\)表示事件A首次发生时已进行的次数，则称X服从参数为p的几何分布，记作\(X\sim G(p)\) $$ P\{X = k\} = p(1-p)^{k-1},~k=1,2,\cdots $$

第三节 随机变量的分布函数

一、随机变量的分布函数

随机变量\(X\)的分布函数：\(F(X) = P\{X \leq x\} \)

分布函数的性质：

  • 非负性：\(0\leq F(x)\leq 1,~x\in (-\infty, +\infty)\)
  • 单调不减性：当\(x_1 < x_2\)时，有\(F(x_1)\leq F(x_2)\)
  • 规范性：\(F(+\infty) = \mathop{lim}\limits_{x\rightarrow +\infty}F(x)=1\)，\(F(-\infty) = \mathop{lim}\limits_{x\rightarrow -\infty}F(x)=0\)

第四节 连续型随机变量及其概率密度

若对随机变量\(X\)的分布函数\(F(x)\)存在非负的可积函数\(f(x)\)，使得对于任意实数\(x\)有\(F(x) = \int^x_{-\infty}f(t)dt\)

常见的连续型随机变量的分布：

  • 均匀分布：若连续型随机变量\(X\)的概率密度为 $$f(x)= \begin{align} \begin{cases} \frac{1}{b-a},~&a\leq x\leq b\\ 0,~&others \end{cases} \end{align}$$ 称\(X\)在区间[a,b]上服从均匀分布，记作\(X\sim U[a,b]\)。其分布函数为 $$F(x)= \begin{align} \begin{cases} 0,~&x< a\\ \frac{x-a}{b-a},~&a\leq x\leq b\\ 1,~&x>b \end{cases} \end{align}$$
  • 指数分布：若连续型随机变量\(X\)的概率密度为 $$f(x)= \begin{align} \begin{cases} \lambda e^{-\lambda x},~&x>0\\ 0,~&x\leq 0 \end{cases} \end{align},~\lambda > 0 $$ 称\(X\)服从参数为\(\lambda\)的指数分布，记作\(X\sim e(\lambda)\)。其分布函数为 $$F(x)= \begin{align} \begin{cases} 1-e^{-\lambda x},~&x>0\\ 0,~&x\leq 0 \end{cases} \end{align} $$
  • 正态分布：若连续型随机变量\(X\)的概率密度为 $$ f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}},~-\infty< x < +\infty $$ 称\(X\)服从参数为\(\mu,\lambda\)的正态分布，记作\(X\sim N(\mu, \sigma^2)\)
